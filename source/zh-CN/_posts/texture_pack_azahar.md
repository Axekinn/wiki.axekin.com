# 完整指南：在 Azahar 模拟器上安装纹理包

## 简介

**Azahar** 模拟器基于 Citra，允许您通过纹理包显著改善任天堂 3DS 游戏的视觉体验。这些包用高清版本替换原始纹理，提供明显增强的图形效果。

## 步骤 1：模拟器准备

### 1.1 启用纹理包支持
1. 在 Azahar 中，转到 **模拟** → **设置**
2. 选择 **图形** 选项卡
3. 勾选 **使用自定义纹理**
4. 勾选 **预加载自定义纹理**（可选，可提高性能）
5. 点击 **确定**

## 步骤 2：获取纹理包

### 2.1 推荐来源
- **纹理包合集**：https://vikingfile.com/folder/-ryt9sVCLmtmdMo4zGrv5NfgK2MeJHIc_Y_2uVDYaYk (临时链接，将很快更新)

### 2.2 兼容性验证
确保纹理包与以下内容匹配：
- 您的 ROM 区域（EUR、USA、JPN）
- 游戏版本
- Azahar/Citra 兼容性

## 步骤 3：纹理包安装

### 3.1 访问游戏的纹理文件夹
1. 在 Azahar 的游戏列表中，**右键单击**所需游戏
2. 从上下文菜单中选择 **打开**
3. 点击 **自定义纹理位置**
4. 将自动打开一个具有正确游戏 ID 的文件夹（例如，0004000000055D00）

### 3.2 安装纹理包
1. **下载** 纹理包（通常是 .zip 或 .7z 文件）
2. 将内容直接 **提取** 到打开的文件夹中
3. **验证** 纹理文件 (.png) 是否正确放置在此文件夹中

### 3.3 最终文件夹结构
提取后，您的文件夹应包含：

```
0004000000055D00/
├── tex1_0x12345678_14.png
├── tex1_0x87654321_14.png
└── ... (其他文件)
```

### 正确的文件夹结构

```
textures/
└── 0004000000055D00/
    ├── tex1_0x12345678_14.png
    ├── tex1_0x87654321_14.png
    └── ... (其他纹理文件)
```

### 纹理包组织
创建一个有组织的结构：
```
textures/
├── Pokemon_X_Y/
├── Mario_Kart_7/
├── Zelda_ALBW/
└── backup/
```

## 提示和最佳实践

### 性能
- **推荐使用 SSD** 以加快加载时间
- **16 GB RAM** 用于无影响的预加载
- **现代 GPU** 用于高分辨率

### 兼容性
- 始终使用您的存档进度的备份进行测试

### 维护
- 定期更新 Azahar
- 检查新的纹理包版本

## 结论

在 Azahar 上安装纹理包可以完全改变任天堂 3DS 游戏的视觉体验。通过遵循此分步指南，您应该能够以改进的图形享受您喜爱的游戏。

随时尝试不同的纹理包和配置，以找到适合您系统的完美设置。
