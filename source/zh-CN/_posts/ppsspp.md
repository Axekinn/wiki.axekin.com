## 简介

**PPSSPP** 是 **PlayStation Portable (PSP)** 的参考模拟器，可让您在 PC、Android 和其他平台上玩 PSP 游戏。本指南详细介绍了实现最佳性能和图形质量的最佳配置。

## 安装和先决条件

### PPSSPP 下载

**🔗 官方网站**：`https://www.ppsspp.org/`

**可用版本：**
- **稳定版**：推荐用于一般用途
- **开发版**：最新功能（可能不稳定）
- **便携版**：无需安装

## 图形设置配置

### 基本设置

#### 渲染后端

**可用渲染选项：**
- **Vulkan**：推荐用于现代 GPU（性能最佳）
- **OpenGL**：与大多数系统兼容
- **Direct3D 11**：Windows 的高性能替代方案
- **软件**：最后的选择（非常慢但兼容性非常好）

#### 分辨率和缩放

**内部渲染分辨率：**
```
1x PSP (480x272) - 原始分辨率
2x PSP (960x544) - 推荐用于大多数系统
3x PSP (1440x816) - 适用于中端 GPU
4x PSP (1920x1088) - 适用于高端 GPU
5x PSP (2400x1360) - 适用于非常高的分辨率
6x PSP+ - 适用于 4K 显示器
```

**缩放模式：**
- **线性**：简单平滑（推荐）
- **最近邻**：像素清晰，复古风格
- **混合**：线性/最近邻混合用于纹理
- **双三次**：高质量但要求更高

### 视觉增强

#### 抗锯齿 (MSAA)

**可用级别：**
- **关闭**：最高性能
- **2x**：良好的质量/性能折衷
- **4x**：显著的视觉改善
- **8x**：仅适用于强大的 GPU

#### 纹理过滤

**过滤类型：**
- **自动**：自动检测（推荐）
- **最近邻**：像素清晰，复古风格
- **线性**：纹理平滑
- **自动最大质量**：最高质量

**纹理增强：**
- **纹理缩放**：xBRZ、混合、双三次以改善纹理
- **去色带**：减少渐变上的条带效应
- **各向异性过滤**：提高远处纹理的清晰度

### 渲染和兼容性

#### 渲染模式

**缓冲渲染：**
- ✅ **启用**：推荐用于兼容性（默认）
- ❌ **禁用**：仅在出现性能问题时使用

**将帧缓冲读取到内存：**
- **关闭**：最高性能
- **开启**：某些视觉效果需要
- **GPU**：使用 GPU 进行传输（更快）

#### PSP 效果模拟

**块传输效果：**
- ✅ **模拟块传输**：某些视觉效果需要
- ✅ **禁用较慢的效果**：禁用要求高的效果

**片段深度：**
- ✅ **测试新的清晰赛车文本**：改善某些游戏的渲染

### 后期处理和着色器

#### 后期处理效果

**流行的着色器：**
- **FXAA**：快速高效的抗锯齿
- **Bloom**：光照和亮度效果
- **Cartoon**：卡通/卡通渲染风格
- **CRT**：CRT 屏幕模拟
- **Scanlines**：复古扫描线
- **Upscale**：纹理增强
- **4xHqGLSL**：高质量滤镜

**着色器配置：**
1. **启用**“后期处理着色器”
2. 从列表中**选择**着色器
3. 如果可用，**调整**强度
4. 如果需要，**链接**多个着色器

## 控制配置

### 支持的控制器类型

**推荐的控制器：**
- **Xbox 360/One/Series**：出色的原生支持
- **PlayStation 3/4/5**：通过 Steam Input 或 DS4Windows
- **Nintendo Switch Pro 控制器**：原生支持
- **8BitDo 控制器**：出色的兼容性
- **键盘/鼠标**：完全可定制的配置

### 自动配置

#### 自动检测

**对于流行的控制器：**
1. 在启动 PPSSPP 之前**连接**您的控制器
2. **转到**设置 > 控制
3. 在“设备”中**选择**您的设备
4. **点击**“自动配置”进行自动配置

## 音频配置

### 基本设置

#### 音频质量

**采样率：**
- **44100 Hz**：标准且推荐
- **48000 Hz**：质量稍好
- **22050 Hz**：适用于性能较差的系统

**音频延迟：**
- **低（更快）**：40-60ms - 推荐用于动作游戏
- **中**：80-100ms - 良好的通用折衷
- **高（更稳定）**：120ms+ - 如果您有音频中断

#### 音频后端

**后端选择：**
- **自动**：自动检测（推荐）
- **WASAPI**：Windows（低延迟）
- **DirectSound**：Windows（兼容）
- **SDL**：跨平台
- **OpenSL ES**：Android

### 高级音频优化

#### 同步

**音频同步：**
- ✅ **启用**：与视频同步（推荐）
- ❌ **禁用**：可能导致速度变慢但能提高性能

**音频时间拉伸：**
- ✅ **启用**：在速度变化期间保持音高
- 对快进有用

#### 重采样质量

**音频重采样器：**
- **线性**：基本质量，最高性能
- **HermitePolynomial**：良好的质量/性能平衡（推荐）
- **BLAM**：高质量但更耗资源

### 每游戏设置

**音量设置：**
- **全局音量**：PPSSPP 的总音量
- **BGM 音量**：背景音乐音量
- **SE 音量**：音效音量

## 网络和多人游戏设置

### Ad Hoc 配置

#### 网络基础设施

**基本设置：**
- ✅ **启用网络/WLAN**：激活网络功能
- **更改 Mac 地址**：必要时修改虚拟 MAC 地址

### 本地和在线多人游戏

#### 多人游戏配置

**与朋友一起玩：**
1. **启用**所有网络选项
2. **配置**相同的 Ad Hoc 服务器
3. **使用** Hamachi、ZeroTier 或直接连接
4. 与您的朋友**分享**服务器 IP
5. 在所有设备上**启动**相同的游戏

#### 聊天和通信

**社交功能：**
- **启用内置聊天**：激活游戏内聊天
- **聊天按钮位置**：聊天按钮的位置
- **聊天屏幕位置**：聊天屏幕的位置

### 基础设施服务器和高级多人游戏

#### 多人游戏服务器类型

**可用服务器：**
- **内置 Ad Hoc 服务器**：简单快速，适合与朋友一起玩
- **社区服务器**：由社区托管的公共服务器
- **自定义服务器**：配置您自己的服务器
- **AdhocServer PSP**：完整的 PSP Ad Hoc 系统模拟

#### 基础设施服务器配置

**流行的公共服务器：**
```
服务器名称：Cold Bird Server
IP：coldbird.net
端口：27312
状态：活跃 - 推荐用于大多数游戏

服务器名称：RetroArch Ad Hoc Server
IP：lobby.libretro.com
端口：6000
状态：稳定 - 可靠的替代方案

服务器名称：Community Server EU
IP：ppsspp-eu.net
端口：27312
状态：专为欧洲设计
```

#### 多人游戏网络优化

**延迟设置：**
- **连接超时**：5-10 秒（在稳定性和响应性之间取得平衡）
- **丢包容忍度**：建议最高 5%
- **Ping 限制**：<150ms 以获得最佳体验
- **缓冲区大小**：根据您的带宽进行调整

**Windows 防火墙配置：**
```cmd
# 为 PPSSPP 打开端口 27312
netsh advfirewall firewall add rule name="PPSSPP AdHoc Server" dir=in action=allow protocol=TCP localport=27312
netsh advfirewall firewall add rule name="PPSSPP AdHoc Server UDP" dir=in action=allow protocol=UDP localport=27312
```

**路由器配置：**
- **端口转发**：27312 (TCP/UDP) 到您的本地 IP
- **UPnP**：启用以进行自动配置

## 高级系统设置

### PSP 系统配置

#### 模拟的 PSP 型号

**PSP 型号：**
- **PSP-1000**：原始 PSP（RAM 较少）
- **PSP-2000/3000**：PSP Slim（RAM 更多，推荐）
- **PSP-Go**：适用于特定的 PSP Go 游戏

#### 固件和 BIOS

**PSP 固件：**
- **版本**：通常为 6.60 或 6.61（最新版本）
- **快速内存**：✅ 启用以获得更好的性能
- **CPU 时钟**：CPU 频率（默认：222MHz，最大：333MHz）

## 工具和实用程序

### 存档管理器

#### 数据管理

**重要文件夹：**
```
/PSP/SAVEDATA/ - 游戏存档
/PSP/SYSTEM/ - PSP 系统文件
/PSP/GAME/ - 自制软件和演示
/seplugins/ - 自定义插件
```

**存档工具：**
- **存档状态管理器**：存档状态管理
- **存档数据管理器**：导入/导出存档

### 自制软件管理器

#### 自制软件支持

**自制软件和演示：**
- 完全支持 PSP 自制软件应用程序
- 通过 `/PSP/GAME/` 文件夹安装
- 与大多数流行的自制软件兼容

> 有可能通过隐藏设置进行操作。我不会在本教程中展示它们，因为我们只关注主要设置。您可以自己随意操作。

## 结论

通过适当的配置，PPSSPP 提供了卓越的 PSP 模拟体验。成功的关键在于根据您的硬件平衡视觉质量和性能。

**要记住的关键点：**
- **开始**时使用适合您配置的推荐设置
- **Vulkan** 通常是现代 GPU 的最佳后端
- **测试**每个游戏的不同设置以优化体验
- 通过每游戏配置文件**保存**您的配置
- **查阅**官方文档以获取最新更新

**其他资源：**
- [游戏兼容性列表](https://report.ppsspp.org/games)
- [社区支持论坛](https://forums.ppsspp.org/)

随意尝试各种设置，为您的最爱游戏找到完美的配置！
