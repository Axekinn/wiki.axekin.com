> 원활하고 안정적인 설정으로 RPCN을 통해 PS3 게임을 온라인으로 플레이하세요. 빠른 시작 체크리스트를 따르고 필요할 때 더 깊이 파고드세요.

> 📝 저자 노트  
> RPCS3는 멀티플레이어 게임을 위해 설정하기가 그 어느 때보다 쉬워졌습니다. 기술에 더 정통한 분들을 위해 매우 잘 설명된 RPCS3 튜토리얼을 포함했습니다.

## ⚡ 빠른 시작 체크리스트

- 최신 RPCS3 빌드로 업데이트
- RPCN 계정 생성 및 확인
- 게임별: 네트워크 = 연결됨, PSN = RPCN
- DNS 8.8.8.8 (게임이 다른 것을 필요로 하지 않는 한)
- PC/라우터에서 UPnP 활성화 또는 UDP 3658 포트 포워딩
- RPCN 창에서 계정 테스트

## ℹ️ RPCN이란?

RPCN은 지원되는 게임에서 온라인 기능을 활성화하기 위해 RPCS3에서 사용하는 커뮤니티 제작 PSN 대체품입니다.

> ⚠️ 면책 조항  
> 99%의 경우 방화벽, NAT/UPnP 또는 포트 포워딩 설정을 변경할 필요가 없습니다. RPCS3 + RPCN은 대부분의 네트워크에서 즉시 작동해야 합니다. 아래 단계를 따른 후 다른 플레이어와의 온라인 연결에 실패한 경우에만 이 설정을 변경하십시오.

## ✅ 요구 사항

| 항목 | 권장 사항 | 참고 |
|---|---|---|
| RPCS3 | 최신 빌드 | 자동 업데이터 또는 새로 다운로드 |
| 포트 | UDP 3658 | UPnP 또는 수동 포트 포워딩 |
| NAT | 개방/유형 A | 이중 NAT/CGNAT 피하기 |
| 방화벽 | RPCS3 허용 | OS 및 보안 제품군 |

---

## 🔧 전역 구성

1. RPCS3 열기 → 구성 → **RPCN**
2. **계정** 클릭
3. 서버 선택: **공식 RPCN 서버**
4. **계정 만들기** 클릭
5. **사용자 이름** 설정 → 확인
6. **비밀번호**를 두 번 입력 → 확인
7. **이메일**을 두 번 입력 → 확인
8. 요약 확인 → **예**
9. 성공 메시지 → "RPCN: 사용자 이름" 창이 나타남
10. RPCN **토큰**에 대한 사서함 확인 → **토큰**에 붙여넣기 → 확인
11. **계정 테스트** 클릭

> 💡 전문가 팁  
> 토큰을 비공개로 유지하세요. 분실한 경우 계정 편집에서 "토큰 재전송"을 사용하세요.

---

## 🎯 게임별 구성

1. 게임 목록에서 제목을 마우스 오른쪽 버튼으로 클릭 → **사용자 지정 구성 만들기**
2. **네트워크**로 이동
3. **네트워크 상태**: 연결됨
4. **PSN 상태**: RPCN
5. **DNS**: `8.8.8.8` (게임이 다른 것을 요구하지 않는 한)
6. 게임에 사용자 지정 서버가 필요한 경우 커뮤니티에서 제공한 IP/호스트 스위치를 입력합니다.

> 💡 게임별 재정의  
> 이 설정은 이 게임에만 적용되며 전역 구성은 변경되지 않습니다.

## 🌐 연결 필수 사항

- RPCS3에서 UPnP 활성화: 구성 → 네트워크 → **UPnP 활성화**
- 라우터에서 UPnP 활성화 (설명서는 모델마다 다름)
- 또는 포트 **3658/UDP**를 PC의 로컬 IP로 전달

<details>
  <summary>포트 포워딩 참고 사항 표시</summary>

- 전달: UDP 3658 → PC IP (DHCP를 통해 예약)  
- 이중 NAT 피하기 (필요한 경우 ISP 모뎀을 브리지)  
- VPN은 인바운드 UDP를 차단할 수 있습니다. VPN에서 포트 포워딩을 구성하거나 플레이하는 동안 비활성화하십시오.
</details>

> 🚧 NAT 및 ISP 제약  
> 엄격한 NAT 또는 CGNAT는 피어 연결을 끊을 수 있습니다. ISP에 공용 IPv4를 요청하거나 포트 포워딩을 지원하는 VPN을 사용하십시오.

---

## 🛠️ 문제 해결

### 다른 플레이어와 연결할 수 없음
- RPCS3 및 라우터에서 UPnP 활성화
- 또는 UDP 3658을 PC로 전달
- 방화벽/보안 제품군을 통해 RPCS3 허용
- VPN 비활성화 또는 재구성
- RPCN → 계정 → **계정 테스트**

<details>
  <summary>고급 확인</summary>

- PC의 IP가 변경되지 않았는지 확인 (DHCP 예약)  
- 이중 NAT 감지 (모뎀 + 라우터 모두 라우팅)  
- 라우터 NAT 테스트는 개방/유형 A를 목표로 해야 함
</details>

## 🔐 RPCN 계정 복구

### RPCN 토큰 재전송
1. RPCS3 → 구성 → RPCN → **계정**
2. **계정 편집** → **토큰 재전송**
3. 확인 → 확인 → "RPCN용 토큰" 이메일 확인
4. **토큰** 필드에 토큰 붙여넣기 → 확인 → **저장**

### RPCN 비밀번호 재설정
1. RPCS3 → 구성 → RPCN → **계정**
2. **계정 편집** → **비밀번호 변경**
3. 사용자 이름 확인 → 확인
4. "비밀번호 재설정 토큰이 이미 있습니까?"라는 질문에 → **아니요**
5. 가입 이메일을 두 번 입력 → 확인
6. "RPCN용 비밀번호 재설정 코드" 이메일 확인
7. 토큰 붙여넣기 → 확인
8. 새 비밀번호를 두 번 입력 → 확인
9. 계정 편집으로 돌아가기 → **비밀번호 설정** → 새 비밀번호 입력 → 확인 → **저장**
10. RPCN: **계정 테스트**

### 좋은 습관
- 강력하고 고유한 비밀번호 사용  
- 토큰을 공유하지 마십시오  
- 최신 RPCN 수정을 위해 RPCS3를 최신 상태로 유지  

> 게임 라이선스 및 온라인 서비스 약관을 존중하십시오. 게임을 구매하여 개발자를 지원하십시오.
