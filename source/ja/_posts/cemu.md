## 🎮 Cemuの紹介

**Cemu**は、利用可能な中で最も高性能で完成度の高いNintendo Wii Uエミュレータです。2015年から開発されており、PCでWii Uの独占タイトルを壮大なグラフィック向上、MOD、優れた互換性でプレイすることができます。

## 🔧 ステップ1：Cemuのダウンロード

### 推奨オプション：公式バージョン
```bash
# 公式ウェブサイト
https://github.com/cemu-project/Cemu/releases
```

1. **クリック**してOSを選択
2. `cemu_X.X.X.zip`（最新バージョン）を**ダウンロード**
3. 専用フォルダ（例：`Emulators\Cemu`またはセカンダリドライブのルート）に**展開**

## 📦 ゲームのダウンロード

### ステップ1：ゲームライブラリへのアクセス

**[🎮 Axekin Games - Nintendo Wii U](https://www.axekin.com/games?platform=wiiu)**にアクセスして、Nintendo Wii Uゲームコレクションにアクセスします。

### ステップ2：ゲームのダウンロード

1. リストから目的のゲームを**検索**
2. ダウンロードボタンを**クリック**
3. ダウンロードが完了するのを**待つ**

### ステップ3：ファイルの展開

ダウンロードが完了したら、次のいずれかのソフトウェアでアーカイブを**展開**します。

| ソフトウェア | ダウンロードリンク | 互換性 |
|----------|---------------|---------------|
| **7-Zip** | [7-Zipをダウンロード](https://www.7-zip.org/) | Windows/Linux |
| **WinRAR** | [WinRARをダウンロード](https://www.win-rar.com/) | Windows |

> **💡 ヒント**：7-Zipは無料でオープンソースですが、WinRARは試用期間後にライセンスが必要です。

**結果**：Cemuで使用できるゲームファイルが手に入ります！ 🎯

### コンテンツの種類


- **code/** 📁: ゲームのメイン.rpxファイルが含まれています
- **content/** 📁: ゲームアセット（テクスチャ、サウンド、ビデオ）
- **meta/** 📁: メタデータとアイコン
- **Updates/** 📁: ゲームのアップデート（互換性のために重要）



✅ **自動検出**：Cemuはゲームを自動的にスキャンして追加します  
✅ **簡素化された管理**：Cemuの明確なインターフェース  
✅ **簡単なアップデート**：アップデートとDLC用の専用フォルダ
✅ **最大限の互換性**：公式のWii Uフォーマットを尊重した構造


## 💡 ポータブル設定（推奨）


Cemuはデフォルトでポータブルです。すべての設定は`mlc01`インストールフォルダに保存され、バックアップと移植性が容易になります。


### ポータブルインストールの構造

```
📁 Cemu/
├── 🎮 Cemu.exe
├── 📁 gameProfiles/      # ゲームプロファイル
├── 📁 graphicPacks/      # グラフィックパックとMOD
├── 📁 memorySearcher/    # メモリ検索ツール
├── 📁 mlc01/            # 仮想Wii Uシステムデータ
├── 📁 shaderCache/      # コンパイル済みシェーダーキャッシュ
└── 📄 settings.xml      # メイン設定
```


**✅ 簡単なバックアップ**：バックアップするフォルダは1つだけ  
**✅ 完全な移植性**：どのPCにも移動可能  
**✅ 設定の維持**：再インストール中に失われない  
**✅ 簡単な共有**：簡単に共有できる設定


## ⚙️ ステップ3：インストールと初回起動

### 初回起動

1. `Cemu.exe`を管理者として**実行**（初回起動のみ）
2. `Game paths`でゲームフォルダを**設定**
3. 「Wii U Games」フォルダを**追加**
4. グラフィックパックを**ダウンロード**
5. ゲームがライブラリに表示されることを**確認**

- その後、閉じるをクリックし、後でコントローラーを設定します

## ⚙️ ステップ4：最適な設定

### 一般設定

```
Options → General settings
```

**一般タブ：**
- **インターフェース言語**：日本語
- **Discord Presence**：✅ 有効（オプション）
- **アップデートの確認**：✅ 有効

### 最適なグラフィック設定

```
Options → General settings → Graphics
```

#### 一般タブ

| パラメータ | 推奨値 | 説明 |
|-----------|------------------|-------------|
| **グラフィックAPI** | Vulkan | 最高のパフォーマンス（問題がある場合はOpenGL） |
| **グラフィックデバイス** | メインGPU | 利用可能な最も強力なGPU |
| **VSync** | ✅ 有効 | 画面のティアリングを防ぐ |

- 残りのパラメータはデフォルトのままで十分です

### オーディオ設定

```
Options → General settings → Audio
```

- **オーディオAPI**：XAudio2（Windows）またはDirectSound
- **オーディオチャンネル**：設定に応じてステレオまたは5.1

### コントロール設定

#### コントローラーの自動設定方法

```
Options → Input settings
```

1. **エミュレートされたコントローラー**：Wii U GamePad
2. **コントローラーAPI**：XInput（Xboxコントローラー）またはDirectInput
3. 「Calibrate」を**クリック**
4. 画面の指示に**従う**
5. すべてのボタンとジョイスティックを**テスト**

## 🎨 ステップ5：グラフィックパックとMOD

### グラフィックパックのインストール

1. **グラフィックパックのダウンロード**：`Options → Download community graphic packs`
2. Cemuを**再起動**
3. **ゲームごとに設定**：ゲームを右クリック → `Edit graphic packs`

**MODの推奨サイト：**
- **GameBanana**: [gamebanana.com](https://gamebanana.com)
- **NexusMods**: [nexusmods.com](https://nexusmods.com)
- **Reddit r/cemu**: アクティブなコミュニティとサポート

## ❓ FAQ - よくある質問

### 一般的な質問

**Q: すべてのWii UゲームはCemuで動作しますか？**
**A:** 約95％のゲームが完全にプレイ可能です。Cemuのウェブサイトで公式の互換性リストを確認してください。

**Q: 実際のWii Uのセーブデータを使用できますか？**
**A:** はい、Saviineまたは専用の自作ソフトを使用すれば、セーブデータを転送できます。

**Q: GamePadは本当に機能しますか？**
**A:** はい、タッチスクリーンはマウスでエミュレートされ、GamePadインターフェースは完全に機能します。

### 技術的な質問

**Q: VulkanとOpenGLのどちらが良いですか？**
**A:** 一般的にVulkanの方がパフォーマンスが優れています。互換性の問題がある場合はOpenGLを使用してください。

**Q: シェーダーのコンパイルに時間がかかるのはなぜですか？**
**A:** ゲームの初回起動時には正常です。シェーダーはその後、後続のセッションのためにキャッシュされます。

### 一般的な問題

**Q: ゲームが突然クラッシュしますか？**
**A:** 有効になっているグラフィックパックを確認し、CemuとGPUドライバーを更新してください。

**Q: 音が途切れますか？**
**A:** オーディオの遅延を64msに増やすか、オーディオAPI（XAudio2/DirectSound）を変更してください。

**Q: テクスチャがぼやけていますか？**
**A:** 16x異方性フィルタリングを有効にし、解像度が正しく設定されていることを確認してください。

### ゲームフォーマット

**Q: .wudと.wuxの違いは何ですか？**
**A:** .wuxは.wudの圧縮版です。どちらも完全に動作し、.wuxはディスクスペースを節約します。

**Q: DLCとアップデートをインストールするにはどうすればよいですか？**
**A:** ベースゲームと同じ構造でUpdates/フォルダに配置してください。
