## 🎮 SteamAutoCrackの紹介

**SteamAutoCrack**は、Steamゲームのパッチ適用プロセスを簡素化する自動化ツールです。Steam認証なしでゲームを動作させるために必要な変更を自動的に適用することで、手順全体を合理化します。

## 🚀 ステップ4：SteamAutoCrackの使用

### SteamAutoCrackインターフェース

**SteamAutoCrack**を起動すると、これらのメインフィールドが表示されます：

#### 1. ゲームディレクトリパス
```
フィールド："Game Path"
目的：クリーンなSteamゲームの場所
例：C:\Games\YourGame\steamapps\common\GameName
```

**入力方法：**
- 展開したゲームに移動します
- パスを選択します：`steamapps/common/GameName/`
- これにはゲーム全体が含まれている必要があります

#### 2. Steam APIキー
```
フィールド："Steam API Key"
目的：認証用のSteam Web APIキー
例：1234567890ABCDEF1234567890ABCDEF12345678
```

**入力方法：**
- Steamから生成したAPIキーを貼り付けます
- これにより、ツールがゲーム情報を取得できます

#### 3. アカウント名
```
フィールド："Account Name"
目的：パッチ適用後のゲームの表示名
デフォルト：通常は自動入力されます
例：YourUsername
```

**入力方法：**
- 好みの名前に変更します（オプション）
- この名前は、該当する場合ゲーム内に表示されます

#### 4. Steam AppID
```
フィールド："Steam AppID"
目的：Steam AppIDを見つける
デフォルト：空、https://steamdb.infoから入力する必要があります
例：221100（DayZの例）
```
