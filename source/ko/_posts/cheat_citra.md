> **⚠️ 중요 경고**: 치트를 사용하면 세이브 파일이 손상되거나 특정 게임이 불안정해질 수 있습니다. 사용에 따른 책임은 본인에게 있습니다!

## 소개

**Citra** 에뮬레이터에서 치트를 사용하는 것은 신규 사용자에게 혼란스러울 수 있습니다. 이 종합 가이드는 에뮬레이트된 Nintendo 3DS 게임에서 치트 코드를 효과적으로 구성하고 사용하는 방법을 **단계별로** 설명합니다.

### 일반적인 과정

1. **Citra 실행** → 게임 로드
2. **에뮬레이션 메뉴** → 치트
3. **치트 추가** → 코드 입력
4. **치트 활성화** → 게임 내에서 테스트
5. 문제가 있는 경우 **버전/지역 확인**

## 전제 조건

시작하기 전에 다음을 확인하십시오:

- ✅ **Citra**가 설치되어 있고 작동하는지 (권장 버전: 최신 안정 버전)
- ✅ **3DS 게임**이 로드되어 있고 작동하는지
- ✅ 게임 버전과 호환되는 **치트 코드**
- ✅ 진행 상황 **백업** (강력 권장)

## 1단계: 치트 메뉴 접근

### 초기 실행

1. **Citra를 시작**하고 에뮬레이터가 게임을 나열할 때까지 기다립니다.
2. 게임이 표시되면 **마우스 오른쪽 버튼 클릭** 후 **속성**으로 이동한 다음 마지막 탭 **치트**로 이동합니다.

### 치트 인터페이스

새 창이 열리며 다음이 표시됩니다:
- **사용 가능한 치트 목록** (처음에는 비어 있음)
- 새 치트를 추가하기 위한 **"치트 추가" 버튼**
- 치트를 활성화/비활성화하기 위한 **체크박스**
- **편집 및 삭제 버튼**

## 2단계: 새 치트 추가

### 치트 생성

1. 치트 창에서 **"치트 추가"**를 클릭합니다.
2. 세 개의 필드가 있는 대화 상자가 나타납니다:
   - **이름**: 치트에 대한 설명적인 이름
   - **메모**: 선택적 설명
   - **코드**: 치트 코드 자체

### 이름 지정 옵션

**권장되는 설명적인 이름:**
- `무한 돈`
- `무제한 생명`
- `모든 아이템`

**추가할 유용한 메모:**
- 호환되는 게임 버전
- 지역 (EU/US/JP)
- 코드 효과

**Citra 코드 형식:**
```
[치트 이름]
주소 값
```

## 4단계: 활성화 및 테스트

### 활성화 절차

1. 해당 필드에 **코드를 입력**합니다.
2. 치트에 **명시적인 이름을 부여**합니다.
3. **"확인"**을 클릭하여 저장합니다.
4. 치트 옆의 **체크박스를 선택**하여 활성화합니다.
5. **게임으로 돌아가** 테스트합니다.

### 효과 확인

치트가 활성화되면 다음을 관찰해야 합니다:

**즉각적인 수정:**
- 돈 값의 변경
- 통계 수정
- 인벤토리에 아이템 추가

**재시작 후 효과:**
- 일부 치트는 재시작이 필요합니다.
- 재시작하기 전에 저장하십시오.
- 효과 지속성 확인

## 문제 해결 - 문제 해결

### 🚫 치트가 작동하지 않음

**가능한 원인:**
- 호환되지 않는 게임 버전
- 다른 지역 (US vs EU vs JP)
- 잘못된 형식의 코드
- 잘못된 메모리 주소

**해결책:**
1. 버전/지역 일치 여부 확인
2. 동일한 게임의 다른 코드로 테스트
3. 전문 포럼 참조
4. 다른 ROM으로 시도

### 💥 게임이 충돌하거나 불안정해짐

**즉각적인 조치:**
1. 모든 치트를 **즉시 비활성화**합니다.
2. 에뮬레이터를 **다시 시작**합니다.
3. 이전 세이브를 **복원**합니다.
4. **치트를 하나씩 테스트**하여 문제를 식별합니다.

**예방:**
- 새 치트를 사용하기 전에 저장하십시오.
- 동시에 너무 많은 치트를 활성화하지 마십시오.
- 신뢰할 수 있는 출처의 코드를 사용하십시오.

### ⚠️ 부분적으로 작동하는 코드

**증상:**
- 일부 효과는 작동하지만 다른 효과는 작동하지 않음
- 잘못된 값이 적용됨
- 일시적인 효과만 있음

**해결책:**
- 코드가 여러 오프셋을 필요로 하는지 확인하십시오.
- 모든 코드 오프셋을 적용해 보십시오.
- 소스 코드 문서를 참조하십시오.

## FAQ - 자주 묻는 질문

**Q: 치트가 모든 버전의 Citra에서 작동하나요?**
A: 대부분의 치트는 최신 버전의 Citra에서 작동합니다. 그러나 일부 코드는 특정 버전에만 해당될 수 있습니다. 가급적 최신 안정 버전을 사용하십시오.

**Q: 여러 치트를 동시에 사용할 수 있나요?**
A: 예, 하지만 주의해야 합니다. 일부 치트가 충돌할 수 있습니다. 결합하기 전에 개별적으로 치트를 테스트하는 것부터 시작하십시오.

**Q: 나만의 코드를 어떻게 만드나요?**
A: 코드를 만들려면 메모리 분석에 대한 지식이 필요합니다. Cheat Engine과 같은 도구를 사용하거나 리버스 엔지니어링 튜토리얼을 참조하십시오.
