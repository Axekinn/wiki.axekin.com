> **⚠️ 重要警告**：此过程涉及在安全模式下重新启动并完全卸载您的驱动程序。请仔细遵循每个步骤，并确保在开始之前已下载驱动程序。

## 简介

**干净地重新安装显卡驱动程序**通常是解决性能问题、游戏崩溃或显示错误的最有效方法。此过程使用 **DDU (Display Driver Uninstaller)** 在安装新驱动程序之前完全删除旧驱动程序。

此方法对于以下情况特别有效：
- 解决新旧版本之间的冲突
- 纠正游戏中的性能问题
- 消除持续的显示错误
- 更换显卡后进行清理

### 一般流程

1. **下载**新驱动程序和 DDU
2. **断开**互联网并以安全模式重新启动
3. 使用 DDU **完全删除**旧驱动程序
4. **重新启动**并安装新驱动程序
5. **重新连接**互联网并验证功能

## 先决条件和下载

### 识别您的显卡

在开始之前，请识别您的 GPU：

**方法 1：设备管理器**
1. **右键单击**“此电脑”→ **属性**
2. **设备管理器** → **显示适配器**
3. **记下**您显卡的确切型号

**方法 2：系统命令**
```cmd
dxdiag
```
在 **“显示”** 选项卡中查看您的 GPU。

### 必要下载

#### 对于 NVIDIA 显卡

**🔗 [NVIDIA 官方网站](https://www.nvidia.cn/Download/index.aspx?lang=cn)**

**下载步骤：**
1. **选择**您的显卡系列（RTX 40、RTX 30、GTX 16 等）
2. **选择**您的确切型号
3. **下载**最新的 **“Game Ready”** 驱动程序
4. **保存**文件到您的桌面

**NVIDIA 驱动程序类型：**
- **Game Ready Driver**：针对游戏优化（推荐）
- **Studio Driver**：针对内容创作优化

#### 对于 AMD 显卡

**🔗 [AMD 官方网站](https://www.amd.com/zh-hans/support)**

**下载步骤：**
1. **选择**“显卡”→ 您的系列（RX 7000、RX 6000 等）
2. **选择**您的确切型号
3. **下载**完整的 **“Adrenalin”** 软件包
4. **保存**文件到您的桌面

#### Display Driver Uninstaller (DDU)

**🔗 [下载 DDU](https://www.guru3d.com/download/display-driver-uninstaller-download/)**

**重要信息：**
- **免费**且使用安全
- **便携式**：无需安装
- **兼容**所有最近的 Windows 版本
- **定期更新**

## 步骤 1：系统准备

### 重要备份

在开始之前：

**🛡️ 基本备份点：**
- **游戏配置文件**（NVIDIA GeForce Experience、AMD Software 设置）
- **自定义显示设置**
- **应用的超频**（记下数值）

### 关闭程序

**关闭所有与 GPU 相关的软件：**
- GeForce Experience / AMD Software
- MSI Afterburner / GPU-Z
- 游戏和 3D 应用程序
- 直播软件（OBS 等）

### 断开互联网（至关重要）

> **🔥 关键步骤**：在以安全模式重新启动之前，物理拔下以太网电缆或禁用 Wi-Fi。

**为什么要断开互联网？**
- 防止 Windows 自动安装通用驱动程序
- 避免在重新安装过程中发生冲突
- 保证 DDU 的干净环境

## 步骤 2：进入安全模式

### 对于 Windows 10

#### 通过设置

1. **打开**“设置”（Windows + I）
2. **点击**“更新和安全”
3. 在左侧菜单中**选择**“恢复”
4. 在 **“高级启动”**下，点击 **“立即重新启动”**

### 对于 Windows 11

#### 通过设置

1. **打开**“设置”（Windows + I）
2. 在左侧菜单中**点击**“Windows 更新”
3. **选择**“高级选项”
4. **点击**“恢复”
5. 在 **“高级启动”**下，点击 **“立即重新启动”**

### 在高级启动菜单中导航

PC 重新启动后，您将看到一个带有选项的蓝色屏幕：

#### 导航顺序

1. **点击**“疑难解答”
2. **点击**“高级选项”
3. **点击**“启动设置”
4. **点击**“重新启动”

#### 启动设置菜单

重新启动后，您将看到一个编号列表：

**可用选项：**
```
1) 启用调试
2) 启用启动日志记录
3) 启用低分辨率视频
4) 启用安全模式          ← 选择此选项
5) 启用带网络的安全模式
6) 启用带命令提示符的安全模式
7) 禁用系统失败时自动重新启动
8) 禁用驱动程序签名强制
9) 禁用早期启动反恶意软件保护
```

**按“4”键**进入标准安全模式。

## 步骤 3：使用 DDU

### 首次启动

进入安全模式后：

1. **导航**到您解压 DDU 的文件夹
2. **右键单击**“Display Driver Uninstaller.exe”
3. **选择**“以管理员身份运行”

### DDU 配置

#### 选择设备类型

在 **“选择设备类型”** 下拉菜单中：
- **选择**“GPU”

#### 选择制造商

在 **“选择设备”** 菜单中：
- **对于 NVIDIA**：选择“NVIDIA”
- **对于 AMD**：选择“AMD”
- **对于 Intel**：选择“INTEL”（如果适用）

### 卸载过程

#### 启动卸载

1. **点击**“清理并重新启动”
2. 如果出现提示，**确认**操作
3. **等待** DDU 完成过程（2-5 分钟）

#### 清理进度

DDU 将：
- **删除**所有驱动程序文件
- **清理** Windows 注册表
- **删除**与驱动程序相关的服务
- **删除**临时文件夹
- **自动重新启动**

**进度指示器：**
- 绿色进度条
- 实时状态消息
- 已删除项目列表

## 步骤 4：安装新驱动程序

### DDU 后验证

自动重新启动后：

**成功卸载的迹象：**
- 基本屏幕分辨率（1024x768 或类似）
- 设备管理器中没有 GPU 驱动程序
- Windows 使用基本显示驱动程序

## 高级故障排除

### 常见问题

#### 🚫 安装失败

**可能的原因：**
- 未删除旧驱动程序的残留物
- 防病毒软件阻止安装
- 安装文件损坏

**解决方案：**
1. 在安全模式下**重新运行** DDU
2. **重新下载**驱动程序
3. **尝试**在兼容模式下安装

#### ⚠️ 安装后黑屏

**紧急措施：**
1. 在安全模式下**重新启动**（启动时按 F8）
2. **重新运行** DDU 再次清理
3. **安装**以前的稳定版本
4. **检查** GPU/驱动程序兼容性

#### 💻 性能下降

**验证：**
- **GPU 温度**（MSI Afterburner / HWMonitor）
- 任务管理器中的 **GPU 使用率**
- **Windows 电源设置**（最高性能）

### 性能监控

**监控工具：**
| 工具 | 功能 | 用途 |
|---|---|---|
| **MSI Afterburner** | 完整监控 | 温度、使用率、FPS |
| **GPU-Z** | 详细信息 | 规格、传感器 |
| **HWiNFO64** | 系统监控 | 高级诊断 |

### 警告信号

**需要重新安装的指标：**
- 频繁的游戏崩溃
- 图形伪影
- 无法解释的性能下降

## FAQ - 常见问题

**问：我应该在每次驱动程序更新时都使用 DDU 吗？**
**答：**不，DDU 仅在出现问题或进行重大更改（新 GPU、损坏的版本等）时才需要。

**问：我可以在没有安全模式的情况下使用 DDU 吗？**
**答：**强烈不建议。安全模式可确保没有进程干扰卸载。

**问：如果 DDU 没有检测到我的显卡怎么办？**
**答：**在使用 DDU 之前，请检查您的 GPU 是否已正确安装并在设备管理器中被识别。

**问：我的游戏设置会丢失吗？**
**答：**不会，游戏设置通常单独存储。只有 GPU 控制面板配置文件会丢失。

**问：整个过程需要多长时间？**
**答：**整个过程大约需要 30-45 分钟，包括下载和重新启动。
