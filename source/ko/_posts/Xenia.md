## 🎮 Xenia 소개

**Xenia**는 가장 진보된 오픈 소스 Xbox 360 에뮬레이터입니다. 2013년부터 개발되어 Xbox 360의 PowerPC Xenon 아키텍처를 에뮬레이트하며 Microsoft 콘솔의 많은 상징적인 게임과의 호환성이 점차 증가하고 있습니다.

> 💡 **중요 참고**: Xenia는 단일 스레드 성능이 좋은 강력한 CPU가 필요합니다. Vulkan을 지원하는 최신 GPU가 최상의 결과를 제공합니다.

> 💡 **Xenia Manager**: Xenia를 더 잘 이해하고 사용하기 위해 Xenia Manager를 사용할 것입니다. 이것은 Xenia 개발자의 공식적인 것은 아니지만 이 에뮬레이터 초보자에게는 이보다 더 좋은 것은 없습니다.

## 🔧 1단계: Xenia Manager 다운로드

### 마스터 버전 (안정)
```bash
# 공식 사이트
https://github.com/xenia-manager/xenia-manager/releases
```

1. "xenia_manager.zip"을 **클릭**합니다.
2. `xenia_master.zip`을 **다운로드**합니다.
3. 전용 폴더(예: `Documents\Xenia` 또는 다른 드라이브의 루트, C 드라이브 아님)에 **압축을 풉니다**.

## 게임 다운로드

### 1단계: 게임 라이브러리 액세스

**[🎮 Axekin Games - Xbox 360](https://www.axekin.com/games?platform=xbox360)**으로 이동하여 Xbox 360 게임 컬렉션에 액세스하십시오.

### 2단계: 게임 다운로드

1. 목록에서 원하는 게임을 **검색**합니다.
2. 다운로드 버튼을 **클릭**합니다.
3. 다운로드가 완료될 때까지 **기다립니다**.

### 3단계: 파일 추출

다운로드가 완료되면 다음 소프트웨어 중 하나로 아카이브를 **추출**합니다.

| 소프트웨어 | 다운로드 링크 | 호환성 |
|----------|---------------|---------------|
| **7-Zip** | [7-Zip 다운로드](https://www.7-zip.org/) | Windows/Linux |
| **WinRAR** | [WinRAR 다운로드](https://www.win-rar.com/) | Windows |

> **💡 팁**: 7-Zip은 무료이며 오픈 소스이지만 WinRAR은 평가판 기간이 지나면 라이선스가 필요합니다.

**결과**: Xenia와 함께 사용할 준비가 된 게임 파일을 얻게 됩니다! 🎯

## 📁 2단계: 폴더 구조

이 구조로 게임을 정리하십시오.

```
Xbox 360 게임/
├── Forza Horizon/
│   ├── BASE/                # 🎮 기본 게임 (메인 파일)
│   ├── UPDATE/              # 🔄 게임 업데이트 (권장)
│   └── DLC/                 # 📦 다운로드 가능 콘텐츠 (선택 사항)
├── Skate 3/
│   ├── BASE/                # 🎮 기본 게임
│   ├── UPDATE/              # 🔄 게임 업데이트
│   └── DLC/                 # 📦 추가 DLC
├── Juju/
│   ├── BASE/
│   ├── UPDATE/
│   └── DLC/
└── Gears of War/
    ├── BASE/
    └── UPDATE/              # (이 게임에는 DLC 없음)
```

### 콘텐츠 유형


- **BASE** 📁: 다운로드하는 메인 게임
- **UPDATE** 📁: 업데이트 및 패치 (권장)
- **DLC** 📁: 추가 유료 콘텐츠 (선택 사항이지만 권장)


✅ **자동 감지**: 게임이 에뮬레이터에 자동으로 추가됩니다.  
✅ **간소화된 관리**: 각 ROM을 수동으로 선택할 필요가 없습니다.  
✅ **명확한 구성**: 각 게임에 어떤 콘텐츠가 있는지 쉽게 확인할 수 있습니다.  
✅ **쉬운 유지 관리**: 콘텐츠 추가/제거가 간소화됩니다.

> **📝 중요 참고**: 게임에 업데이트나 DLC가 없는 경우 해당 폴더를 만들 필요가 없습니다. 실제로 가지고 있는 콘텐츠에 대해서만 폴더를 만드십시오.

## ⚙️ 3단계: 첫 실행 및 구성

## 첫 실행

### 1단계: 자동 업데이트


`XeniaManager.Updater.exe`를 **실행**하여 최신 Xenia 빌드로 자동 업데이트합니다.

이론적으로는 여기서 멈출 수 있지만 더 나아가 봅시다! 일반적으로 안정 버전으로 충분하지만 Xenia의 **Canary 버전**은 메인라인에 비해 렌더링/결과가 더 좋고 버그가 적습니다.

### 2단계: Xenia Canary 설치

1. `Install Xenia Canary`를 **클릭**합니다.
2. 창이 나타나면 → "Create profile"을 **클릭**합니다.
3. 사용자 이름을 **입력**합니다(온라인 모드에서 사용됨).
4. 창을 **닫습니다**.


- **Xenia MouseHook**: 키보드/마우스로 플레이하려는 경우에만 설치하십시오.
- **Xenia Netplay**: 온라인 플레이용(프로필 생성도 필요).

5. "Welcome to Xenia Manager" 창을 **닫습니다**.

### 3단계: 인터페이스 구성

우리는 미치지 않았으니 다크 모드를 활성화합시다! 🌙

**탐색**: `Settings`(왼쪽) → `Theme Switcher` → **Dark**

### 4단계: 일반 구성

같은 창에서 다음 옵션을 **확인**하십시오.


✅ **Automatic Game Parsing and Selection**  
✅ **Automatic Save Backup**

### 5단계: 에뮬레이터 구성

`Xenia Settings`로 이동하여 구성합니다.

#### 🔊 오디오 및 디스플레이
| 매개변수 | 권장 값 | 참고 |
|-----------|-------------------|------|
| **Audio System** | `SDL` | 모든 시스템과 호환 |
| **Full Screen** | `On` | 창 모드를 선호하는 경우 `Off`로 두십시오. |
| **Internal Display Resolution** | `1920x1080` | ⚠️ 높을수록 더 많은 성능 필요 |

#### 🎮 그래픽
| 매개변수 | 권장 값 | 대안 |
|-----------|-------------------|-------------|
| **Graphics API** | `Vulkan` | 문제/오래된 하드웨어의 경우 `OpenGL` |
| **Anti Aliasing** | 지원되는 최대 값 | 제한된 구성인 경우 건드리지 마십시오. |
| **Scaling and Sharpening** | 지원되는 최대 값 | 위와 동일 |

#### 🏆 게임 경험
| 매개변수 | 값 | 설명 |
|-----------|-------|-------------|
| **Show Achievement Notifications** | `On` | 옛날의 추억을 위해! |
| **User Country** | 귀하의 국가 | 귀하의 위치에 따라 |
| **User Language** | 귀하의 언어 | 귀하의 위치에 따라 |

#### 🎯 컨트롤
| 매개변수 | 사용법 | 설명 |
|-----------|-------|-------------|
| **XInput** | Xbox/호환 컨트롤러 | 대부분의 컨트롤러 표준 |
| **SDL2** | 인식되지 않는 컨트롤러 | XInput이 작동하지 않는 경우 |
| **Keyboard** | 키보드/마우스 | PC 플레이어용 |

### 6단계: 저장


오른쪽 상단의 `Save` 버튼을 **클릭**하여 모든 설정을 저장합니다.

## Xenia에 게임 추가

### 폴더 구조를 사용한 빠른 방법

권장 폴더 구성을 따랐다면:


1. 오른쪽 상단의 작은 `+`를 **클릭**합니다.
2. 원하는 게임을 **추가**합니다.
3. `Full Auto`를 **클릭**합니다.
4. **게임이** 메뉴에 자동으로 나타납니다!

추가하려는 모든 게임에 대해 이 작업을 **반복**합니다.

### 지금 플레이하고 싶으신가요?


참을성이 없어서 바로 플레이하고 싶다면 할 수 있습니다! 그렇지 않다면 경험을 최적화해 봅시다... 🎮

## DLC 및 업데이트 설치

### 설치 방법


1. 게임을 **마우스 오른쪽 버튼으로 클릭** → `Content` → `Install DLC/Updates`
2. DLC/업데이트 파일을 **선택**합니다.
3. `content/[TitleID]/`에 **자동 설치**됩니다.
4. 콘텐츠를 활성화하려면 게임을 **다시 시작**하십시오.

### 설치 후 구조

```
📁 Xenia/
├── 📁 content/
│   └── 📁 [TitleID]/
│       ├── 📄 기본 게임
│       ├── 📄 업데이트 파일
│       └── 📄 DLC 콘텐츠
```

### 콘텐츠 유형

| 유형 | 확장자 | 설명 |
|------|-----------|-------------|
| **기본 게임** | `.iso`, `.xex` | 메인 게임 파일 |
| **업데이트** | `.xbla` | 패치 및 개선 사항 |
| **DLC** | `.xbla` | 다운로드 가능 콘텐츠 |


새 콘텐츠를 인식하려면 설치 후 게임을 **다시 시작**하십시오!

🎯 **결과**: 이제 게임에 사용 가능한 모든 업데이트와 DLC가 추가되었습니다!

## ❓ FAQ - 자주 묻는 질문

### 일반적인 질문

**Q: Xenia는 Linux/Mac에서 작동합니까?**
**A:** 아니요, Xenia는 Windows 10/11 64비트 전용입니다. 공식 포팅 계획은 없습니다.

**Q: Xenia로 온라인 플레이를 할 수 있습니까?**
**A:** 예

**Q: Xenia는 오리지널 Xbox 게임을 플레이할 수 있습니까?**
**A:** 아니요, Xenia는 Xbox 360만 에뮬레이트합니다. 오리지널 Xbox에는 xemu를 사용하십시오.

### 기술적인 질문

**Q: Vulkan 또는 D3D12?**
**A:** 일반적으로 Vulkan이 더 성능이 좋고 D3D12는 때때로 더 안정적입니다. 게임에 따라 테스트하십시오.

**Q: 최소 RAM은 얼마입니까?**
**A:** 최소 8GB, 16GB 강력 권장, 대용량 게임의 경우 32GB.

**Q: 제 GPU는 충분합니까?**
**A:** 최소 DirectX 12 또는 Vulkan 1.1이 필요합니다. 괜찮은 성능을 위해서는 GTX 1060/RX 580.

### 일반적인 문제

**Q: 실행 시 게임이 충돌합니까?**
**A:** 공식 호환성을 확인하고 다른 API를 테스트하고 파일 무결성을 확인하십시오.

**Q: 성능이 매우 나쁩니까?**
**A:** CPU가 주요 제한 요소입니다. 가능한 한 많은 앱을 닫고 SSD를 사용하십시오.

**Q: 오디오가 동기화되지 않습니까?**
**A:** 오디오 버퍼를 늘리고 샘플 속도를 변경하고 다른 오디오 앱을 닫으십시오.

**Xenia로 Xbox 360 독점작을 재발견하십시오! 🎮✨**
