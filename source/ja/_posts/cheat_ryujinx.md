> **⚠️ 重要**：チートやMODをインストールする前に、必ずセーブファイルをバックアップしてください。一部の改造はゲームの安定性に影響を与える可能性があります。

## はじめに

**Ryujinx**は、Nintendo Switchのゲーム体験を向上させるために、**チートコード**と**ゲームの改造（MOD）**の両方をサポートしています。無限の体力、すべてのアイテムのアンロック、またはゲームのグラフィックを完全に変えたい場合でも、このガイドではそれらをインストールする方法を正確に示します。

### 違いは何ですか？

**チート vs MOD：**
- **チート**：小さなコードの変更（無限のお金、最大ステータスなど）
- **MOD**：ゲームコンテンツの完全な変更（新しいテクスチャ、キャラクター、レベルなど）

どちらもRyujinxで異なるインストール方法とフォルダ構造を使用します。

## Ryujinxにチートをインストールする

### ステップ1：Modディレクトリにアクセスする

#### Ryujinxを開き、ゲームを見つける

1. Ryujinxエミュレータを**起動**します
2. チートを追加したいゲームを**右クリック**します
3. コンテキストメニューから**「Modディレクトリを開く」**を選択します

### ステップ2：フォルダ構造を理解する

#### ディレクトリ構造

Modディレクトリを開くと、次のようなパスが表示されます：
```
Ryujinx\mods\contents\XXXXXXXXXXXXXXXX\
```

ここで `XXXXXXXXXXXXXXXX` は、選択したゲームの**一意のゲームID**（タイトルID）です。

**ゲームIDの例：**
- ゼルダの伝説 ブレス オブ ザ ワイルド：`01007EF00011E000`
- スーパーマリオ オデッセイ：`0100000000010000`
- ポケットモンスター ソード：`0100ABF008968000`

### ステップ3：チートファイルをインストールする

#### チートファイルを展開して配置する

1. チートファイル（通常は.zip形式）を**ダウンロード**します
2. アーカイブの内容を**展開**します
3. **適切なフォルダ構造を作成します：**

```
Ryujinx\mods\contents\XXXXXXXXXXXXXXXX\
└── チート名\
    └── cheats\
        ├── cheat1.txt
        ├── cheat2.txt
        └── other_cheats.txt
```

#### 詳細なインストール手順

4. ゲームのModディレクトリに**移動**します
5. チート名で**新しいフォルダを作成**します（例：「無限のお金」）
6. **このフォルダ内**に、**「cheats」**という名前のサブフォルダを作成します
7. **すべての.txtチートファイル**を「cheats」フォルダに**配置**します

### ステップ4：チートを有効にする

#### ゲーム内でチートを有効にする

1. Ryujinxで**ゲームを起動**します
2. ゲームタイトルを再度**右クリック**します
3. **「チートの管理」**を選択するか、メニューからアクセスします
4. 有効にしたいチートの横にある**チェックボックスをオン**にします
5. 変更を**適用**し、必要に応じてゲームを再起動します

## RyujinxにMODをインストールする

### ステップ1：Modディレクトリにアクセスする

#### チートと同じ初期プロセス

1. **Ryujinxを開きます**
2. ターゲットのゲームを**右クリック**します
3. **「Modディレクトリを開く」**をクリックします

これにより、`AppData\Roaming\Ryujinx\mods\contents\<ゲームID>\`が開きます。

### ステップ2：MODフォルダ構造を作成する

#### MODを正しく整理する

1. ゲームIDディレクトリの下に**新しいサブフォルダを作成**します
2. **わかりやすい名前を付けます**（例：「無限のライフ」、「HDテクスチャ」、「カスタムミュージック」）
3. これにより、**複数のMODを簡単に管理**できます

**構造の例：**
```
Ryujinx\mods\contents\01007EF00011E000\
├── HDグラフィックMOD\
│   └── romfs\
├── 無限ライフ\
│   └── exefs\
└── カスタムサウンドトラック\
    └── romfs\
```

### ステップ3：MODファイルをインストールする

#### MODコンテンツをコピーする

1. **ダウンロードしたMODファイルを展開**します
2. MODから**romfsまたはexefsフォルダを見つけます**
3. **これらのフォルダを**新しく作成したMODディレクトリに**コピー**します

**MODフォルダの種類：**
- **romfs**：ゲームコンテンツの変更（テクスチャ、モデル、オーディオ）
- **exefs**：実行ファイルの変更（コードパッチ、ゲームプレイの変更）

#### 完全なインストール

4. MODの展開フォルダに**移動**します
5. **romfsまたはexefsフォルダ全体をコピー**します
6. 作成したMODディレクトリに**貼り付け**ます

### ステップ4：MODを有効にする

#### インストールされたMODを有効にする

1. **Ryujinxを起動**します
2. ゲームを**右クリック**します
3. **「MODの管理」**を選択します
4. 好みに応じてMODを**有効/無効**にします
5. **ゲームを開始**して変更を確認します

## フォルダ構造の例

### 完全なチートインストール

```
Ryujinx\mods\contents\01007EF00011E000\
└── BOTW_Cheats\
    └── cheats\
        ├── infinite_stamina.txt
        ├── max_hearts.txt
        ├── all_weapons.txt
        └── infinite_rupees.txt
```

### 完全なMODインストール

```
Ryujinx\mods\contents\01007EF00011E000\
├── HD_Graphics_Pack\
│   └── romfs\
│       ├── Model\
│       ├── Texture\
│       └── Effect\
└── Gameplay_Overhaul\
    ├── romfs\
    │   └── GameData\
    └── exefs\
        └── main.npdm
```

#### MODの互換性

**互換性テスト：**
1. **一度に1つのMODをインストール**します
2. **ゲームの安定性をテスト**します
3. 他を追加する前に**競合を確認**します
4. 動作する組み合わせについて**メモを取ります**

## 一般的な問題のトラブルシューティング

### チートが機能しない

#### 一般的な問題と解決策

**🚫 チートファイルが検出されない：**
- フォルダ構造を確認してください：`ゲームID\チート名\cheats\*.txt`
- ファイル拡張子を確認してください（.txt、.txt.txtではない）
- フォルダ名に余分なスペースがないことを確認してください

**⚠️ チートがクラッシュを引き起こす：**
- すべてのチートを無効にして、1つずつテストしてください
- チートとゲームバージョンの互換性を確認してください
- 信頼できるソースからチートを使用してください
- チートのバージョンがゲームのバージョンと一致することを確認してください

### MODがロードされない

#### MODの問題のデバッグ

**📁 不正なフォルダ構造：**
```
❌ 不正解：
Ryujinx\mods\contents\ゲームID\romfs\...

✅ 正解：
Ryujinx\mods\contents\ゲームID\MOD名\romfs\...
```

**🎮 ゲーム固有の問題：**
- MODとゲームバージョンの互換性を確認してください
- MODが特定のファームウェアバージョンを必要とするか確認してください
- MODを無効にしてテストし、原因であることを確認してください

## FAQ - よくある質問

**Q: チートとMODを同時に使用できますか？**
A: はい、ただし、一部の組み合わせは競合やクラッシュを引き起こす可能性があるため、慎重にテストしてください。

**Q: MODはゲームのパフォーマンスに影響しますか？**
A: MODによります。グラフィックMODは通常より多くのリソースを必要としますが、単純なチートは最小限の影響しかありません。

**Q: RyujinxのMODはYuzuと互換性がありますか？**
A: 時々ありますが、フォルダ構造が異なります。クロス互換性のためにファイルを再編成する必要がある場合があります。

**Q: 独自のチートを作成できますか？**
A: はい、メモリアドレスとゲーム構造の知識があれば可能です。Cheat Engineのようなツールは、変更する値を特定するのに役立ちます。

**Q: MODがゲームを壊した場合どうすればよいですか？**
A: MODを無効にするか、MODディレクトリから削除するだけです。元のゲームファイルはそのまま残ります。
