> **⚠️ 중요**: 치트나 모드를 설치하기 전에 항상 세이브 파일을 백업하십시오. 일부 수정 사항은 게임 안정성에 영향을 줄 수 있습니다.

## 소개

**Ryujinx**는 Nintendo Switch 게임 경험을 향상시키기 위해 **치트 코드**와 **게임 수정(모드)**을 모두 지원합니다. 무한 체력, 모든 아이템 잠금 해제 또는 게임 그래픽을 완전히 변형하고 싶다면 이 가이드에서 정확한 설치 방법을 보여줍니다.

### 차이점은 무엇인가요?

**치트 vs 모드:**
- **치트**: 작은 코드 수정 (무한 돈, 최대 능력치 등)
- **모드**: 전체 게임 콘텐츠 수정 (새로운 텍스처, 캐릭터, 레벨 등)

두 가지 모두 Ryujinx에서 다른 설치 방법과 폴더 구조를 사용합니다.

## Ryujinx에 치트 설치하기

### 1단계: 모드 디렉토리 접근

#### Ryujinx를 열고 게임 찾기

1. Ryujinx 에뮬레이터를 **실행**합니다.
2. 치트를 추가하려는 게임을 **마우스 오른쪽 버튼으로 클릭**합니다.
3. 컨텍스트 메뉴에서 **"모드 디렉토리 열기"**를 선택합니다.

### 2단계: 폴더 구조 이해하기

#### 디렉토리 구조

모드 디렉토리를 열면 다음과 같은 경로가 표시됩니다.
```
Ryujinx\mods\contents\XXXXXXXXXXXXXXXX\
```

여기서 `XXXXXXXXXXXXXXXX`는 선택한 게임의 **고유 게임 ID**(타이틀 ID)입니다.

**게임 ID 예시:**
- 젤다의 전설: 브레스 오브 더 와일드: `01007EF00011E000`
- 슈퍼 마리오 오디세이: `0100000000010000`
- 포켓몬스터 소드: `0100ABF008968000`

### 3단계: 치트 파일 설치

#### 치트 파일 압축 해제 및 배치

1. 치트 파일(일반적으로 .zip 형식)을 **다운로드**합니다.
2. 아카이브 내용을 **압축 해제**합니다.
3. **적절한 폴더 구조를 만듭니다:**

```
Ryujinx\mods\contents\XXXXXXXXXXXXXXXX\
└── 치트_이름\
    └── cheats\
        ├── cheat1.txt
        ├── cheat2.txt
        └── other_cheats.txt
```

#### 상세 설치 단계

4. 게임의 모드 디렉토리로 **이동**합니다.
5. 치트 이름으로 **새 폴더를 만듭니다**(예: "무한_돈").
6. **이 폴더 안에** **"cheats"**라는 하위 폴더를 만듭니다.
7. **모든 .txt 치트 파일**을 "cheats" 폴더에 **넣습니다**.

### 4단계: 치트 활성화

#### 게임 내에서 치트 활성화

1. Ryujinx에서 **게임을 시작**합니다.
2. 게임 제목을 다시 **마우스 오른쪽 버튼으로 클릭**합니다.
3. **"치트 관리"**를 선택하거나 메뉴를 통해 접근합니다.
4. 활성화하려는 치트 옆의 **체크박스를 선택**합니다.
5. 변경 사항을 **적용**하고 필요한 경우 게임을 다시 시작합니다.

## Ryujinx에 모드 설치하기

### 1단계: 모드 디렉토리 접근

#### 치트와 동일한 초기 과정

1. **Ryujinx를 엽니다.**
2. 대상 게임을 **마우스 오른쪽 버튼으로 클릭**합니다.
3. **"모드 디렉토리 열기"**를 클릭합니다.

그러면 `AppData\Roaming\Ryujinx\mods\contents\<게임 ID>\`가 열립니다.

### 2단계: 모드 폴더 구조 만들기

#### 모드를 올바르게 구성하기

1. 게임 ID 디렉토리 아래에 **새 하위 폴더를 만듭니다.**
2. **설명적인 이름을 지정합니다**(예: "무한_생명", "HD_텍스처", "사용자_지정_음악").
3. 이렇게 하면 **여러 모드를 쉽게 관리**할 수 있습니다.

**구조 예시:**
```
Ryujinx\mods\contents\01007EF00011E000\
├── HD_그래픽_모드\
│   └── romfs\
├── 무한_생명\
│   └── exefs\
└── 사용자_지정_사운드트랙\
    └── romfs\
```

### 3단계: 모드 파일 설치

#### 모드 콘텐츠 복사

1. **다운로드한 모드 파일을 압축 해제**합니다.
2. 모드에서 **romfs 또는 exefs 폴더를 찾습니다.**
3. **이 폴더들을** 새로 만든 모드 디렉토리에 **복사**합니다.

**모드 폴더 유형:**
- **romfs**: 게임 콘텐츠 수정 (텍스처, 모델, 오디오)
- **exefs**: 실행 파일 수정 (코드 패치, 게임 플레이 변경)

#### 전체 설치

4. 모드의 압축 해제 폴더로 **이동**합니다.
5. **전체 romfs 또는 exefs 폴더를 복사**합니다.
6. 생성한 모드 디렉토리에 **붙여넣습니다.**

### 4단계: 모드 활성화

#### 설치된 모드 활성화

1. **Ryujinx를 실행**합니다.
2. 게임을 **마우스 오른쪽 버튼으로 클릭**합니다.
3. **"모드 관리"**를 선택합니다.
4. 기본 설정에 따라 모드를 **활성화/비활성화**합니다.
5. **게임을 시작**하여 변경 사항을 확인합니다.

## 폴더 구조 예시

### 전체 치트 설치

```
Ryujinx\mods\contents\01007EF00011E000\
└── BOTW_치트\
    └── cheats\
        ├── infinite_stamina.txt
        ├── max_hearts.txt
        ├── all_weapons.txt
        └── infinite_rupees.txt
```

### 전체 모드 설치

```
Ryujinx\mods\contents\01007EF00011E000\
├── HD_그래픽_팩\
│   └── romfs\
│       ├── Model\
│       ├── Texture\
│       └── Effect\
└── 게임플레이_오버홀\
    ├── romfs\
    │   └── GameData\
    └── exefs\
        └── main.npdm
```

#### 모드 호환성

**호환성 테스트:**
1. **한 번에 하나의 모드만 설치**합니다.
2. **게임 안정성을 테스트**합니다.
3. 다른 모드를 추가하기 전에 **충돌을 확인**합니다.
4. 작동하는 조합에 대해 **메모를 작성**합니다.

## 일반적인 문제 해결

### 치트가 작동하지 않음

#### 일반적인 문제 및 해결책

**🚫 치트 파일이 감지되지 않음:**
- 폴더 구조 확인: `게임_ID\치트_이름\cheats\*.txt`
- 파일 확장자 확인 (.txt, .txt.txt 아님)
- 폴더 이름에 추가 공백이 없는지 확인

**⚠️ 치트로 인한 충돌:**
- 모든 치트를 비활성화하고 하나씩 테스트합니다.
- 치트와 게임 버전의 호환성을 확인합니다.
- 신뢰할 수 있는 출처의 치트를 사용합니다.
- 치트 버전이 게임 버전과 일치하는지 확인하십시오.

### 모드가 로드되지 않음

#### 모드 문제 디버깅

**📁 잘못된 폴더 구조:**
```
❌ 잘못됨:
Ryujinx\mods\contents\게임_ID\romfs\...

✅ 올바름:
Ryujinx\mods\contents\게임_ID\모드_이름\romfs\...
```

**🎮 게임별 문제:**
- 모드와 게임 버전의 호환성을 확인합니다.
- 모드가 특정 펌웨어 버전을 필요로 하는지 확인합니다.
- 모드를 비활성화하여 원인인지 확인합니다.

## FAQ - 자주 묻는 질문

**Q: 치트와 모드를 동시에 사용할 수 있나요?**
A: 예, 하지만 일부 조합은 충돌이나 충돌을 일으킬 수 있으므로 신중하게 테스트하십시오.

**Q: 모드가 게임 성능에 영향을 미치나요?**
A: 모드에 따라 다릅니다. 그래픽 모드는 일반적으로 더 많은 리소스를 필요로 하는 반면 간단한 치트는 최소한의 영향을 미칩니다.

**Q: Ryujinx 모드는 Yuzu와 호환되나요?**
A: 때로는 그렇지만 폴더 구조가 다릅니다. 교차 호환성을 위해 파일을 재구성해야 할 수 있습니다.

**Q: 나만의 치트를 만들 수 있나요?**
A: 예, 메모리 주소와 게임 구조에 대한 지식이 있으면 가능합니다. Cheat Engine과 같은 도구는 수정할 값을 식별하는 데 도움이 될 수 있습니다.

**Q: 모드가 게임을 망가뜨리면 어떻게 하나요?**
A: 모드 디렉토리에서 모드를 비활성화하거나 제거하기만 하면 됩니다. 원본 게임 파일은 그대로 유지됩니다.
