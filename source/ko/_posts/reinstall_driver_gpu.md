## 소개

**그래픽 드라이버 클린 재설치**는 성능 문제, 게임 충돌 또는 디스플레이 오류를 해결하는 가장 효과적인 방법인 경우가 많습니다. 이 프로세스는 **DDU(Display Driver Uninstaller)**를 사용하여 새 드라이버를 설치하기 전에 이전 드라이버를 완전히 제거합니다.

이 방법은 특히 다음과 같은 경우에 효과적입니다.
- 이전 버전과 새 버전 간의 충돌 해결
- 게임 내 성능 문제 수정
- 지속적인 디스플레이 오류 제거
- 그래픽 카드 변경 후 정리

### 일반적인 과정

1. 새 드라이버와 DDU **다운로드**
2. 인터넷 **연결 해제** 및 안전 모드로 재부팅
3. DDU로 이전 드라이버 **완전히 제거**
4. **재부팅**하고 새 드라이버 설치
5. 인터넷 **재연결** 및 기능 확인

## 사전 요구 사항 및 다운로드

### 그래픽 카드 식별

시작하기 전에 GPU를 식별하십시오.

**방법 1: 장치 관리자**
1. "내 PC"에서 **마우스 오른쪽 버튼 클릭** → **속성**
2. **장치 관리자** → **디스플레이 어댑터**
3. 카드의 정확한 모델 **메모**

**방법 2: 시스템 명령**
```cmd
dxdiag
```
**"디스플레이"** 탭에서 GPU를 확인하십시오.

### 필요한 다운로드

#### NVIDIA 카드용

**🔗 [NVIDIA 공식 사이트](https://www.nvidia.co.kr/Download/index.aspx?lang=kr)**

**다운로드 절차:**
1. 카드 시리즈(RTX 40, RTX 30, GTX 16 등) **선택**
2. 정확한 모델 **선택**
3. 최신 **"Game Ready"** 드라이버 **다운로드**
4. 바탕 화면에 파일 **저장**

**NVIDIA 드라이버 유형:**
- **Game Ready Driver**: 게임에 최적화됨(권장)
- **Studio Driver**: 콘텐츠 제작에 최적화됨

#### AMD 카드용

**🔗 [AMD 공식 사이트](https://www.amd.com/ko/support)**

**다운로드 절차:**
1. "그래픽 카드" → 시리즈(RX 7000, RX 6000 등) **선택**
2. 정확한 모델 **선택**
3. 전체 **"Adrenalin"** 패키지 **다운로드**
4. 바탕 화면에 파일 **저장**

#### Display Driver Uninstaller (DDU)

**🔗 [DDU 다운로드](https://www.guru3d.com/download/display-driver-uninstaller-download/)**

**중요 정보:**
- **무료**이며 안전하게 사용 가능
- **포터블**: 설치 필요 없음
- 모든 최신 Windows 버전과 **호환**
- **정기적으로 업데이트**됨

## 1단계: 시스템 준비

### 중요 백업

시작하기 전에:

**🛡️ 필수 백업 지점:**
- **게임 프로필**(NVIDIA GeForce Experience, AMD Software 설정)
- **사용자 지정 디스플레이 설정**
- **적용된 오버클러킹**(값 기록)

### 프로그램 종료

**GPU 관련 모든 소프트웨어 닫기:**
- GeForce Experience / AMD Software
- MSI Afterburner / GPU-Z
- 게임 및 3D 애플리케이션
- 스트리밍 소프트웨어(OBS 등)

### 인터넷 연결 해제(중요)

> **🔥 중요 단계**: 안전 모드로 재부팅하기 전에 이더넷 케이블을 물리적으로 뽑거나 Wi-Fi를 비활성화하십시오.

**인터넷을 연결 해제하는 이유**
- Windows가 일반 드라이버를 자동으로 설치하는 것을 방지
- 재설치 중 충돌 방지
- DDU를 위한 깨끗한 환경 보장

## 2단계: 안전 모드 액세스

### Windows 10용

1. "설정" 열기(Windows + I)
2. "업데이트 및 보안" 클릭
3. 왼쪽 메뉴에서 "복구" 선택
4. **"고급 시작"**에서 **"지금 다시 시작"** 클릭

### Windows 11용

#### 설정을 통해

1. "설정" 열기(Windows + I)
2. 왼쪽 메뉴에서 "Windows 업데이트" 클릭
3. "고급 옵션" 선택
4. "복구" 클릭
5. **"고급 시작"**에서 **"지금 다시 시작"** 클릭

### 고급 시작 메뉴 탐색

PC가 재부팅되면 옵션이 있는 파란색 화면이 표시됩니다.

#### 탐색 순서

1. "문제 해결" 클릭
2. "고급 옵션" 클릭
3. "시작 설정" 클릭
4. "다시 시작" 클릭

#### 시작 설정 메뉴

재부팅 후 번호가 매겨진 목록이 표시됩니다.

**사용 가능한 옵션:**
```
1) 디버깅 사용
2) 부팅 로깅 사용
3) 저해상도 비디오 사용
4) 안전 모드 사용          ← 이 옵션 선택
5) 네트워킹으로 안전 모드 사용
6) 명령 프롬프트로 안전 모드 사용
7) 시스템 오류 시 자동 다시 시작 사용 안 함
8) 드라이버 서명 적용 사용 안 함
9) 맬웨어 방지 조기 실행 보호 사용 안 함
```

표준 안전 모드의 경우 **"4" 키를 누릅니다**.

## 3단계: DDU 사용

### 첫 실행

안전 모드에 들어가면:

1. DDU를 추출한 폴더로 **이동**
2. "Display Driver Uninstaller.exe"에서 **마우스 오른쪽 버튼 클릭**
3. "관리자 권한으로 실행" **선택**

### DDU 구성

#### 장치 유형 선택

**"장치 유형 선택"** 드롭다운에서:
- "GPU" **선택**

#### 제조업체 선택

**"장치 선택"** 메뉴에서:
- **NVIDIA의 경우**: "NVIDIA" 선택
- **AMD의 경우**: "AMD" 선택
- **Intel의 경우**: "INTEL" 선택(해당하는 경우)

### 제거 프로세스

#### 제거 시작

1. "정리하고 다시 시작" **클릭**
2. 메시지가 표시되면 작업 **확인**
3. DDU가 프로세스를 완료할 때까지 **대기**(2-5분)

#### 정리 진행 상황

DDU는 다음을 수행합니다.
- 모든 드라이버 파일 **제거**
- Windows 레지스트리 **정리**
- 드라이버 관련 서비스 **제거**
- 임시 폴더 **삭제**
- 자동으로 **재부팅**

**진행률 표시기:**
- 녹색 진행률 표시줄
- 실시간 상태 메시지
- 제거된 항목 목록

## 4단계: 새 드라이버 설치

### DDU 후 확인

자동 재부팅 후:

**성공적인 제거의 징후:**
- 기본 화면 해상도(1024x768 또는 유사)
- 장치 관리자에 GPU 드라이버 없음
- Windows가 기본 디스플레이 드라이버 사용

### NVIDIA 드라이버 설치

#### GeForce Game Ready Driver

1. 다운로드한 .exe 파일 **찾기**
2. **마우스 오른쪽 버튼 클릭** → "관리자 권한으로 실행"
3. 설치 마법사 **따르기**

**권장 설치 옵션:**
```
☑ 그래픽 드라이버
☑ HD 오디오 드라이버
☑ PhysX 시스템 소프트웨어
☑ GeForce Experience(선택 사항)
```

**설치 유형:**
- **Express**: 기본 설정으로 빠른 설치
- **사용자 지정**: 구성 요소의 전체 제어(권장)

#### 사용자 지정 설치(권장)

**필수 구성 요소:**
- **디스플레이 드라이버**: 주 드라이버(필수)
- **PhysX**: 게임의 고급 물리
- **NVIDIA HD 오디오**: HDMI/DisplayPort를 통한 사운드

**선택적 구성 요소:**
- **GeForce Experience**: 자동 게임 최적화

**설치 옵션:**
- **클린 설치**: ☑ (권장)
- **공장 초기화**: ☑ (설정을 0으로 재설정)

### AMD 드라이버 설치

#### AMD Adrenalin Software

1. 다운로드한 .exe 파일 **찾기**
2. **마우스 오른쪽 버튼 클릭** → "관리자 권한으로 실행"
3. 설치 유형 **선택**

**AMD 설치 유형:**
- **설치**: 전체 설치(권장)
- **최소 설치**: 드라이버만
- **공장 초기화**: 전체 재설정

#### AMD Adrenalin 구성 요소

**주요 구성 요소:**
```
☑ AMD 디스플레이 드라이버
☑ AMD 오디오 드라이버
☑ AMD Software Adrenalin Edition
☑ AMD 칩셋 소프트웨어(해당하는 경우)
```

**포함된 기능:**
- **Radeon Software**: 주 제어 인터페이스
- **ReLive**: 게임 플레이 녹화 및 스트리밍
- **Radeon Image Sharpening**: 선명도 향상

### 설치 완료

#### 최종 재부팅

1. 설치가 완료되면 PC **재부팅**
2. 인터넷 **재연결**(이더넷 케이블 또는 Wi-Fi)
3. 화면 해상도가 올바른지 **확인**

#### 기능 확인

**기본 테스트:**
- 기본 해상도 자동 감지
- 장치 관리자에 드라이버 표시
- 제어 소프트웨어 작동(GeForce Experience / AMD Software)

## 고급 문제 해결

### 일반적인 문제

#### 🚫 설치 실패

**가능한 원인:**
- 제거되지 않은 이전 드라이버의 잔여물
- 바이러스 백신이 설치 차단
- 손상된 설치 파일

**해결책:**
1. 안전 모드에서 DDU **재실행**
2. 드라이버 **재다운로드**
3. 호환성 모드에서 설치 **시도**

#### ⚠️ 설치 후 검은 화면

**긴급 조치:**
1. 안전 모드로 **재부팅**(시작 시 F8)
2. DDU를 **재실행**하여 다시 정리
3. 이전 안정 버전 **설치**
4. GPU/드라이버 호환성 **확인**

#### 💻 성능 저하

**확인:**
- **GPU 온도**(MSI Afterburner / HWMonitor)
- 작업 관리자의 **GPU 사용량**
- **Windows 전원 설정**(최고 성능)

### 성능 모니터링

**모니터링 도구:**
| 도구 | 기능 | 유용성 |
|---|---|---|
| **MSI Afterburner** | 전체 모니터링 | 온도, 사용량, FPS |
| **GPU-Z** | 상세 정보 | 사양, 센서 |
| **HWiNFO64** | 시스템 모니터링 | 고급 진단 |

### 경고 신호

**재설치가 필요한 지표:**
- 잦은 게임 충돌
- 그래픽 아티팩트
- 설명할 수 없는 성능 저하

## FAQ - 자주 묻는 질문

**Q: 모든 드라이버 업데이트에 DDU를 사용해야 합니까?**
**A:** 아니요, DDU는 문제가 있거나 주요 변경(새 GPU, 손상된 버전 등)이 있는 경우에만 필요합니다.

**Q: 안전 모드 없이 DDU를 사용할 수 있습니까?**
**A:** 강력히 권장하지 않습니다. 안전 모드는 제거를 방해하는 프로세스가 없음을 보장합니다.

**Q: DDU가 내 그래픽 카드를 감지하지 못하면 어떻게 합니까?**
**A:** DDU를 사용하기 전에 GPU가 올바르게 설치되고 장치 관리자에서 인식되는지 확인하십시오.

**Q: 게임 설정이 손실됩니까?**
**A:** 아니요, 게임 설정은 일반적으로 별도로 저장됩니다. GPU 제어판 프로필만 손실됩니다.

**Q: 전체 절차는 얼마나 걸립니까?**
**A:** 다운로드 및 재부팅을 포함하여 전체 프로세스에 30-45분이 소요됩니다.
