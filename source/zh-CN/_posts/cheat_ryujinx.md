> **⚠️ 重要提示**：在安装金手指或模组之前，请务必备份您的存档文件。某些修改可能会影响游戏稳定性。

## 简介

**Ryujinx** 支持**金手指代码**和**游戏修改（模组）**，以增强您的任天堂 Switch 游戏体验。无论您是想要无限生命、解锁所有物品，还是彻底改变游戏画面，本指南都将向您展示如何准确安装它们。

### 有什么区别？

**金手指 vs 模组：**
- **金手指**：小的代码修改（无限金钱、最大属性等）
- **模组**：完整的游戏内容修改（新纹理、角色、关卡等）

两者在 Ryujinx 中使用不同的安装方法和文件夹结构。

## 在 Ryujinx 上安装金手指

### 第 1 步：访问模组目录

#### 打开 Ryujinx 并找到您的游戏

1. **启动** Ryujinx 模拟器
2. 在您想添加金手指的游戏上**右键单击**
3. 从上下文菜单中选择**“打开模组目录”**

### 第 2 步：了解文件夹结构

#### 目录结构

当您打开模组目录时，您会看到如下路径：
```
Ryujinx\mods\contents\XXXXXXXXXXXXXXXX\
```

其中 `XXXXXXXXXXXXXXXX` 是您所选游戏的**唯一游戏 ID**（Title ID）。

**游戏 ID 示例：**
- 塞尔达传说：旷野之息：`01007EF00011E000`
- 超级马力欧 奥德赛：`0100000000010000`
- 宝可梦 剑：`0100ABF008968000`

### 第 3 步：安装金手指文件

#### 解压并放置金手指文件

1. **下载**您的金手指文件（通常为 .zip 格式）
2. **解压**存档内容
3. **创建适当的文件夹结构：**

```
Ryujinx\mods\contents\XXXXXXXXXXXXXXXX\
└── 金手指名称\
    └── cheats\
        ├── cheat1.txt
        ├── cheat2.txt
        └── other_cheats.txt
```

#### 详细安装步骤

4. **导航**到您游戏的模组目录
5. **创建一个新文件夹**，并以金手指名称命名（例如：“无限金钱”）
6. **在此文件夹内**，创建一个名为**“cheats”**的子文件夹
7. **将所有 .txt 金手指文件**放入“cheats”文件夹中

### 第 4 步：启用金手指

#### 在游戏中启用金手指

1. 在 Ryujinx 中**启动您的游戏**
2. 再次在游戏标题上**右键单击**
3. 选择**“管理金手指”**或通过菜单访问
4. **勾选**您想启用的金手指旁边的框
5. **应用**更改并在必要时重新启动游戏

## 在 Ryujinx 上安装模组

### 第 1 步：访问模组目录

#### 与金手指相同的初始过程

1. **打开 Ryujinx**
2. 在您的目标游戏上**右键单击**
3. 点击**“打开模组目录”**

这将打开：`AppData\Roaming\Ryujinx\mods\contents\<游戏 ID>\`

### 第 2 步：创建模组文件夹结构

#### 正确组织您的模组

1. 在游戏 ID 目录下**创建一个新的子文件夹**
2. **为其命名**（例如：“无限生命”、“高清纹理”、“自定义音乐”）
3. 这有助于您轻松**管理多个模组**

**结构示例：**
```
Ryujinx\mods\contents\01007EF00011E000\
├── 高清图形模组\
│   └── romfs\
├── 无限生命\
│   └── exefs\
└── 自定义音轨\
    └── romfs\
```

### 第 3 步：安装模组文件

#### 复制模组内容

1. **解压您下载的模组文件**
2. **找到**模组中的 **romfs 或 exefs 文件夹**
3. **将这些文件夹复制**到您新创建的模组目录中

**模组文件夹类型：**
- **romfs**：游戏内容修改（纹理、模型、音频）
- **exefs**：可执行文件修改（代码补丁、游戏玩法更改）

#### 完成安装

4. **导航**到您的模组提取文件夹
5. **复制整个 romfs 或 exefs 文件夹**
6. **粘贴**到您创建的模组目录中

### 第 4 步：启用模组

#### 启用已安装的模组

1. **启动 Ryujinx**
2. 在您的游戏上**右键单击**
3. 选择**“管理模组”**
4. 根据您的偏好**启用/禁用**模组
5. **启动游戏**以查看更改

3. 在添加其他模组之前**检查冲突**
4. **记录**有效的组合

## 常见问题故障排除

### 金手指不工作

#### 常见问题及解决方案

**🚫 未检测到金手指文件：**
- 检查文件夹结构：`游戏_ID\金手指_名称\cheats\*.txt`
- 检查文件扩展名（.txt，而不是 .txt.txt）
- 确保文件夹名称中没有多余的空格

**⚠️ 金手指导致崩溃：**
- 禁用所有金手指并逐一测试
- 检查金手指与游戏版本的兼容性
- 使用来自可靠来源的金手指
- 确保金手指版本与游戏版本匹配

### 模组未加载

#### 调试模组问题

**📁 文件夹结构不正确：**
```
❌ 错误：
Ryujinx\mods\contents\游戏_ID\romfs\...

✅ 正确：
Ryujinx\mods\contents\游戏_ID\模组_名称\romfs\...
```

**🎮 游戏特定问题：**
- 检查模组与您的游戏版本的兼容性
- 检查模组是否需要特定的固件版本
- 禁用模组进行测试以确认其是问题原因

## FAQ - 常见问题

**问：我可以同时使用金手指和模组吗？**
答：可以，但要仔细测试，因为某些组合可能会导致冲突或崩溃。

**问：模组会影响游戏性能吗？**
答：这取决于模组。图形模组通常需要更多资源，而简单的金手指影响最小。

**问：Ryujinx 模组与 Yuzu 兼容吗？**
答：有时兼容，但文件夹结构不同。您可能需要重新组织文件以实现交叉兼容性。

**问：我可以创建自己的金手指吗？**
答：可以，需要了解内存地址和游戏结构。像 Cheat Engine 这样的工具可以帮助识别要修改的值。

**问：如果模组破坏了我的游戏怎么办？**
答：只需从模组目录中禁用或删除该模组即可。您的原始游戏文件将保持不变。
