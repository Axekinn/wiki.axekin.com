## 🎮 Citra 소개

**Citra**는 가장 진보된 오픈 소스 Nintendo 3DS 에뮬레이터입니다. 2013년부터 개발되어 PC에서 3DS 게임을 상당한 그래픽 향상과 거의 완벽한 호환성으로 플레이할 수 있습니다. Azahar는 그 포크입니다.

## 🔧 1단계: Citra 다운로드

### 옵션 1: 공식 Azahar
```bash
# 공식 사이트
https://github.com/azahar-emu/azahar/releases
```

1. **"Download for Windows"**를 클릭합니다.
2. **`azahar-XXXX-rc1-windows-msvc-installer.exe`**를 다운로드합니다.
3. 전용 폴더(예: `Documents\Azahar` 또는 C 드라이브가 아닌 다른 드라이브의 루트)에 **압축을 풉니다**.

## 게임 다운로드

### 1단계: 게임 라이브러리 액세스

**[🎮 Axekin Games - Nintendo 3DS](https://www.axekin.com/games?platform=3ds)**로 이동하여 Nintendo 3DS 게임 컬렉션에 액세스합니다.

### 2단계: 게임 다운로드

1. 목록에서 원하는 게임을 **검색**합니다.
2. 다운로드 버튼을 **클릭**합니다.
3. 다운로드가 완료될 때까지 **기다립니다**.

### 3단계: 파일 압축 풀기

다운로드가 완료되면 다음 소프트웨어 중 하나로 아카이브의 **압축을 풉니다**.

| 소프트웨어 | 다운로드 링크 | 호환성 |
|----------|---------------|---------------|
| **7-Zip** | [7-Zip 다운로드](https://www.7-zip.org/) | Windows/Linux |
| **WinRAR** | [WinRAR 다운로드](https://www.win-rar.com/) | Windows |

> **💡 팁**: 7-Zip은 무료 오픈 소스이며 WinRAR은 평가판 기간 이후 라이선스가 필요합니다.

**결과**: 3DS에서 사용할 준비가 된 게임 파일을 얻게 됩니다! 🎯

## 📁 2단계: 폴더 구조

파일을 정리하기 위해 이 폴더 구조를 만듭니다.

라이브러리를 최적으로 구성하기 위한 권장 트리 구조는 다음과 같습니다.

````
3DS 게임/
├── 동물의 숲 - 해피 홈 디자이너/
│   ├── BASE/                # 🎮 기본 게임 (메인 파일)
│   ├── UPDATE/              # 🔄 게임 업데이트 (권장)
│   └── DLC/                 # 📦 다운로드 가능한 콘텐츠 (선택 사항)
├── 데드 오어 얼라이브 - 디멘션즈/
│   ├── BASE/                # 🎮 기본 게임
│   ├── UPDATE/              # 🔄 게임 업데이트
│   └── DLC/                 # 📦 추가 DLC
├── 벤 10 - 갤럭틱 레이싱/
│   ├── BASE/
│   ├── UPDATE/
│   └── DLC/
└── 캡틴 아메리카 - 슈퍼 솔저/
    ├── BASE/
    └── UPDATE/              # (이 게임에는 DLC가 없습니다)
````


### 콘텐츠 유형


- **BASE** 📁: 다운로드한 메인 게임 파일
- **UPDATE** 📁: 업데이트 및 패치 (권장)
- **DLC** 📁: 추가 유료 콘텐츠 (선택 사항이지만 권장)


✅ **자동 감지**: 게임이 에뮬레이터에 자동으로 추가됩니다.  
✅ **간소화된 관리**: 각 ROM을 수동으로 추가할 필요가 없습니다.  
✅ **명확한 구성**: 게임별로 어떤 콘텐츠가 있는지 쉽게 확인할 수 있습니다.  
✅ **쉬운 유지 관리**: 콘텐츠 추가/제거가 간단합니다.

> **📝 중요 참고**: 게임에 업데이트나 DLC가 없는 경우 해당 폴더를 만들 필요가 없습니다. 소유한 콘텐츠에 대해서만 폴더를 만드세요.

### 결과

이 구조가 마련되면 에뮬레이터가 "3DS 게임" 폴더의 모든 게임을 자동으로 감지하고 라이브러리에 깔끔하게 표시합니다! 🎯

## 💡 휴대용 구성 (권장)


**첫 실행 전**에 Azahar 설치 루트에 `user`라는 폴더를 만듭니다. 이렇게 하면 설치가 완전히 휴대용이 됩니다.

### 휴대용 모드의 장점

| 측면 | 일반 모드 | 휴대용 모드 |
|--------|-------------|---------------|
| **구성** | 시스템에 저장 | 폴더 내에 저장 |
| **저장** | 컴퓨터 전체에 분산 | 한 폴더에 중앙 집중화 |
| **휴대성** | ❌ 시스템에 연결됨 | ✅ 쉽게 이동 가능 |
| **백업** | 찾기 어려움 | 백업하기 쉬움 |

### 휴대용 폴더 생성 후 구조

```
📁 Azahar/
├── 🎮 Azahar.exe
├── 📁 user/              # ← 이 폴더를 만드세요!
│   ├── 📁 bis/
│   ├── 📁 games/
│   ├── 📁 profiles/
│   └── 📁 system/
└── 📄 기타 파일...
```


**모든 데이터**(구성, 저장, 프로필)는 `user` 폴더 내에 저장되어 관리 및 백업이 훨씬 쉬워집니다!

### 생성 단계

1. Azahar 설치 폴더로 이동합니다.
2. 정확히 `user`라는 새 폴더를 만듭니다.
3. Azahar를 실행하면 휴대용 모드를 자동으로 감지합니다.

> **📝 참고**: 이 작업은 **첫 실행 전**에 수행해야 효과가 있습니다!

**상단 화면:**

- **위치 X/Y**: `80px, 500px` (약간 중앙 정렬)

## ⚙️ 단계 🌐 멀티플레이어 (빠른 설정)

로컬 구성을 편집하여 Citra/Azahar를 외부 멀티플레이어 서비스에 연결할 수 있습니다. 경고: 구성 파일을 편집하기 전에 Citra/Azahar를 완전히 닫으십시오. 그렇지 않으면 변경 사항이 덮어쓰여집니다.

1. 구성 폴더를 엽니다.

   - 표준 Windows 설치의 경우: `C:\\Users\\<사용자 이름>\\AppData\\Roaming\\citra\\config\\`
   - 휴대용 설치의 경우: Azahar/Citra 루트의 `user` 폴더를 엽니다.

2. `qt-config.ini` 파일을 (메모장 또는 텍스트 편집기로) 편집하고 다음 두 줄을 변경/추가합니다.

```
web_api_url\\default=false
web_api_url=http://88.198.47.46:5000/
```

3. 편집기를 저장하고 닫은 다음 Citra/Azahar를 다시 시작합니다.

전용 방 호스팅 (Windows `host.bat` 예제)

전용 배치 파일을 통해 호스팅하려면 `host.bat`를 만들거나 편집하고 아래 예제를 붙여넣습니다. 방 이름과 타이틀 ID를 변경할 수 있습니다.

```
citra-room ^
  --room-name "방 이름" ^
  --room-description "모든 게임" ^
  --preferred-game "슈퍼 스매시브라더스" ^
  --preferred-game-id "00040000000EDF00" ^
  --port 24872 ^
  --max_members 16 ^
  --token "aaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" ^
  --enable-citra-mods ^
  --web-api-url http://88.198.47.46:5000 ^
  --ban-list-file "여기에 차단 목록 파일 경로 삽입"
pause
```

참고:

- 필요한 경우 `--token`을 실제 토큰으로 교체하십시오.
- `--preferred-game-id`는 게임의 타이틀 ID(16진수 16자리)에 해당합니다. 모든 게임을 허용하려면 비워 둡니다.
- `--web-api-url`은 API 서버를 가리켜야 합니다(여기서는 `http://88.198.47.46:5000`).

이 단계를 통해 Citra/Azahar는 지정된 멀티플레이어 서비스를 사용하고 `host.bat`를 사용하여 컴퓨터에서 방을 호스팅할 수 있습니다.

## ❓ FAQ - 자주 묻는 질문

### 일반적인 질문

**Q: Azahar가 모든 3DS 게임을 실행할 수 있나요?**  
**A:** 약 90%의 3DS 게임이 양호에서 완벽하게 실행됩니다. 특정 타이틀에 대한 호환성 데이터베이스를 확인하세요.

**Q: 입체 3D가 작동하나요?**  
**A:** 예, 3D 지원 디스플레이 또는 VR용 나란히 모드를 사용하면 작동합니다.

**Q: 실제 3DS 저장 파일을 사용할 수 있나요?**  
**A:** 예 — Checkpoint 또는 JKSM을 사용하면 양방향으로 저장 파일을 전송할 수 있습니다.

### 기술적인 질문

**Q: 일부 게임이 느린 이유는 무엇인가요?**  
**A:** 3DS 게임은 특정 하드웨어에 최적화되어 있습니다. 에뮬레이션은 특히 복잡한 타이틀의 경우 더 많은 리소스를 필요로 합니다.

**Q: OpenGL 또는 Vulkan?**  
**A:** OpenGL이 더 안정적이고 호환성이 좋습니다. Vulkan은 최신 GPU에서 더 나은 성능을 제공할 수 있습니다.

**Q: New 3DS 모드를 활성화하면 모든 것이 향상되나요?**  
**A:** 주로 New 3DS 기능을 기본적으로 사용하는 게임(제노블레이드, 몬스터 헌터 등) 및 특정 홈브류에 해당됩니다.

### 일반적인 문제

**Q: "시스템 지역을 확인할 수 없습니다"?**  
**A:** 시스템 파일(aes_keys.txt)을 설치하고 설정에서 지역을 구성하세요.

**Q: HD 텍스처가 표시되지 않나요?**  
**A:** "사용자 지정 텍스처"가 활성화되어 있고 타이틀 ID가 폴더와 일치하는지 확인하세요.

**Q: Android에서 성능을 향상시키는 방법은 무엇인가요?**  
**A:** Citra MMJ를 사용하고 해상도를 낮추고 필요한 경우 프레임 건너뛰기를 활성화하세요.

고화질로 3DS 게임을 즐기세요! 🎮✨
