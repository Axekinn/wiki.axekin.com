# Полное руководство: Установка пакетов текстур на эмулятор Azahar

## Введение

Эмулятор **Azahar**, основанный на Citra, позволяет значительно улучшить визуальное восприятие игр Nintendo 3DS с помощью пакетов текстур. Эти пакеты заменяют оригинальные текстуры на версии высокого разрешения, обеспечивая заметно улучшенную графику.

## Шаг 1: Подготовка эмулятора

### 1.1 Включение поддержки пакетов текстур
1. В Azahar перейдите в **Эмуляция** → **Настроить**
2. Выберите вкладку **Графика**
3. Установите флажок **Использовать пользовательские текстуры**
4. Установите флажок **Предварительно загружать пользовательские текстуры** (необязательно, улучшает производительность)
5. Нажмите **OK**

## Шаг 2: Получение пакетов текстур

### 2.1 Рекомендуемые источники
- **Сборник пакетов текстур**: https://vikingfile.com/folder/-ryt9sVCLmtmdMo4zGrv5NfgK2MeJHIc_Y_2uVDYaYk (Временная ссылка, скоро будет обновлена)

### 2.2 Проверка совместимости
Убедитесь, что пакет текстур соответствует:
- Региону вашего ROM (EUR, USA, JPN)
- Версии игры
- Совместимости с Azahar/Citra

## Шаг 3: Установка пакета текстур

### 3.1 Доступ к папке текстур игры
1. В списке игр Azahar **щелкните правой кнопкой мыши** по нужной игре
2. Выберите **Открыть** в контекстном меню
3. Нажмите на **Расположение пользовательских текстур**
4. Автоматически откроется папка с правильным ID игры (например, 0004000000055D00)

### 3.2 Установка пакета текстур
1. **Загрузите** пакет текстур (обычно это файл .zip или .7z)
2. **Извлеките** содержимое прямо в открытую папку
3. **Убедитесь**, что файлы текстур (.png) правильно размещены в этой папке

### 3.3 Окончательная структура папки
После извлечения ваша папка должна содержать:

```
0004000000055D00/
├── tex1_0x12345678_14.png
├── tex1_0x87654321_14.png
└── ... (другие файлы)
```

### Правильная структура папки

```
textures/
└── 0004000000055D00/
    ├── tex1_0x12345678_14.png
    ├── tex1_0x87654321_14.png
    └── ... (другие файлы текстур)
```

### Организация пакетов текстур
Создайте организованную структуру:
```
textures/
├── Pokemon_X_Y/
├── Mario_Kart_7/
├── Zelda_ALBW/
└── backup/
```

## Советы и лучшие практики

### Производительность
- **Рекомендуется SSD** для более быстрой загрузки
- **16 ГБ ОЗУ** для предварительной загрузки без влияния на производительность
- **Современный ГП** для высоких разрешений

### Совместимость
- Всегда тестируйте с резервной копией вашего прогресса

### Обслуживание
- Регулярно обновляйте Azahar
- Проверяйте наличие новых версий пакетов текстур

## Заключение

Установка пакетов текстур на Azahar полностью преображает визуальное восприятие игр Nintendo 3DS. Следуя этому пошаговому руководству, вы сможете наслаждаться любимыми играми с улучшенной графикой.

Не стесняйтесь экспериментировать с различными пакетами текстур и конфигурациями, чтобы найти идеальную настройку для вашей системы.
