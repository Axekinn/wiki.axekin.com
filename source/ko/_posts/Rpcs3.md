## 🎮 RPCS3 소개

**RPCS3**는 세계에서 가장 진보된 오픈 소스 PlayStation 3 에뮬레이터입니다. C++로 작성되었으며, PS3의 Cell 아키텍처를 에뮬레이션하기 위해 해석과 재컴파일을 결합한 하이브리드 접근 방식을 사용합니다. 수년간의 개발 끝에 현재 PS3 게임의 60% 이상과 인상적인 호환성을 제공합니다.

> ⚠️ **법적 고지**: 이 에뮬레이터는 합법적으로 소유한 게임만 플레이하기 위한 것입니다. 백업 및 연구 목적의 에뮬레이션은 합법이지만 불법 복제는 그렇지 않습니다.

## 📋 시스템 요구 사항

### 최소 구성
- **OS**: Windows 10 (64비트) / Linux / FreeBSD
- **CPU**: Intel Core i5-4430 / AMD FX-6300 (4코어, 3.0GHz)
- **RAM**: 8GB
- **GPU**: DirectX 12 호환 / Vulkan 1.1 호환
- **저장 공간**: 10GB 여유 공간 + 게임 공간

### 권장 구성
- **CPU**: Intel Core i7-8700 / AMD Ryzen 7 2700 (8코어, 3.2+ GHz)
- **RAM**: 16GB
- **GPU**: NVIDIA GTX 1060 / AMD RX 580 이상
- **저장 공간**: 50GB 이상의 여유 공간이 있는 SSD

### 최적 구성
- **CPU**: Intel Core i9-12900K / AMD Ryzen 9 5900X (12코어 이상, 3.5+ GHz)
- **RAM**: 32GB DDR4-3200+
- **GPU**: NVIDIA RTX 3070 / AMD RX 6700 XT 이상
- **저장 공간**: 100GB 이상의 여유 공간이 있는 NVMe SSD

> 💡 **중요 사항**: RPCS3는 CPU 요구 사항이 매우 높습니다. 많은 코어를 가진 강력한 프로세서가 필수적입니다.

## 🔧 1단계: RPCS3 다운로드

### 공식 버전 (권장)
```bash
# 공식 웹사이트
https://rpcs3.net/download
```

1. **"Windows용 다운로드"**를 클릭합니다.
2. `rpcs3-v0.0.XX-windows_x64.7z`를 **다운로드**합니다.
3. 7-Zip을 사용하여 전용 폴더(예: `C:\RPCS3`)에 **압축을 풉니다**.

### 개발 빌드
```bash
# 최신 개선 사항이 포함된 버전
https://github.com/RPCS3/rpcs3-binaries-win/releases

장점:
- 최근 버그 수정
- 새로운 최적화
- 향상된 호환성

위험:
- 잠재적으로 불안정
- 회귀 가능성
```

## 📁 2단계: 폴더 구조

파일을 정리하기 위해 이 폴더 구조를 만듭니다.

라이브러리를 최적으로 구성하기 위한 권장 트리 구조는 다음과 같습니다.

````
PS3 게임/
├── GTA V/
│   ├── BASE/                # 🎮 기본 게임 (메인 파일)
│   ├── UPDATE/              # 🔄 게임 업데이트 (권장)
│   └── DLC/                 # 📦 다운로드 가능한 콘텐츠 (선택 사항)
├── TLOU/
│   ├── BASE/                # 🎮 기본 게임
│   ├── UPDATE/              # 🔄 게임 업데이트
│   └── DLC/                 # 📦 추가 DLC
├── BOII/
│   ├── BASE/
│   ├── UPDATE/
│   └── DLC/
└── CoD BO1/
    ├── BASE/
    └── UPDATE/              # (이 게임에는 DLC가 없습니다 (예시입니다))
````

### 콘텐츠 유형


- **BASE** 📁: 다운로드하는 메인 게임
- **UPDATE** 📁: 업데이트 및 패치 (권장)
- **DLC** 📁: 추가 유료 콘텐츠 (선택 사항이지만 권장)


✅ **자동 감지**: 게임이 에뮬레이터에 자동으로 추가됩니다.  
✅ **간소화된 관리**: 각 ROM을 수동으로 선택할 필요가 없습니다.  
✅ **명확한 구성**: 각 게임에 어떤 콘텐츠가 있는지 쉽게 확인할 수 있습니다.  
✅ **쉬운 유지 관리**: 콘텐츠 추가/제거가 간소화됩니다.

> **📝 중요 사항**: 게임에 업데이트나 DLC가 없는 경우 해당 폴더를 만들 필요가 없습니다. 실제로 가지고 있는 콘텐츠에 대해서만 폴더를 만드십시오.

### 결과

이 구조가 마련되면 에뮬레이터가 "PS3 게임" 폴더에 있는 모든 게임을 자동으로 감지하고 라이브러리에 올바르게 구성합니다! 🎯

## ⚙️ 3단계: PS3 펌웨어 설치

### 공식 펌웨어 다운로드

> ⚠️ **필수**: RPCS3를 실행하려면 Sony 공식 PS3 펌웨어가 필요합니다.

```bash
# Sony 공식 사이트
https://www.playstation.com/ko-kr/support/hardware/ps3/system-software/

1. 최신 버전(PS3UPDAT.PUP)을 다운로드합니다.
2. RPCS3/firmware/에 배치합니다.
```

### RPCS3에 설치

1. `rpcs3.exe`를 **실행**합니다.
2. 첫 실행 시 사용 약관에 **동의**합니다.
3. `파일 → 펌웨어 설치`
4. `PS3UPDAT.PUP` 파일을 **선택**합니다.
5. 전체 설치가 완료될 때까지 **기다립니다**(1분).
6. RPCS3를 **다시 시작**합니다.

## ⚙️ 4단계: 최적의 CPU 구성

### CPU 설정
```
구성 → CPU
```

**권장 구성:**
- **PPU 디코더**: Recompiler (LLVM) [가장 빠름, 모든 게임에 권장]
- **SPU 디코더**: Recompiler (LLVM) [가장 빠름, 모든 게임에 권장]
- **추가 설정**:
  - **스레드 스케줄러 활성화**: ✅ 활성화 (성능 향상)
  - **SPU 루프 감지 활성화**: ✅ 활성화 (무한 루프 방지)
  - **SPU 캐시 활성화**: ✅ 활성화 (로딩 시간 개선)
  - **SPU 블록 크기 활성화**: Safe (호환성을 위해 권장)

### 고급 CPU 옵션
```
구성 → CPU → 추가 설정
```

- **SPU 블록 크기**: Safe (대부분의 게임에 권장)
- **선호하는 SPU 스레드**: 자동 (RPCS3가 자동으로 최적화하도록 함)
- **정확한 GETLLAR 활성화**: ❌ 비활성화 (특정 게임에서 요구하지 않는 한)
- **정확한 PUTLLUC 활성화**: ❌ 비활성화 (특정 게임에서 요구하지 않는 한)
- **정확한 RSX 예약 액세스 활성화**: ❌ 비활성화 (특정 게임에서 요구하지 않는 한)
- **SPU 확인**: ❌ 비활성화 (디버깅용으로만 활성화)
- **SPU 캐시**: ✅ 활성화 (성능을 크게 향상)
- **스레드 스케줄러 활성화**: ✅ 활성화 (더 나은 스레드 관리)

## ⚙️ 5단계: 최적의 GPU 구성

### 그래픽 설정
```
구성 → GPU
```

**권장 구성:**
- **렌더러**: Vulkan (최고의 성능 및 호환성)
- **그래픽 장치**: 전용 GPU 선택
- **화면 비율**: 16:9 (또는 원본 화면 비율의 경우 자동)
- **프레임 제한**: 끄기 (게임이 의도한 속도로 실행되도록 함)
- **ZCULL 정확도**: 완화 (성능과 정확도 간의 균형)

### 해상도 및 렌더링
```
구성 → GPU → 해상도
```

**최적 설정:**
- **해상도**: 1280x720 (네이티브 PS3 해상도)
- **해상도 스케일**: 100% (강력한 GPU가 있는 경우에만 증가)
- **해상도 스케일 임계값**: 16x16 (기본값)
- **안티앨리어싱**: 비활성화 (여유가 있는 경우에만 활성화)
- **비등방성 필터**: 자동 (성능이 허용하는 경우 16x)

### 고급 그래픽 옵션
```
구성 → GPU → 고급
```

**성능 구성:**
- **기본 셰이더 유형**: Vertex (권장)
- **셰이더 모드**: 비동기 (더 나은 성능)
- **셰이더 정밀도**: 자동 (RPCS3가 결정하도록 함)
- **색상 버퍼 쓰기**: ❌ 비활성화 (게임에서 요구하는 경우에만 활성화)
- **깊이 버퍼 쓰기**: ❌ 비활성화 (게임에서 요구하는 경우에만 활성화)
- **색상 버퍼 읽기**: ❌ 비활성화 (게임에서 요구하는 경우에만 활성화)
- **깊이 버퍼 읽기**: ❌ 비활성화 (게임에서 요구하는 경우에만 활성화)
- **VSync**: ❌ 비활성화 (입력 지연 감소)
- **GPU 텍스처 스케일링**: ❌ 비활성화 (필요한 경우에만 활성화)
- **디스플레이 영역으로 늘리기**: ❌ 비활성화
- **CPU 블릿 강제**: ❌ 비활성화 (더 나은 성능을 위해 GPU 사용)
- **ZCull 오클루전 쿼리 비활성화**: ❌ 비활성화
- **정점 캐시 비활성화**: ❌ 비활성화
- **FIFO 재정렬 비활성화**: ❌ 비활성화
- **프레임 건너뛰기 활성화**: ❌ 비활성화
- **디스크 셰이더 캐시 비활성화**: ❌ 비활성화 (캐시 활성화 유지)
- **Vulkan 메모리 할당자 비활성화**: ❌ 비활성화
- **GPU 렌더 스레드 사용**: ✅ 활성화 (성능 향상)

### 셰이더 로딩
```
구성 → GPU → 셰이더 로딩
```

- **셰이더 컴파일 힌트**: 비동기 사용 (끊김 감소)
- **셰이더 로딩 대화 상자**: 표시 (처음 로딩 시 유용)
- **PPU 재컴파일러 캐시 생성 전 경고**: ✅ 활성화
- **SPU 재컴파일러 캐시 생성 전 경고**: ✅ 활성화

## ⚙️ 6단계: 오디오 구성

### 오디오 설정
```
구성 → 오디오
```

**권장 구성:**
- **오디오 출력**: Cubeb (크로스 플랫폼, 권장)
- **오디오 설정**:
  - **파일로 덤프**: ❌ 비활성화
  - **16비트로 변환**: ❌ 비활성화
  - **오디오 버퍼 지속 시간**: 100ms (기본값, 필요한 경우 조정)
  - **버퍼링 활성화**: ✅ 활성화
  - **오디오 백엔드**: Cubeb (가장 안정적)
- **오디오 장치**: @default (시스템 기본값 사용)
- **마스터 볼륨**: 100%
- **오디오 렌더러**: XAudio2 (Windows) / ALSA (Linux)

### 오디오 품질
```
구성 → 오디오 → 고급
```

- **오디오 버퍼**: 0.1초 (100ms - 좋은 균형)
- **오디오 백엔드**: Cubeb (안정성을 위해 권장)
- **시간 스트레칭 활성화**: ❌ 비활성화 (오디오 동기화 문제가 없는 한)

## ⚙️ 7단계: 입/출력 구성

### 컨트롤러 설정
```
구성 → 입/출력
```

**권장 구성:**
- **키보드 핸들러**: Null (키보드를 컨트롤러로 사용하지 않는 한)

- **카메라 핸들러**: Null (카메라 게임을 사용하지 않는 한)

- **음악 핸들러**: Null
- **패드 핸들러**: 
  - **XInput** (Xbox 컨트롤러용)
  - **DualShock 4** (PS4 컨트롤러용)
  - **evdev** (Linux 일반 컨트롤러)

## ⚙️ 8단계: 고급 시스템 구성

### 시스템 설정
```
구성 → 시스템
```

**권장 구성:**
- **시스템 언어**: 시스템 기본값 (또는 선호하는 언어)
- **키보드 유형**: 영어 키보드 (또는 레이아웃)
- **Enter 버튼 할당**: 십자 (일본어) / 원 (서양)
- **콘솔 언어**: 영어 (미국) 또는 해당 지역
- **콘솔 지역**: 자동 (또는 특정 지역)

### 네트워크 설정
```
구성 → 네트워크
```

**네트워크 구성:**
- **네트워크 상태**: 연결됨
- **네트워크 인터페이스**: 기본값
- **PSN 상태**: RPCN (홈브류 온라인 기능용)
- **DNS**: 자동 (필요한 경우 사용자 지정 DNS)

### 고급 시스템 옵션
```
구성 → 고급
```

**시스템 조정:**
- **디버그 콘솔 모드**: ❌ 비활성화 (디버깅하지 않는 한)
- **정확한 xfloat**: ❌ 비활성화 (필요한 경우에만 활성화)
- **근사 xfloat**: ✅ 활성화 (더 나은 성능)
- **완화된 xfloat**: ❌ 비활성화
- **CPU 블릿 강제**: ❌ 비활성화
- **네이티브 float16 지원 비활성화**: ❌ 비활성화
- **정확한 GETLLAR**: ❌ 비활성화 (필요한 경우에만 활성화)
- **정확한 PUTLLUC**: ❌ 비활성화 (필요한 경우에만 활성화)
- **정확한 RSX 예약 액세스**: ❌ 비활성화 (필요한 경우에만 활성화)
- **절전 타이머 정확도**: 호스트와 동일 (권장)

## ❓ FAQ - 자주 묻는 질문

### 일반적인 질문

**Q: RPCS3는 모든 PS3 게임을 실행할 수 있습니까?**
**A:** 약 65%의 게임이 플레이 가능하며 90%가 시작됩니다. 호환성은 지속적으로 향상되고 있습니다.

**Q: RPCS3를 사용하려면 PS3가 필요합니까?**
**A:** 펌웨어는 필요하지 않습니다(합법적으로 다운로드 가능). 게임을 합법적으로 덤프하려면 필요합니다.

**Q: RPCS3는 왜 그렇게 까다롭습니까?**
**A:** PS3의 Cell 아키텍처는 에뮬레이션하기 복잡합니다. 1개의 PPU + 6개의 SPU는 많은 CPU 성능을 필요로 합니다.

### 기술적인 질문

**Q: Vulkan 또는 D3D12?**
**A:** 일반적으로 Vulkan이 더 성능이 좋습니다. D3D12는 일부 시스템에서 더 안정적입니다. 둘 다 테스트해 보세요.

**Q: 최소 RAM은 얼마입니까?**
**A:** 최소 8GB, 권장 16GB, 가장 무거운 게임의 경우 32GB.

**Q: 제 CPU는 충분합니까?**
**A:** 최소 6코어 권장. 8코어 이상이 최적입니다. 단일 스레드 성능이 중요합니다.

### 일반적인 문제

**Q: 강력한 PC에도 불구하고 게임이 지연됩니까?**
**A:** 일부 게임에서는 정상입니다. RPCS3는 복잡한 아키텍처를 에뮬레이션합니다. 사용 가능한 패치를 확인하십시오.

**Q: 성능을 향상시키는 방법은 무엇입니까?**
**A:** SPU 캐시 + 사전 컴파일, 캐시용 SSD, 백그라운드 앱 닫기, 적당한 CPU 오버클럭.

**Q: 오디오가 동기화되지 않습니까?**
**A:** 오디오 버퍼를 늘리고, 오디오 백엔드를 변경하고, 프레임 페이싱을 확인하십시오.
