> スムーズで信頼性の高い設定で、RPCNを介してPS3ゲームをオンラインでプレイしましょう。クイックチェックリストに従い、必要に応じてさらに詳しく掘り下げてください。

> 📝 著者ノート  
> RPCS3は、マルチプレイヤーゲームの設定がこれまでになく簡単になりました。技術に詳しい方のために、非常によく説明されているRPCS3のチュートリアルを含めました。

## ⚡ クイックスタートチェックリスト

- 最新のRPCS3ビルドに更新する
- RPCNアカウントを作成して確認する
- ゲームごと：ネットワーク = 接続済み、PSN = RPCN
- DNS 8.8.8.8（ゲームが別のものを必要としない限り）
- PC/ルーターでUPnPを有効にするか、UDP 3658をポートフォワーディングする
- RPCNウィンドウでアカウントをテストする

## ℹ️ RPCNとは？

RPCNは、RPCS3がサポートされているゲームでオンライン機能を有効にするために使用する、コミュニティ製のPSNの代替品です。

> ⚠️ 免責事項  
> 99%の場合、ファイアウォール、NAT/UPnP、またはポートフォワーディングの設定を変更する必要はありません。RPCS3 + RPCNは、ほとんどのネットワークでそのまま動作するはずです。以下の手順に従っても他のプレイヤーとのオンライン接続に失敗した場合にのみ、これらの設定を触ってください。

## ✅ 要件

| 項目 | 推奨事項 | 注意 |
|---|---|---|
| RPCS3 | 最新ビルド | 自動アップデーターまたは新規ダウンロード |
| ポート | UDP 3658 | UPnPまたは手動ポートフォワーディング |
| NAT | オープン/タイプA | ダブルNAT/CGNATを避ける |
| ファイアウォール | RPCS3を許可 | OSおよびセキュリティスイート |

---

## 🔧 グローバル設定

1. RPCS3を開く → 設定 → **RPCN**
2. **アカウント**をクリック
3. サーバーを選択：**公式RPCNサーバー**
4. **アカウントを作成**をクリック
5. **ユーザー名**を設定 → OK
6. **パスワード**を2回入力 → OK
7. **メールアドレス**を2回入力 → OK
8. 概要を確認 → **はい**
9. 成功メッセージ → 「RPCN: ユーザー名」ウィンドウが表示されます
10. RPCN**トークン**のメールボックスを確認 → **トークン**に貼り付け → OK
11. **アカウントをテスト**をクリック

> 💡 プロのヒント  
> トークンは非公開にしてください。紛失した場合は、「アカウントの編集」で「トークンを再送信」を使用してください。

---

## 🎯 ゲーム固有の設定

1. ゲームリストで、タイトルを右クリック → **カスタム設定を作成**
2. **ネットワーク**に移動
3. **ネットワークステータス**：接続済み
4. **PSNステータス**：RPCN
5. **DNS**：`8.8.8.8`（ゲームが別のものを必要としない限り）
6. ゲームがカスタムサーバーを必要とする場合は、コミュニティから提供されたIP/ホストスイッチを入力します

> 💡 ゲームごとの上書き  
> これらの設定はこのゲームにのみ適用され、グローバル設定は変更されません。

## 🌐 接続の基本

- RPCS3でUPnPを有効にする：設定 → ネットワーク → **UPnPを有効にする**
- ルーターでUPnPを有効にする（マニュアルはモデルによって異なります）
- または、ポート**3658/UDP**をPCのローカルIPに転送します

<details>
  <summary>ポートフォワーディングの注意事項を表示</summary>

- 転送：UDP 3658 → PCのIP（DHCPで予約）  
- ダブルNATを避ける（必要に応じてISPモデムをブリッジモードにする）  
- VPNは着信UDPをブロックする可能性があります。VPNでポートフォワーディングを設定するか、プレイ中は無効にしてください
</details>

> 🚧 NATとISPの制約  
> 厳格なNATまたはCGNATは、ピア接続を破壊する可能性があります。ISPにパブリックIPv4を要求するか、ポートフォワーディングをサポートするVPNを使用してください。

---

## 🛠️ トラブルシューティング

### 他のプレイヤーに接続できない
- RPCS3とルーターでUPnPが有効になっている
- またはUDP 3658をPCに転送する
- ファイアウォール/セキュリティスイートでRPCS3を許可する
- VPNを無効にするか再設定する
- RPCN → アカウント → **アカウントをテスト**

<details>
  <summary>高度なチェック</summary>

- PCのIPが変わっていないことを確認する（DHCP予約）  
- ダブルNATを検出する（モデムとルーターの両方がルーティングしている）  
- ルーターのNATテストはオープン/タイプAを対象とすべきです
</details>

## 🔐 RPCNアカウントの回復

### RPCNトークンを再送信する
1. RPCS3 → 設定 → RPCN → **アカウント**
2. **アカウントを編集** → **トークンを再送信**
3. 確認 → OK → 「RPCNのトークン」というメールを確認
4. **トークン**フィールドにトークンを貼り付け → OK → **保存**

### RPCNパスワードをリセットする
1. RPCS3 → 設定 → RPCN → **アカウント**
2. **アカウントを編集** → **パスワードを変更**
3. ユーザー名を確認 → OK
4. 「パスワードリセットトークンを既にお持ちですか？」と尋ねられたら → **いいえ**
5. 登録メールアドレスを2回入力 → OK
6. 「RPCNのパスワードリセットコード」というメールを確認
7. トークンを貼り付け → OK
8. 新しいパスワードを2回入力 → OK
9. アカウントの編集に戻り → **パスワードを設定** → 新しいパスワードを入力 → OK → **保存**
10. RPCN：**アカウントをテスト**

---

### 良い習慣
- 強力でユニークなパスワードを使用する  
- トークンを共有しない  
- 最新のRPCN修正のためにRPCS3を最新の状態に保つ  

> ゲームのライセンスとオンライン利用規約を尊重してください。ゲームを購入して開発者をサポートしてください。

🎯 結果：RPCNを介してオンラインでプレイする準備ができました！
