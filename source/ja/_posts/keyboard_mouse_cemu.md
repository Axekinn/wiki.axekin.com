> **⚠️ 重要事項**: 可能ではありますが、Cemuでのキーボードとマウスの体験は、コントローラーを使った場合ほど快適ではありません。この解決策は、互換性のあるゲームパッドを持っていないプレイヤーに適しています。

## はじめに

**Cemu**エミュレータで**Wii U**のゲームを**キーボードとマウス**でプレイするには、コントローラーをシミュレートするためのサードパーティ製ツールを使用する必要があります。このガイドでは、**Mouse2Joystick**と**ViGEmBus**を使った設定方法を説明します。

この方法は、特に次のような場合に役立ちます：
- 互換性のあるコントローラーを持っていないプレイヤー
- ゲームパッドに投資せずにゲームを素早くテストしたい場合
- キーボードマウスに適した特定のゲームジャンル

### 全体的なプロセス

1. **ダウンロード** ViGEmBusとMouse2Joystick
2. **インストール** ViGEmBus（仮想コントローラードライバー）
3. 好みに応じて**設定** Mouse2Joystick
4. コントロールを認識するように**セットアップ** Cemu
5. **テスト**して設定を調整

## 前提条件とダウンロード

### 必要なソフトウェア

**必須アプリケーション：**
- **ViGEmBus**: 仮想コントローラー用ドライバー
- **Mouse2Joystick**: キーボード/マウスの信号をコントローラー信号に変換
- **Cemu**: Wii Uエミュレータ（最新の安定版）

### ダウンロード

#### ViGEmBus（仮想コントローラードライバー）

**🔗 [ViGEmBusをダウンロード](https://github.com/ViGEm/ViGEmBus/releases)**

**ダウンロード手順：**
1. リリースページに**アクセス**
2. 最新の`.msi`ファイルを**ダウンロード**
3. **例**: `ViGEmBus_Setup_1.17.333.msi`

**ViGEmBusの機能：**
- **無料**でオープンソース
- Windows 10/11と**互換性**あり
- **安定**しており、定期的に更新
- Xboxコントローラーのシミュレートに**必須**

#### Mouse2Joystick（コントロールコンバーター）

**🔗 [Mouse2Joystickをダウンロード](https://github.com/memethyl/Mouse2Joystick/releases)**

**ダウンロード手順：**
1. 最新の`.zip`ファイルを**ダウンロード**
2. **例**: `Mouse2Joystick_v2.3.1.zip`
3. 専用のフォルダに**展開**（例：`C:\Tools\Mouse2Joystick\`）

**Mouse2Joystickの機能：**
- **ポータブル**：インストール不要
- **カスタマイズ可能**：キーマッピングを設定可能
- **軽量**：パフォーマンスへの影響が最小限
- **リアルタイム**：瞬時の動きの変換

## ステップ1：ViGEmBusのインストール

### インストールプロセス

#### インストーラーの実行

1. ダウンロードした`.msi`ファイルを**見つける**
2. **ダブルクリック**してインストールを開始
3. プロンプトが表示されたら管理者権限を**承認**

#### インストール設定

**インストール手順：**
1. **ようこそ** → 「Next」をクリック
2. **ライセンス契約** → 同意して「Next」をクリック
3. **インストール先フォルダ** → デフォルトのまま「Next」をクリック
4. **インストールの準備完了** → 「Install」をクリック
5. **完了** → 「Finish」をクリック

**インストールされるコンポーネント：**
- 仮想バスドライバー
- 必要なシステムライブラリ
- デバイス管理インターフェース

### インストールの確認

#### デバイスマネージャーでの確認

1. 「PC」を**右クリック** → **デバイスマネージャー**
2. 「システムデバイス」を**展開**
3. 「ViGEmBus Device」の存在を**確認**


## ステップ2：Mouse2Joystickの設定

### 展開と初回使用

#### ソフトウェアの準備

1. ダウン로드した`.zip`ファイルに**移動**
2. **右クリック** → **すべて展開**
3. 永続的なフォルダを**選択**（例：`C:\Tools\Mouse2Joystick\`）
4. `Mouse2Joystick.exe`を**起動**

### インターフェースと基本設定

#### 初回起動

起動すると、Mouse2Joystickはシンプルなインターフェースを表示します：


#### 推奨設定

**推奨される基本設定：**

**「Mouse Settings」セクション：**
```
Mouse Sensitivity X: 3.0
Mouse Sensitivity Y: 3.0
Mouse Deadzone: 0.1
Invert Y-axis: 好みに応じて
```

**「Controller Settings」セクション：**
```
Controller Type: Xbox 360 Controller
Vibration: 無効（不要）
```

### キーマッピング

#### コントロール設定

**推奨される標準マッピング：**

| Wii Uアクション | キーボードキー | 機能 |
|--------------|--------------|----------|
| **左スティック** | WASD | 移動 |
| **右スティック** | マウス | カメラ |
| **Aボタン** | スペース | 主要アクション |
| **Bボタン** | 左Ctrl | 副次アクション |
| **Xボタン** | E | インタラクション |
| **Yボタン** | Q | メニュー/インベントリ |
| **L/R** | Shift/Tab | バンパー |
| **ZL/ZR** | 左/右クリック | デジタルトリガー |

#### 詳細設定

**マウス感度：**
- **FPSゲーム**：高感度 (4.0-5.0)
- **アドベンチャーゲーム**：中感度 (2.5-3.5)
- **プラットフォームゲーム**：低感度 (1.5-2.5)

**デッドゾーン：**
- **マウス**：0.05-0.15（微小な動きを避ける）
- **キーボード**：0.0（即時反応）

## ステップ3：Cemuの設定

### コントロール設定へのアクセス

#### Cemuでのナビゲーション

1. Cemuを**起動**
2. **メニュー**「Options」→ **「Input Settings」**
3. **「-」**ボタンですべての既存コントローラーを**削除**


### 既存コントローラーの削除

#### 設定のクリーンアップ

**削除手順：**
1. リストされているすべてのコントローラーを**特定**
2. 各コントローラーの**「-」**ボタンを**クリック**
3. プロンプトが表示されたら削除を**確認**
4. リストが空であることを**確認**


### 新しいコントローラーの追加

#### XInput設定（Mouse2Joystick）

**Mouse2Joystickを開いた状態で：**
1. **「+」**ボタンを**クリック**
2. APIとして「XInput」を**選択**
3. 表示されるはずの「Controller 1」を**選択**
4. 選択を**確認**


#### キーボード設定（追加キー）

**マップされていないキーの場合：**
1. 再度**「+」**を**クリック**
2. APIとして「Keyboard」を**選択**
3. 「Controller 1」を**選択**
4. 特定のWii Uキーを**設定**


### 最終的なキー設定

#### Cemuでのマッピング

**推奨設定：**

**XInputコントローラー（Mouse2Joystick）：**
```
左スティック: キャラクターの移動
右スティック: カメラ操作
Aボタン: 主要アクション
Bボタン: 戻る/キャンセル
Xボタン: 副次アクション
Yボタン: メニュー
L/Rバンパー: クイックトリガー
L/Rトリガー: アナログトリガー
```

**キーボードコントローラー（特殊キー）：**
```
プラス: Enter（メインメニュー）
マイナス: Escape（一時停止）
ホーム: Tab（ホームメニュー）
スクリーンショット: F12
```

## 結論

**Mouse2Joystick**と**ViGEmBus**のおかげで、**Cemu**でキーボードとマウスを使って**Wii U**のゲームをプレイすることが可能です。すべてのゲームで最適な体験が得られるわけではありませんが、この解決策により、コントローラーに投資することなく、優れたWii Uのカタログを発見できます。


### 最終的なヒント

**より良い体験のために：**
- 適応したゲームから始める（ゼルダ、RPG）
- 感度を調整する時間をかける
- 設定を保存する
- いずれはコントローラーの購入を検討する

## FAQ - よくある質問

**Q: この方法はすべてのWii Uゲームで機能しますか？**
A: いいえ、一部のゲーム（マリオカート、スマッシュブラザーズなど）は、適切な体験のために絶対にコントローラーが必要です。

**Q: この設定を他のエミュレータで使用できますか？**
A: はい、Mouse2JoystickはXInputをサポートするほとんどのエミュレータで動作します。

**Q: 遅延は大きいですか？**
A: 遅延は一般的に低いですが、ハードウェア構成によって異なる場合があります。

**Q: Mouse2Joystickを常に開いておく必要がありますか？**
A: はい、コントローラーのシミュレーションを維持するために、プレイ中はMouse2Joystickを開いたままにしておく必要があります。

**Q: この方法はCemuのパフォーマンスに影響しますか？**
A: Mouse2Joystickはリソースをほとんど消費しないため、影響は最小限です。
