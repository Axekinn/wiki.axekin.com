## 🎮 Ryujinxの紹介

**Ryujinx**は、C#で書かれたオープンソースのNintendo Switchエミュレータです。PCでNintendo Switchのゲームを優れたパフォーマンスと絶えず向上する互換性でプレイすることができます。

## 🔧 ステップ1：Ryujinxのダウンロード

### オプション1：安定版（推奨）
```bash
# 公式ウェブサイトにアクセス
https://ryujinx.app/
```

1. **"Download"**をクリック
2. アーカイブを専用のフォルダに展開します（例：`Documents\Ryujinx`または別のドライブのルート、Cドライブ以外）

## ゲームのダウンロード

### ステップ1：ゲームライブラリにアクセス

**[🎮 Axekin Games - Nintendo Switch](https://www.axekin.com/games?platform=switch)**にアクセスして、Nintendo Switchゲームコレクションにアクセスします。

### ステップ2：ゲームをダウンロード

1. リストから目的のゲームを**検索**
2. ダウンロードボタンを**クリック**
3. ダウンロードが完了するのを**待つ**

### ステップ3：ファイルの展開

ダウンロードが完了したら、次のいずれかのソフトウェアでアーカイブを**展開**します。

| ソフトウェア | ダウンロードリンク | 互換性 |
|----------|---------------|---------------|
| **7-Zip** | [7-Zipをダウンロード](https://www.7-zip.org/) | Windows/Linux |
| **WinRAR** | [WinRARをダウンロード](https://www.win-rar.com/) | Windows |

> **💡 ヒント**：7-Zipは無料でオープンソースですが、WinRARは試用期間後にライセンスが必要です。

**結果**：Ryujinxで使用できるゲームファイルが手に入ります！ 🎯

## 📁 ステップ2：フォルダ構造

ファイルを整理するために、このフォルダ構造を作成します。

ライブラリを最適に整理するために、推奨されるツリー構造は次のとおりです。

````
Switch Games/
├── Mario Kart 8 Deluxe/
│   ├── BASE/                # 🎮 ベースゲーム（メインファイル）
│   ├── UPDATE/              # 🔄 ゲームアップデート（推奨）
│   └── DLC/                 # 📦 ダウンロードコンテンツ（オプション）
├── New Super Mario Bros U Deluxe/
│   ├── BASE/                # 🎮 ベースゲーム
│   ├── UPDATE/              # 🔄 ゲームアップデート
│   └── DLC/                 # 📦 追加DLC
├── The Legend of Zelda BOTW/
│   ├── BASE/
│   ├── UPDATE/
│   └── DLC/
└── Super Mario Odyssey/
    ├── BASE/
    └── UPDATE/              # (このゲームにはDLCがありません)
````

### コンテンツの種類


- **BASE** 📁：ダウンロードするメインゲーム
- **UPDATE** 📁：アップデートとパッチ（推奨）
- **DLC** 📁：追加の有料コンテンツ（オプションですが推奨）


✅ **自動検出**：ゲームは自動的にエミュレータに追加されます  
✅ **簡素化された管理**：各ROMを手動で選択する必要はありません  
✅ **明確な整理**：各ゲームにどのコンテンツがあるかが簡単にわかります  
✅ **簡単なメンテナンス**：コンテンツの追加/削除が簡素化されます

> **📝 重要事項**：ゲームにアップデートやDLCがない場合、対応するフォルダを作成する必要はありません。実際に持っているコンテンツのフォルダのみを作成してください。

### 結果

この構造が整うと、エミュレータは「Switch Games」フォルダに存在するすべてのゲームを自動的に検出し、ライブラリに適切に整理します！ 🎯

## 💡 ポータブル設定（推奨）


**最初の起動前**に、Ryujinxディレクトリのルートに`portable`という名前のフォルダを作成します。この操作により、インストールが完全にポータブルになります。

### ポータブルモードの利点

| 側面 | 通常モード | ポータブルモード |
|--------|-------------|---------------|
| **設定** | システムに保存 | フォルダに保存 |
| **セーブデータ** | コンピュータ上に散在 | フォルダに集中 |
| **ポータビリティ** | ❌ システムに依存 | ✅ 簡単に移動可能 |
| **バックアップ** | 見つけにくい | バックアップが簡単 |

### portableフォルダ作成後の構造

```
📁 Ryujinx/
├── 🎮 Ryujinx.exe
├── 📁 portable/              # ← このフォルダを作成してください！
│   ├── 📁 bis/
│   ├── 📁 games/
│   ├── 📁 profiles/
│   └── 📁 system/
└── 📄 その他のファイル...
```


**すべてのデータ**（設定、セーブデータ、プロファイル）は`portable`フォルダに保存され、管理とバックアップが大幅に容易になります！

### 作成手順

1. Ryujinxのインストールフォルダに**移動**します
2. `portable`という名前の新しいフォルダを**作成**します
3. Ryujinxを**起動**します - ポータブルモードが自動的に検出されます

> **📝 注意**：この操作は、反映されるために**最初の起動前**に実行する必要があります！

## ⚙️ ステップ3：初期設定

### 最初の起動
1. `ryujinx.exe`を**実行**します
2. **キーをインストール**します
- [キーバージョン18.X.X](https://files-prodkeys.b-cdn.net/prodkeys/ProdKeys.net-v18.-1-0.zip)
- [キーバージョン19.X.X](https://files.prodkeys.net/ProdKeys.net-v19.0.1.zip)
- [キーバージョン20.X.X](https://files.prodkeys.net/ProdKeys.net-v20.0.1.zip)

#### Ryujinxでのインストール
1. Ryujinxを**開きます**
2. `ファイル → Ryujinxフォルダを開く`
3. `system/`フォルダに**移動**します
4. `prod.keys`と`title.keys`（存在する場合、しかし不要）を中に**コピー**します
5. Ryujinxを**再起動**します

### Nintendo Switchファームウェアのインストール

- [ファームウェア20.2.0](https://github.com/THZoria/NX_Firmware/releases/download/19.0.1/Firmware.20.2.0.zip)
- [ファームウェア20.3.0](https://github.com/THZoria/NX_Firmware/releases/download/19.0.1/Firmware.20.3.0.zip)
- [ファームウェア20.4.0](https://github.com/THZoria/NX_Firmware/releases/download/19.0.1/Firmware.20.4.0.zip)
- [ファームウェア20.5.0](https://github.com/THZoria/NX_Firmware/releases/download/20.5.0/Firmware.20.5.0.zip)

#### Ryujinxでのインストール
1. `アクション → ファームウェアをインストール → XCIまたはZIPからファームウェアをインストール`
2. ファームウェアの.zipファイルを**選択**します
3. 完全なインストールを**待ちます**

### 基本設定の構成

#### インターフェース設定
```
オプション → 設定 → ユーザーインターフェース
```
- **ゲームディレクトリ**：以前に作成した**Switch Games**ディレクトリを設定します
- **DLC/アップデートを自動ロード**：上記と同じ
- **テーマ**：好きなものを選択

#### 入力設定
```
オプション → 設定 → 入力
```
- **入力デバイス**：コントローラーを接続し、それを選択します
- **自動設定**：設定は自動で行われます。後で好きなボタンを変更できます
- **キーボード/マウス**：キーボード/マウスプレイヤーは、キーを調整します

#### システム設定
```
オプション → 設定 → システム
```
- **システム地域**：あなたの国
- **システム言語**：あなたの言語
- **オーディオバックエンド**：SDL2（推奨）
- **音量**：100%

## 🚀 パフォーマンスの最適化

### パフォーマンス向上のための詳細設定

#### CPU
```
設定 → CPU
- PTCを有効にする（プロファイルされた翻訳キャッシュ）：✅ 有効
- メモリマネージャモード：ホスト
```

#### GPU
```
設定 → グラフィックス
- シェーダーキャッシュ：✅ 有効
- テクスチャ再圧縮：✅ 有効
- 解像度スケール：1x（または十分なパフォーマンスがあればそれ以上）。アンチエイリアシング、スケーリングフィルター、異方性フィルタリングも同様に、PCのパワーに応じて値を調整します
```

## 🌐 マルチプレイヤーモードの設定

### オンラインモードの有効化


`オプション → 設定 → ネットワーク → モード`に**移動**し、**RyuLDN**を選択します。

### 推奨される接続モード

| 接続タイプ | ゲーム内でのアクション | 結果 |
|-----------------|----------------|--------|
| **❌ 避けるべき** | 「オンライン」または「インターネット」オプション | 任天堂サーバーへの接続 |
| **✅ 推奨** | 「ローカルでプレイ」または「ローカルワイヤレス」オプション | RyuLDN経由の接続 |

### 接続プロセス

```
1. RyuLDN設定が有効 ✅
2. ゲームの起動
3. メニューで「ローカルでプレイ」を選択
4. 他のRyujinxプレイヤーと自動的に接続
```


最適な接続のために、すべてのプレイヤーがゲームの**同じバージョン**を使用し、**RyuLDNが有効**になっていることを確認してください！

## 🛠️ 一般的なトラブルシューティング

### 問題：ゲームが起動しない
```bash
解決策：
1. ファームウェアがインストールされていることを確認する
2. prod.keysが存在することを確認する
3. 異なるグラフィック設定で試す
```

### 問題：パフォーマンスが悪い
```bash
解決策：
1. 解像度を1xに下げる
2. V-Syncを一時的に無効にする
3. 他のアプリケーションを閉じる
4. OpenGLの代わりにVulkanを使用する
```

### 問題：頻繁なクラッシュ
```bash
解決策：
1. Ryujinxを最新バージョンに更新する
2. ゲームファイルの整合性を確認する
3. ファームウェアを再インストールする
4. デフォルト設定で試す
```

## ❓ FAQ - よくある質問

### 一般的な質問

**Q：Ryujinxは合法ですか？**
**A：** はい、エミュレーションは合法です。ただし、使用するすべてのゲームとファームウェアを合法的に所有している必要があります。

**Q：Ryujinxでオンラインプレイはできますか？**
**A：** はい、ローカルプレイエミュレーションのためにRyuLDN経由で可能です。

**Q：最小システム要件は何ですか？**
**A：** Windows 10 64ビット、Intel Core i5-4430 / AMD FX-6300、8GB RAM、DirectX 11互換GPU。

### 技術的な質問

**Q：VulkanかOpenGLか？**
**A：** Vulkanは一般的に最近のGPUでよりパフォーマンスが高いです。OpenGLは一部のシステムでより安定している場合があります。

**Q：最低RAMはどれくらいですか？**
**A：** 最低8GB、推奨16GB、最も重いゲームには32GB。

**Q：私のGPUは互換性がありますか？**
**A：** DirectX 11またはVulkan 1.1をサポートするGPU。良好なパフォーマンスのためにはGTX 1060/RX 580。
