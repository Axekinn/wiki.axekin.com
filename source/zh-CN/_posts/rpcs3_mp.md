> 通过 RPCN 以流畅、可靠的设置在线玩 PS3 游戏。请遵循快速清单，然后在需要时深入了解。

> 📝 作者注  
> RPCS3 的多人游戏设置从未如此简单。对于那些技术更娴熟的人，我附上了 RPCS3 的教程，解释得非常清楚。

## ⚡ 快速入门清单

- 更新到最新的 RPCS3 版本
- 创建并验证一个 RPCN 帐户
- 每款游戏：网络 = 已连接，PSN = RPCN
- DNS 8.8.8.8 (除非游戏需要其他)
- 在 PC/路由器上启用 UPnP 或转发 UDP 3658 端口
- 在 RPCN 窗口中测试帐户

## ℹ️ 什么是 RPCN？

RPCN 是一个由社区制作的 PSN 替代品，RPCS3 使用它来在支持的游戏中启用在线功能。

> ⚠️ 免责声明  
> 在 99% 的情况下，您不需要更改防火墙、NAT/UPnP 或端口转发设置。RPCS3 + RPCN 在大多数网络上应该可以开箱即用。只有在按照以下步骤操作后与其他玩家的在线连接失败时，才需要调整这些设置。

## ✅ 要求

| 项目 | 建议 | 注意 |
|---|---|---|
| RPCS3 | 最新版本 | 自动更新或全新下载 |
| 端口 | UDP 3658 | UPnP 或手动端口转发 |
| NAT | 开放/A 型 | 避免双重 NAT/CGNAT |
| 防火墙 | 允许 RPCS3 | 操作系统和安全套件 |

## 🔧 全局配置

1. 打开 RPCS3 → 配置 → **RPCN**
2. 点击 **帐户**
3. 选择服务器：**官方 RPCN 服务器**
4. 点击 **创建帐户**
5. 设置一个 **用户名** → 确定
6. 输入 **密码** 两次 → 确定
7. 输入 **电子邮件** 两次 → 确定
8. 确认摘要 → **是**
9. 成功消息 → 出现一个 “RPCN: 用户名” 窗口
10. 检查邮箱以获取 RPCN **令牌** → 粘贴到 **令牌** → 确定
11. 点击 **测试帐户**

> 💡 专业提示  
> 请将您的令牌保密。如果丢失，请在“编辑帐户”中使用“重新发送令牌”。

---

## 🎯 特定游戏配置

1. 在游戏列表中，右键单击标题 → **创建自定义配置**
2. 转到 **网络**
3. **网络状态**：已连接
4. **PSN 状态**：RPCN
5. **DNS**：`8.8.8.8` (除非游戏需要其他)
6. 如果游戏需要自定义服务器，请输入社区提供的 IP/主机切换

> 💡 每款游戏的覆盖设置  
> 这些设置仅适用于此游戏，不会更改您的全局配置。

## 🌐 连接要点

- 在 RPCS3 中启用 UPnP：配置 → 网络 → **启用 UPnP**
- 在您的路由器上启用 UPnP (手册因型号而异)
- 或将端口 **3658/UDP** 转发到您 PC 的本地 IP

<details>
  <summary>显示端口转发说明</summary>

- 转发：UDP 3658 → 您的 PC IP (通过 DHCP 预留)  
- 避免双重 NAT (如果需要，将 ISP 调制解调器桥接)  
- VPN 可能会阻止入站 UDP；请在 VPN 上配置端口转发或在玩游戏时禁用它
</details>

> 🚧 NAT 和 ISP 限制  
> 严格的 NAT 或 CGNAT 可能会破坏对等连接。请向您的 ISP 索要公共 IPv4 或使用支持端口转发的 VPN。

---

## 🛠️ 故障排除

### 无法连接到其他玩家
- 在 RPCS3 和路由器上启用 UPnP
- 或将 UDP 3658 转发到您的 PC
- 允许 RPCS3 通过防火墙/安全套件
- 禁用或重新配置 VPN
- RPCN → 帐户 → **测试帐户**

<details>
  <summary>高级检查</summary>

- 确认您的 PC IP 没有改变 (DHCP 预留)  
- 检测双重 NAT (调制解调器 + 路由器都在路由)  
- 路由器 NAT 测试应目标为开放/A 型
</details>

---

## 🔐 恢复 RPCN 帐户

### 重新发送 RPCN 令牌
1. RPCS3 → 配置 → RPCN → **帐户**
2. **编辑帐户** → **重新发送令牌**
3. 确认 → 确定 → 检查电子邮件“您的 RPCN 令牌”
4. 将令牌粘贴到 **令牌** 字段 → 确定 → **保存**

### 重置 RPCN 密码
1. RPCS3 → 配置 → RPCN → **帐户**
2. **编辑帐户** → **更改密码**
3. 确认用户名 → 确定
4. 当被问及“您是否已有重置密码令牌？”时 → **否**
5. 输入注册电子邮件两次 → 确定
6. 检查电子邮件“您的 RPCN 密码重置代码”
7. 粘贴令牌 → 确定
8. 输入新密码两次 → 确定
9. 返回编辑帐户 → **设置密码** → 输入新密码 → 确定 → **保存**
10. RPCN: **测试帐户**

---

### 良好实践
- 使用强大、独特的密码  
- 不要分享您的令牌  
- 保持 RPCS3 更新以获取最新的 RPCN 修复  

> 尊重游戏许可和在线服务条款。通过购买游戏来支持开发者。

🎯 结果：您已准备好通过 RPCN 在线玩游戏！
