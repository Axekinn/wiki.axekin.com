> **⚠️ 重要事項**: この手順には、セーフモードでの再起動とドライバの完全なアンインストールが含まれます。各ステップを注意深く実行し、開始する前にドライバをダウンロードしていることを確認してください。

## はじめに

**グラフィックドライバのクリーン再インストール**は、パフォーマンスの問題、ゲームのクラッシュ、または表示エラーを解決するための最も効果的な解決策であることがよくあります。このプロセスでは、**DDU (Display Driver Uninstaller)** を使用して、新しいドライバをインストールする前に古いドライバを完全に削除します。

この方法は、特に次の場合に効果的です。
- 古いバージョンと新しいバージョンの間の競合を解決する
- ゲーム内のパフォーマンスの問題を修正する
- 持続的な表示エラーを排除する
- グラフィックカードの交換後にクリーンアップする

### 一般的なプロセス

1. 新しいドライバとDDUを**ダウンロード**する
2. インターネットを**切断**し、セーフモードで再起動する
3. DDUで古いドライバを**完全に削除**する
4. **再起動**して新しいドライバをインストールする
5. インターネットを**再接続**し、機能を確認する

## 前提条件とダウンロード

### グラフィックカードの特定

開始する前に、GPUを特定します。

**方法1：デバイスマネージャー**
1. 「このPC」を**右クリック** → **プロパティ**
2. **デバイスマネージャー** → **ディスプレイアダプター**
3. カードの正確なモデルを**メモ**する

**方法2：システムコマンド**
```cmd
dxdiag
```
**「ディスプレイ」**タブでGPUを確認します。

### 必要なダウンロード

#### NVIDIAカードの場合

**🔗 [NVIDIA公式サイト](https://www.nvidia.co.jp/Download/index.aspx?lang=jp)**

**ダウンロード手順：**
1. カードシリーズ（RTX 40、RTX 30、GTX 16など）を**選択**する
2. 正確なモデルを**選択**する
3. 最新の**「Game Ready」**ドライバを**ダウンロード**する
4. ファイルをデスクトップに**保存**する

**NVIDIAドライバの種類：**
- **Game Ready Driver**: ゲームに最適化（推奨）
- **Studio Driver**: コンテンツ作成に最適化

#### AMDカードの場合

**🔗 [AMD公式サイト](https://www.amd.com/ja/support)**

**ダウンロード手順：**
1. 「グラフィックスカード」→シリーズ（RX 7000、RX 6000など）を**選択**する
2. 正確なモデルを**選択**する
3. 完全な**「Adrenalin」**パッケージを**ダウンロード**する
4. ファイルをデスクトップに**保存**する

#### Display Driver Uninstaller (DDU)

**🔗 [DDUをダウンロード](https://www.guru3d.com/download/display-driver-uninstaller-download/)**

**重要情報：**
- **無料**で安全に使用可能
- **ポータブル**：インストール不要
- すべての最近のWindowsバージョンと**互換性**あり
- **定期的に更新**される

## ステップ1：システムの準備

### 重要なバックアップ

開始する前に：

**🛡️ 不可欠なバックアップポイント：**
- **ゲームプロファイル**（NVIDIA GeForce Experience、AMD Softwareの設定）
- **カスタムディスプレイ設定**
- **適用されたオーバークロック**（値をメモする）

### プログラムの終了

**GPU関連のすべてのソフトウェアを終了します：**
- GeForce Experience / AMD Software
- MSI Afterburner / GPU-Z
- ゲームおよび3Dアプリケーション
- ストリーミングソフトウェア（OBSなど）

### インターネットの切断（重要）

> **🔥 重要なステップ**：セーフモードで再起動する前に、イーサネットケーブルを物理的に抜くか、Wi-Fiを無効にします。

**なぜインターネットを切断するのか？**
- Windowsが汎用ドライバを自動的にインストールするのを防ぐ
- 再インストール中の競合を回避する
- DDUのためのクリーンな環境を保証する

## ステップ2：セーフモードへのアクセス

### Windows 10の場合

#### 設定経由

1. 「設定」を開く（Windows + I）
2. 「更新とセキュリティ」をクリックする
3. 左側のメニューで「回復」を選択する
4. **「高度な起動」**で、**「今すぐ再起動」**をクリックする

### Windows 11の場合

#### 設定経由

1. 「設定」を開く（Windows + I）
2. 左側のメニューで「Windows Update」をクリックする
3. 「詳細オプション」を選択する
4. 「回復」をクリックする
5. **「高度な起動」**で、**「今すぐ再起動」**をクリックする

### 高度な起動メニューでのナビゲーション

PCが再起動すると、オプション付きの青い画面が表示されます。

#### ナビゲーションシーケンス

1. 「トラブルシューティング」をクリックする
2. 「詳細オプション」をクリックする
3. 「スタートアップ設定」をクリックする
4. 「再起動」をクリックする

#### スタートアップ設定メニュー

再起動後、番号付きのリストが表示されます。

**利用可能なオプション：**
```
1) デバッグを有効にする
2) ブートログを有効にする
3) 低解像度ビデオを有効にする
4) セーフモードを有効にする          ← このオプションを選択
5) セーフモードとネットワークを有効にする
6) セーフモードとコマンドプロンプトを有効にする
7) システム障害時の自動再起動を無効にする
8) ドライバ署名の強制を無効にする
9) 早期起動マルウェア対策保護を無効にする
```

標準のセーフモードの場合は**「4」キーを押します**。

## ステップ3：DDUの使用

### 初回起動

セーフモードに入ったら：

1. DDUを解凍したフォルダに**移動**する
2. 「Display Driver Uninstaller.exe」を**右クリック**する
3. 「管理者として実行」を**選択**する

### DDUの設定

#### デバイスタイプの選択

**「デバイスタイプの選択」**ドロップダウンで：
- 「GPU」を**選択**する

#### メーカーの選択

**「デバイスの選択」**メニューで：
- **NVIDIAの場合**：「NVIDIA」を選択
- **AMDの場合**：「AMD」を選択
- **Intelの場合**：「INTEL」を選択（該当する場合）

### アンインストールプロセス

#### アンインストールの開始

1. 「クリーンして再起動」を**クリック**する
2. プロンプトが表示されたらアクションを**確認**する
3. DDUがプロセスを完了するのを**待つ**（2〜5分）

#### クリーニングの進行状況

DDUは次のことを行います：
- すべてのドライバファイルを**削除**する
- Windowsレジストリを**クリーンアップ**する
- ドライバ関連のサービスを**削除**する
- 一時フォルダを**削除**する
- 自動的に**再起動**する

**進行状況インジケータ：**
- 緑色のプログレスバー
- リアルタイムのステータスメッセージ
- 削除されたアイテムのリスト

## ステップ4：新しいドライバのインストール

### DDU後の確認

自動再起動後：

**アンインストール成功の兆候：**
- 基本的な画面解像度（1024x768など）
- デバイスマネージャーにGPUドライバがない
- Windowsが基本ディスプレイアダプタを使用している

### NVIDIAドライバのインストール

#### GeForce Game Ready Driver

1. ダウンロードした.exeファイルを**見つける**
2. **右クリック**→「管理者として実行」
3. インストールウィザードに**従う**

**推奨されるインストールオプション：**
```
☑ グラフィックドライバ
☑ HDオーディオドライバ
☑ PhysXシステムソフトウェア
☑ GeForce Experience（オプション）
```

**インストールの種類：**
- **エクスプレス**：デフォルト設定でのクイックインストール
- **カスタム**：コンポーネントの完全な制御（推奨）

#### カスタムインストール（推奨）

**必須コンポーネント：**
- **ディスプレイドライバ**：メインドライバ（必須）
- **PhysX**：ゲーム内の高度な物理演算
- **NVIDIA HDオーディオ**：HDMI/DisplayPort経由のサウンド

**オプションのコンポーネント：**
- **GeForce Experience**：自動ゲーム最適化

**インストールオプション：**
- **クリーンインストール**：☑（推奨）
- **工場出荷時リセット**：☑（設定をゼロにリセット）

### AMDドライバのインストール

#### AMD Adrenalin Software

1. ダウンロードした.exeファイルを**見つける**
2. **右クリック**→「管理者として実行」
3. インストールの種類を**選択**する

**AMDインストールの種類：**
- **インストール**：完全インストール（推奨）
- **最小インストール**：ドライバのみ
- **工場出荷時リセット**：完全リセット

#### AMD Adrenalinコンポーネント

**主なコンポーネント：**
```
☑ AMDディスプレイドライバ
☑ AMDオーディオドライバ
☑ AMD Software Adrenalin Edition
☑ AMDチップセットソフトウェア（該当する場合）
```

**含まれる機能：**
- **Radeon Software**：メインコントロールインターフェイス
- **ReLive**：ゲームプレイの録画とストリーミング
- **Radeon Image Sharpening**：シャープネスの向上

### インストールの完了

#### 最終的な再起動

1. インストールが完了したらPCを**再起動**する
2. インターネットを**再接続**する（イーサネットケーブルまたはWi-Fi）
3. 画面の解像度が正しいことを**確認**する

#### 機能の確認

**基本的なテスト：**
- ネイティブ解像度が自動的に検出される
- デバイスマネージャーでドライバが表示される
- コントロールソフトウェアが機能する（GeForce Experience / AMD Software）

## インストール後の確認と設定

### 機能のテスト

#### 基本的なパフォーマンステスト

**GPUの使用率：**
1. タスクマネージャーを開く（Ctrl+Shift+Esc）
2. 「パフォーマンス」タブ→**GPU**
3. 簡単な3Dアプリケーションを**起動**してテストする

#### ディスプレイテスト

**ディスプレイの確認：**
- **解像度**：画面のネイティブ解像度
- **リフレッシュレート**：サポートされている最大値
- **色**：32ビット深度
- **複数画面**：自動検出

### 推奨設定

#### NVIDIA設定（NVIDIAコントロールパネル経由）

**最適な3D設定：**
```
電源管理モード：最大パフォーマンスを優先
垂直同期：3Dアプリケーションの設定を使用
```

### プロファイルの復元

#### ゲームプロファイル

**NVIDIA GeForce Experience：**
1. NVIDIAアカウントに**サインイン**する
2. クラウド設定を**同期**する
3. ゲームプロファイルを**復元**する

**AMD Software：**
1. 保存したプロファイルを**インポート**する
2. ゲームごとの設定を**再構成**する
3. 目的の機能を**有効**にする

## 高度なトラブルシューティング

### 一般的な問題

#### 🚫 インストールが失敗する

**考えられる原因：**
- 古いドライバの残骸が削除されていない
- ウイルス対策ソフトがインストールをブロックしている
- インストールファイルが破損している

**解決策：**
1. セーフモードでDDUを**再実行**する
2. ドライバを**再ダウンロード**する
3. 互換モードでインストールを**試す**

#### ⚠️ インストール後に画面が真っ黒になる

**緊急措置：**
1. セーフモードで**再起動**する（起動時にF8キー）
2. DDUを**再実行**して再度クリーンアップする
3. 以前の安定バージョンを**インストール**する
4. GPU/ドライバの互換性を**確認**する

#### 💻 パフォーマンスの低下

**確認事項：**
- **GPU温度**（MSI Afterburner / HWMonitor）
- タスクマネージャーでの**GPU使用率**
- **Windowsの電源設定**（最大パフォーマンス）

### パフォーマンスの監視

**監視ツール：**
| ツール | 機能 | ユーティリティ |
|---|---|---|
| **MSI Afterburner** | 完全な監視 | 温度、使用率、FPS |
| **GPU-Z** | 詳細情報 | 仕様、センサー |
| **HWiNFO64** | システム監視 | 高度な診断 |

### 警告サイン

**再インストールが必要な兆候：**
- 頻繁なゲームのクラッシュ
- グラフィックアーティファクト
- 説明のつかないパフォーマンスの低下

## FAQ - よくある質問

**Q：ドライバを更新するたびにDDUを使用する必要がありますか？**
**A：**いいえ、DDUは問題が発生した場合や、大きな変更（新しいGPU、破損したバージョンなど）があった場合にのみ必要です。

**Q：セーフモードなしでDDUを使用できますか？**
**A：**強くお勧めしません。セーフモードは、アンインストールを妨害するプロセスがないことを保証します。

**Q: DDUが私のグラフィックカードを検出しない場合はどうすればよいですか？**
**A：**DDUを使用する前に、GPUが正しくインストールされ、デバイスマネージャーで認識されていることを確認してください。

**Q：ゲームの設定は失われますか？**
**A：**いいえ、ゲームの設定は通常、別々に保存されます。GPUコントロールパネルのプロファイルのみが失われます。

**Q：完全な手順にはどのくらい時間がかかりますか？**
**A：**ダウンロードと再起動を含め、全プロセスに30〜45分を見込んでください。
