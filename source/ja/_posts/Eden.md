## 🎮 Yuzuの紹介

**Eden**は、Citraチームによって開発されたオープンソースのNintendo Switchエミュレータです。高いパフォーマンスと増え続ける互換性で知られるEdenは、世界で最も人気のあるSwitchエミュレータの1つです。

> ⚠️ **法的注意**: このエミュレータは、法的に所有しているゲームをプレイするためだけに使用されることを目的としています。バックアップおよび研究目的のエミュレーションは合法ですが、海賊行為は違法です。

## ゲームのダウンロード

### ステップ1：ゲームライブラリへのアクセス

**[🎮 Axekin Games - Nintendo Switch](https://www.axekin.com/games?platform=switch)**にアクセスして、Nintendo Switchのゲームコレクションにアクセスします。

### ステップ2：ゲームのダウンロード

1. リストから目的のゲームを**検索**します
2. ダウンロードボタンを**クリック**します
3. ダウンロードが完了するのを**待ちます**

### ステップ3：ファイルの展開

ダウンロードが完了したら、次のいずれかのプログラムを使用してアーカイブを**展開**します：

| ソフトウェア   | ダウンロードリンク                             | 互換性    |
|------------|-------------------------------------------|------------------|
| **7-Zip**  | [7-Zipをダウンロード](https://www.7-zip.org/) | Windows/Linux    |
| **WinRAR** | [WinRARをダウンロード](https://www.win-rar.com/)| Windows         |

> **💡 ヒント**: 7-Zipは無料でオープンソースですが、WinRARは試用期間後にライセンスが必要です。

**結果**: Edenで使用できるゲームファイルが手に入ります！ 🎯

## 🎯 ステップ3：キーとファームウェアのインストール

### 暗号化キーのインストール

#### 自動方法
- [キーバージョン18.X.X](https://files-prodkeys.b-cdn.net/prodkeys/ProdKeys.net-v18.-1-0.zip)
- [キーバージョン19.X.X](https://files.prodkeys.net/ProdKeys.net-v19.0.1.zip)
- [キーバージョン20.X.X](https://files.prodkeys.net/ProdKeys.net-v20.0.1.zip)

#### Edenでのインストール
1. Edenを**開きます**
2. `ファイル → Edenフォルダを開く`
3. `keys/`フォルダに**移動**します
4. `prod.keys`と`title.keys`を中に**コピー**します
5. Edenを**再起動**します

- [ファームウェア 20.2.0](https://github.com/THZoria/NX_Firmware/releases/download/19.0.1/Firmware.20.2.0.zip)

- [ファームウェア 20.4.0](https://github.com/THZoria/NX_Firmware/releases/download/19.0.1/Firmware.20.4.0.zip)

#### Edenでのインストール
1. `ツール → ファームウェアのインストール`
2. ファームウェアの.zipファイルを**選択**します
3. インストールが完了するのを**待ちます**
4. **確認**: `ツール → ファームウェアのバージョン`

## ⚙️ ステップ4：最適な設定

### 一般設定
```
エミュレーション → 設定 → 一般
```
- **速度制限パーセント**: 100% (通常速度用)
- **マルチコアCPUエミュレーション**: ✅ 有効
- **拡張メモリレイアウト**: ✅ 有効 (12GB以上のRAMがある場合)

### システム設定
```
エミュレーション → 設定 → システム
```
- **地域**: Europe (ヨーロッパ在住の場合。それ以外は地域を設定)
- **言語**: French (フランス在住の場合。それ以外は言語を設定)
- **RTCソース**: システムクロック
- **タイムゾーン**: Europe/Paris
- **サウンド出力モード**: ステレオ

### 最適なグラフィック設定

#### APIタブ
```
グラフィック → API
- API: Vulkan (推奨) または OpenGL
- デバイス: メインGPU
- シェーダーバックエンド: GLSL (Vulkan) / GLASM (OpenGL)
```

#### グラフィックスタブ
```
グラフィック → グラフィックス
- ディスクパイプラインキャッシュを使用: ✅ 有効
- 非同期GPUエミュレーションを使用: ✅ 有効
- 非同期シェーダービルディングを使用: ✅ 有効
- ASTCテクスチャデコーディングを高速化: ✅ 有効
- VSyncモード: FIFO (推奨)
```

#### 詳細グラフィックスタブ
```
グラフィック → 詳細
- 精度レベル: Normal (より高い精度にはHigh)
- 高速GPU時間を使用: ✅ 有効
- Vulkanパイプラインキャッシュを使用: ✅ 有効 (Vulkanの場合)
- 異方性フィルタリング: 16x
- 解像度スケーラー: 1x (GPUが強力な場合はそれ以上)
- アンチエイリアシング: なし または FXAA
```

### オーディオ設定
```
オーディオ
- 出力エンジン: auto
- 出力デバイス: Default
- 音量: 100%
```

### コントローラー設定

#### Xbox/PlayStation/Switch/互換性のあるコントローラー
```
コントロール → 設定
1. プレイヤー1 → 接続済みコントローラー: Pro Controller
2. 設定 → 自動的にマッピング
3. すべてのボタンをテスト
```

#### キーボード (一時的)
```
コントロール → 設定
1. プレイヤー1 → 接続済みコントローラー: Handheld
2. 設定 → 手動マッピング
3. ボタンにキーを割り当て
```

## エミュレータでのゲーム設定

### ゲームフォルダの追加

エミュレータメニューで、**ダブルクリック**してから、すべてのゲームが含まれているフォルダを選択します。

### 推奨されるフォルダ構造

ライブラリを最適に整理するために、推奨される構造は次のとおりです：

````
Switch Games/
├── Mario Kart 8 Deluxe/
│   ├── BASE/                # 🎮 メインゲーム (メインファイル)
│   ├── UPDATE/              # 🔄 ゲームアップデート (推奨)
│   └── DLC/                 # 📦 ダウンロード可能コンテンツ (オプション)
├── New Super Mario Bros U Deluxe/
│   ├── BASE/                # 🎮 メインゲーム
│   ├── UPDATE/              # 🔄 ゲームアップデート
│   └── DLC/                 # 📦 追加DLC
├── The Legend of Zelda BOTW/
│   ├── BASE/
│   ├── UPDATE/
│   └── DLC/
└── Super Mario Odyssey/
    ├── BASE/
    └── UPDATE/              # (このゲームにはDLCなし)
````

### コンテンツの種類


- **BASE** 📁: ダウンロードするメインゲーム
- **UPDATE** 📁: アップデートとパッチ (推奨)
- **DLC** 📁: 追加の有料コンテンツ (オプションですが推奨)


✅ **自動検出**: ゲームは自動的にエミュレータに追加されます  
✅ **簡素化された管理**: 各ROMを手動で選択する必要はありません  
✅ **明確な整理**: 各ゲームのコンテンツを簡単に確認できます  
✅ **簡単なメンテナンス**: コンテンツの追加/削除が簡単です

> **📝 重要事項**: ゲームにアップデートやDLCがない場合、対応するフォルダを作成する必要はありません。所有しているコンテンツのフォルダのみを作成してください。

### 結果

この構造が整うと、エミュレータは「Switch Games」フォルダ内のすべてのゲームを自動的に検出し、ライブラリにきれいに整理します！ 🎯

## 🔧 トラブルシューティング

### 一般的な問題と解決策

#### ゲームが起動しない
```bash
優先順位の高い順の解決策：
1. ファームウェアのインストールを確認
2. キーを確認 (prod.keys/title.keys)
3. デフォルト設定でテスト
4. Visual C++ Redistributablesを再インストール
5. Edenを最新バージョンにアップデート
```

#### パフォーマンスが低い
```bash
即時最適化：
1. 解像度スケーラーを1xに下げる
2. マルチコアCPUエミュレーションを有効にする
3. OpenGLの代わりにVulkanを使用する
4. バックグラウンドアプリケーションを閉じる
5. CPU/GPUの温度を確認する

最終的な注意：FPSを向上させるMODをインストールしても、FPSは向上しません。ゲームが30fpsで動作するように設計されている場合、PCで15〜20fpsで動作する場合（こんにちは、ポケモンスカーレット）、MODを追加しても何も変わりません。より良いPCを入手してください。私が嘘をついていると言う人たちには、彼らのナンセンスを話す能力に前もって笑っておきます。
```

#### 頻繁なクラッシュ
```bash
診断：
1. ゲームの整合性を確認 (SHA256)
2. MODなしでテスト
3. 精度レベルを下げる
4. オーバークロックを無効にする
5. 古いバージョンのYuzuでテストする
```

#### 音声の非同期
```bash
解決策：
1. 出力エンジンを変更 (auto → WASAPI)
2. オーディオバッファを調整
3. オーディオドライバを確認
4. Windowsのオーディオ効果を無効にする
```

#### グラフィックの破損
```bash
修正：
1. シェーダーキャッシュを削除
2. グラフィックAPIを変更 (Vulkan ↔ OpenGL)
3. GPUドライバを更新
4. 精度レベルを下げる
5. ASTCデコーディングを一時的に無効にする
```

## ❓ FAQ - よくある質問

### 一般的な質問

**Q: Edenは合法ですか？**  
**A:** はい、エミュレーションは合法です。ただし、使用するすべてのゲーム、キー、およびファームウェアを合法的に所有している必要があります。

**Q: Edenでオンラインプレイはできますか？**  
**A:** はい

### 技術的な質問

**Q: VulkanまたはOpenGL？**  
**A:** Vulkanは一般的に最新のGPUでより高性能です。OpenGLは一部のシステムでより安定している場合があります。

**Q: 最低RAMはどれくらいですか？**  
**A:** 最低8GB、推奨16GB、MODを使用した最も重いゲームには32GB。

**Q: 私のGPUは互換性がありますか？**  
**A:** Vulkan: GTX 1060/RX 580以上。OpenGL: OpenGL 4.6をサポートするすべてのGPU。
