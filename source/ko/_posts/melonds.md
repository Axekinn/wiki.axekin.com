## 🎮 melonDS 소개

**melonDS**는 2016년부터 개발된 현대적이고 정확한 오픈 소스 닌텐도 DS 에뮬레이터입니다. 뛰어난 호환성, 에뮬레이션 정밀도, 로컬 Wi-Fi, 세이브스테이트, 그래픽 향상과 같은 고급 기능으로 유명합니다. DeSmuME의 완벽한 현대적 대안입니다.

> 💡 **중요 참고**: melonDS는 DS의 간단한 아키텍처 덕분에 대부분의 최신 에뮬레이터보다 요구 사양이 낮습니다.

## 🔧 1단계: melonDS 다운로드

### 안정 버전 (권장)
```bash
# 공식 사이트
https://melonds.kuribo64.net/downloads.php
```

1.  "Download Windows x64"를 **클릭**합니다.
2.  `melonds-0.9.5-windows-x64.zip`을 **다운로드**합니다.
3.  전용 폴더(예: `Documents\melonDS` 또는 다른 드라이브의 루트, C 드라이브 제외)에 **압축을 풉니다**.

### 개발 버전
```bash
# 최신 기능이 포함된 자동 빌드
https://github.com/melonDS-emu/melonDS/releases

장점:
- 새로운 개선 사항
- 최근 수정 사항
- 실험적 기능

위험:
- 불안정할 수 있음
- 회귀 가능성
```

### melonDS 안드로이드
```bash
# 공식 모바일 버전
https://play.google.com/store/apps/details?id=me.magnum.melonds

기능:
- 최적화된 터치 인터페이스
- 블루투스 컨트롤러 지원
- 저장 동기화
```

## 게임 다운로드

### 1단계: 게임 라이브러리 접속

**[🎮 Axekin Games - Nintendo DS](https://www.axekin.com/games?platform=ds)**로 이동하여 닌텐도 DS 게임 컬렉션에 접속하세요.

### 2단계: 게임 다운로드

1.  목록에서 원하는 게임을 **검색**합니다.
2.  다운로드 버튼을 **클릭**합니다.
3.  다운로드가 완료될 때까지 **기다립니다**.

### 3단계: 파일 압축 해제

다운로드가 완료되면 다음 소프트웨어 중 하나로 아카이브의 **압축을 해제**합니다.

| 소프트웨어 | 다운로드 링크 | 호환성 |
|----------|---------------|---------------|
| **7-Zip** | [7-Zip 다운로드](https://www.7-zip.org/) | Windows/Linux |
| **WinRAR** | [WinRAR 다운로드](https://www.win-rar.com/) | Windows |

> **💡 팁**: 7-Zip은 무료 오픈 소스이며, WinRAR은 평가판 기간 이후 라이선스가 필요합니다.

**결과**: melonDS에서 사용할 준비가 된 게임 파일을 얻게 됩니다! 🎯

## 📁 2단계: 폴더 구조

파일을 정리하기 위해 이 폴더 구조를 만듭니다.

라이브러리를 최적으로 구성하기 위한 권장 트리 구조는 다음과 같습니다.

````
DS Games/
├── 동물의 숲: 놀러오세요/
│   ├── BASE/                # 🎮 기본 게임 (메인 파일)
├── 아노 1701: 발견의 새벽/
│   ├── BASE/                # 🎮 기본 게임
├── 아트 아카데미/
│   ├── BASE/
└── 어쌔신 크리드: 알타이르 연대기/
    ├── BASE/
````

### 콘텐츠 유형

- **BASE** 📁: 다운로드하는 메인 게임

✅ **자동 감지**: 게임이 에뮬레이터에 자동으로 추가됩니다.  
✅ **간소화된 관리**: 각 ROM을 수동으로 선택할 필요가 없습니다.  
✅ **명확한 구성**: 각 게임에 어떤 콘텐츠가 있는지 쉽게 확인할 수 있습니다.  
✅ **쉬운 유지 관리**: 콘텐츠 추가/제거가 간소화됩니다.

> **📝 중요 참고**: 게임에 업데이트나 DLC가 없는 경우 해당 폴더를 만들 필요가 없습니다. 실제로 가지고 있는 콘텐츠에 대한 폴더만 만드세요.

### 결과

이 구조가 마련되면 에뮬레이터가 "DS Games" 폴더에 있는 모든 게임을 자동으로 감지하고 라이브러리에 올바르게 정리합니다! 🎯

## ⚙️ 3단계: 닌텐도 DS BIOS 설치

### 필요한 BIOS 파일

> ⚠️ **(필수 아님)**: 최적의 기능을 위해서는 닌텐도 DSi BIOS 파일이 필요합니다.

**필요한 파일:**
- `biosdsi7.bin` (16KB) - ARM7 BIOS
- `biosdsi9.bin` (4KB) - ARM9 BIOS  
- `dsifirmware.bin` (256KB) - DS 펌웨어

**📥 닌텐도 DSi BIOS 다운로드:**

[닌텐도 DSi BIOS 다운로드](https://cdn.discordapp.com/attachments/1332956018665328702/1381293934109786246/Nintendo_DSi_Bios_for_emulators.zip?ex=6846fdb9&is=6845ac39&hm=e36162ccf2877cea45ea2ab9e004f2d38fddd8a7583be41f7e8414edc49e1383&)

> **📋 내용:** `biosdsi7.bin`, `biosdsi9.bin`, `dsifirmware.bin`

### melonDS에서 구성

1.  `melonDS.exe`를 **실행**합니다.
2.  `Config → Emu settings → DSi-mode`
3.  BIOS 경로를 **구성**합니다.
    - **DS ARM9 BIOS**: `bios/biosdsi9.bin`
    - **DS ARM7 BIOS**: `bios/biosdsi7.bin`
    - **DS Firmware**: `bios/dsifirmware.bin`
4.  melonDS를 **다시 시작**합니다.

## ⚙️ 4단계: 기본 구성

### 일반 설정
```
Config → Emu settings → General
```

**권장 구성:**
- **콘솔 유형**: DS (DSi 게임의 경우 DSi)
- **직접 부팅**: ✅ 활성화 (게임을 직접 시작)
- **JIT 활성화**: ✅ 활성화 (성능을 크게 향상)
- **JIT 최대 블록 크기**: 32 (최적)
- **JIT 리터럴 최적화**: ✅ 활성화
- **JIT 분기 최적화**: ✅ 활성화
- **JIT 빠른 메모리**: ✅ 활성화

### 3D 설정
```
Config → Video Settings
```

**디스플레이 설정**(왼쪽)과 렌더링 설정 **소프트웨어 렌더러** / **OpenGL 렌더러**(오른쪽)의 두 가지 주요 섹션이 있는 창에 액세스하게 됩니다.

#### 3D 렌더링 구성

**렌더링 엔진 선택:**

| 옵션 | 성능 | 품질 | 권장 사항 |
|--------|-------------|---------|----------------|
| **소프트웨어** | 낮음 | 기본 | 오래된 하드웨어 |
| **OpenGL (클래식)** | 우수 | 매우 좋음 | ✅ **권장** |
| **OpenGL (컴퓨트 셰이더)** | 가변 | 최대 | 최신 GPU만 |

#### OpenGL 설정 (권장)

**내부 해상도:** GPU에 따라 `1x 네이티브 (4096x3072)`에서 `16x`까지
- 1x = 원본 DS 해상도
- 2x-3x = 좋은 품질/성능 절충안  
- 4x+ = 강력한 GPU 필요

**향상 옵션:**
- ✅ **개선된 폴리곤 분할**: 기하학적 품질 향상
- ✅ **고해상도 좌표 사용**: 텍스처 정밀도 증가

#### 일반 디스플레이 설정

```
디스플레이 설정 (왼쪽)
```

**권장 구성:**
- **VSync**: ✅ 활성화 (화면 찢어짐 방지)
- **VSync 간격**: `1` (표준 60 FPS)
- **OpenGL 디스플레이**: ✅ OpenGL 디스플레이를 위해 선택

### 시각적 결과

이러한 설정을 통해 DS 게임은 원본 콘솔에 비해 **해상도가 배가**되고 **디스플레이 품질이 상당히 향상**됩니다!

> **💡 성능 팁**: 속도 저하가 발생하면 렌더링 엔진을 변경하기 전에 먼저 **내부 해상도**를 줄이세요.

## ⚙️ 5단계: 비디오 및 오디오 구성

### 비디오 설정
```
View
```

**디스플레이 설정:**
- **DS 화면 레이아웃**: 자연스러움, 수직, 수평 (선택 사항)
- **DS 화면 크기**: 균등 (화면 크기 동일) 또는 상단/하단 강조
- **DS 화면 간격**: 0-96 픽셀 (개인 취향)
- **화면 필터링**: 선형 (더 부드러움) 또는 최근접 (선명한 픽셀 아트)
- **OSD 표시**: ✅ 활성화 (유용한 정보)

### 오디오 설정
```
Config → Audio settings
```

**최적 구성:**
- **오디오 출력**: 자동 (권장)
- **오디오 볼륨**: 256 (최대)
- **마이크 입력**: 자동 또는 특정 장치
- **마이크 노이즈 게이트**: ✅ 활성화 (배경 소음 필터링)

## ⚙️ 6단계: 컨트롤 구성

### 키보드 컨트롤 (기본값)
```
Config → Input and hotkeys → DS keypad
```

**기본 매핑:**
- **A**: X
- **B**: Z  
- **X**: A
- **Y**: S
- **L**: Q
- **R**: W
- **Start**: Return
- **Select**: RShift

(개인적인 예시, 원하는 대로 설정하세요)


## 🔧 문제 해결

### 실행 문제

#### melonDS가 시작되지 않음
```bash
시스템 확인:
1. Visual C++ 재배포 가능 패키지 설치됨
2. GPU에서 OpenGL 3.2+ 지원
3. 그래픽 드라이버 최신 상태
4. 바이러스 백신: melonDS 폴더 예외 처리
5. 필요한 경우 관리자 권한
```

#### ROM이 실행되지 않음
```bash
ROM 진단:
1. 형식: 유효한 .nds인지 확인
2. 크기: 파일이 손상되지 않음
3. BIOS: 파일이 있고 유효함
4. 지역: PAL/NTSC/JPN 호환
5. 보호: 일부 덤프에는 패치가 필요함
```

## ❓ FAQ - 자주 묻는 질문

### 일반 질문

**Q: melonDS는 모든 DS 게임을 플레이할 수 있나요?**
**A:** 약 97%의 게임이 완벽하게 작동합니다. 전반적으로 뛰어난 호환성을 보입니다.

**Q: BIOS 파일이 필요한가요?**
**A:** 예, 최적의 기능과 최대 호환성을 위해 필요합니다.

**Q: melonDS는 DSi 게임을 지원하나요?**
**A:** 부분적으로 지원합니다. 일부 DSi 게임은 작동하지만 다른 게임은 아직 작동하지 않습니다.

### 기술 질문

**Q: OpenGL 또는 소프트웨어 렌더러 중 어느 것이 더 낫나요?**
**A:** 성능 및 시각적 향상을 위해 OpenGL을 권장합니다. 호환성을 위해 소프트웨어를 사용합니다.

**Q: 어떤 해상도를 선택해야 하나요?**
**A:** 좋은 품질/성능 절충안으로 2x-3x. 강력한 GPU가 있다면 4x+.

**Q: JIT는 안전한가요?**
**A:** 예, 좋은 성능을 위해 필요합니다. 보안 위험은 없습니다.

### 일반적인 문제

**Q: 좋은 PC인데도 랙이 걸리나요?**
**A:** JIT가 활성화되었는지 확인하고, OpenGL을 사용하고, 필요한 경우 스케일 팩터를 줄이세요.

**Q: 저장이 작동하지 않나요?**
**A:** 저장 폴더의 쓰기 권한, 올바른 ROM 형식을 확인하세요.

**Q: Wi-Fi가 작동하지 않나요?**
**A:** Wi-Fi는 지원되는 게임에 제한되며 네트워크 구성이 필요합니다.
