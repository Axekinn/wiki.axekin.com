## 🎮 RPCS3 简介

**RPCS3** 是世界上最先进的开源 PlayStation 3 模拟器。它使用 C++ 编写，采用解释和重编译相结合的混合方法来模拟 PS3 的 Cell 架构。经过多年的发展，它现在与超过 60% 的 PS3 游戏具有令人印象深刻的兼容性。

> ⚠️ **法律声明**：此模拟器仅用于玩您合法拥有的游戏。出于备份和研究目的的模拟是合法的，但盗版不是。

## 📋 系统要求

### 最低配置
- **操作系统**：Windows 10 (64位) / Linux / FreeBSD
- **CPU**：Intel Core i5-4430 / AMD FX-6300 (4核, 3.0 GHz)
- **内存**：8 GB
- **GPU**：兼容 DirectX 12 / 兼容 Vulkan 1.1
- **存储**：10 GB 可用空间 + 游戏空间

### 推荐配置
- **CPU**：Intel Core i7-8700 / AMD Ryzen 7 2700 (8核, 3.2+ GHz)
- **内存**：16 GB
- **GPU**：NVIDIA GTX 1060 / AMD RX 580 或更高
- **存储**：具有 50+ GB 可用空间的 SSD

### 最佳配置
- **CPU**：Intel Core i9-12900K / AMD Ryzen 9 5900X (12+ 核, 3.5+ GHz)
- **内存**：32 GB DDR4-3200+
- **GPU**：NVIDIA RTX 3070 / AMD RX 6700 XT 或更高
- **存储**：具有 100+ GB 可用空间的 NVMe SSD

> 💡 **重要提示**：RPCS3 对 CPU 的要求极高。拥有多核心的强大处理器至关重要。

2. 首次启动时**接受**使用条款

5. **等待**完全安装 (1 分钟)
6. **重新启动** RPCS3

## ⚙️ 步骤 4：最佳 CPU 配置

### CPU 设置
```
配置 → CPU
```

**推荐配置：**
- **PPU 解码器**：Recompiler (LLVM) [最快，推荐所有游戏使用]
- **SPU 解码器**：Recompiler (LLVM) [最快，推荐所有游戏使用]
- **附加设置**：
  - **启用线程调度器**：✅ 启用 (提高性能)
  - **启用 SPU 循环检测**：✅ 启用 (防止无限循环)
  - **启用 SPU 缓存**：✅ 启用 (改善加载时间)
  - **启用 SPU 块大小**：Safe (为兼容性推荐)

### 高级 CPU 选项
```
配置 → CPU → 附加设置
```

- **SPU 块大小**：Safe (推荐大多数游戏使用)
- **首选 SPU 线程**：Auto (让 RPCS3 自动优化)
- **启用精确 GETLLAR**：❌ 禁用 (除非特定游戏需要)
- **启用精确 PUTLLUC**：❌ 禁用 (除非特定游戏需要)
- **启用精确 RSX 预留访问**：❌ 禁用 (除非特定游戏需要)
- **SPU 验证**：❌ 禁用 (仅在调试时启用)
- **SPU 缓存**：✅ 启用 (显著提高性能)
- **启用线程调度器**：✅ 启用 (更好的线程管理)

## ⚙️ 步骤 5：最佳 GPU 配置

### 图形设置
```
配置 → GPU
```

**推荐配置：**
- **渲染器**：Vulkan (最佳性能和兼容性)
- **图形设备**：选择您的专用 GPU
- **宽高比**：16:9 (或 Auto 以获得原始宽高比)
- **帧率限制**：Off (让游戏以其预期速度运行)
- **ZCULL 精度**：Relaxed (性能和精度之间的平衡)

### 分辨率和渲染
```
配置 → GPU → 分辨率
```

**最佳设置：**
- **分辨率**：1280x720 (原生 PS3 分辨率)
- **分辨率缩放**：100% (仅在拥有强大 GPU 时增加)
- **分辨率缩放阈值**：16x16 (默认)
- **抗锯齿**：禁用 (仅在有余量时启用)
- **各向异性过滤**：自动 (如果性能允许，则为 16x)

### 高级图形选项
```
配置 → GPU → 高级
```

**性能配置：**
- **默认着色器类型**：Vertex (推荐)
- **着色器模式**：Async (更好的性能)
- **着色器精度**：Auto (让 RPCS3 决定)
- **写入颜色缓冲区**：❌ 禁用 (仅在游戏需要时启用)
- **写入深度缓冲区**：❌ 禁用 (仅在游戏需要时启用)
- **读取颜色缓冲区**：❌ 禁用 (仅在游戏需要时启用)
- **读取深度缓冲区**：❌ 禁用 (仅在游戏需要时启用)
- **VSync**：❌ 禁用 (减少输入延迟)
- **GPU 纹理缩放**：❌ 禁用 (仅在需要时启用)
- **拉伸到显示区域**：❌ 禁用
- **强制 CPU blit**：❌ 禁用 (使用 GPU 以获得更好的性能)
- **禁用 ZCull 遮挡查询**：❌ 禁用
- **禁用顶点缓存**：❌ 禁用
- **禁用 FIFO 重新排序**：❌ 禁用
- **启用跳帧**：❌ 禁用
- **禁用磁盘着色器缓存**：❌ 禁用 (保持缓存启用)
- **禁用 Vulkan 内存分配器**：❌ 禁用
- **使用 GPU 渲染线程**：✅ 启用 (提高性能)

### 着色器加载
```
配置 → GPU → 着色器加载
```

- **着色器编译提示**：使用 Async (减少卡顿)
- **着色器加载对话框**：显示 (首次加载时有帮助)
- **在创建 PPU 重编译器缓存前警告**：✅ 启用
- **在创建 SPU 重编译器缓存前警告**：✅ 启用

## ⚙️ 步骤 6：音频配置

### 音频设置
```
配置 → 音频
```

**推荐配置：**
- **音频输出**：Cubeb (跨平台，推荐)
- **音频设置**：
  - **转储到文件**：❌ 禁用
  - **转换为 16 位**：❌ 禁用
  - **音频缓冲区持续时间**：100ms (默认，需要时调整)

### 音频质量
```

```

- **音频缓冲区**：0.1 秒 (100ms - 良好的平衡)
- **音频后端**：Cubeb (为稳定性推荐)
- **启用时间拉伸**：❌ 禁用 (除非有音频同步问题)

## ❓ FAQ - 常见问题

### 一般问题

**问：RPCS3 能运行所有 PS3 游戏吗？**
**答：**大约 65% 的游戏可以玩，90% 可以启动。兼容性在不断提高。

**问：我需要 PS3 才能使用 RPCS3 吗？**
**答：**固件不需要 (可以合法下载)。但需要合法地转储您的游戏。

**问：为什么 RPCS3 如此耗费资源？**
**答：**PS3 的 Cell 架构模拟起来很复杂。1 个 PPU + 6 个 SPU 需要大量的 CPU 算力。

### 技术问题

**问：Vulkan 还是 D3D12？**
**答：**Vulkan 通常性能更好。D3D12 在某些系统上更稳定。两者都试试。

**问：最低需要多少内存？**
**答：**最低 8GB，推荐 16GB，最耗资源的游戏需要 32GB。

**问：我的 CPU 足够吗？**
**答：**推荐最低 6 核。8+ 核为最佳。单线程性能很重要。

### 常见问题

**问：尽管 PC 很强大，但游戏还是卡顿？**
**答：**对于某些游戏来说是正常的。RPCS3 模拟复杂的架构。检查可用的补丁。

**问：如何提高性能？**
**答：**SPU 缓存 + 预编译，使用 SSD 存放缓存，关闭后台应用程序，适度超频 CPU。

**问：音频不同步？**
**答：**增加音频缓冲区，更改音频后端，检查帧同步。
