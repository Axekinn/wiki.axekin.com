> **⚠️ 중요**: 재설치하기 전에 항상 데이터를 백업하십시오! 이 가이드는 안전하게 진행하는 방법을 보여줍니다.

## 소개

때때로 **Yuzu** 또는 **Ryujinx**는 전체 재설치가 필요한 문제에 직면합니다. 파일 손상, 문제가 있는 업데이트 또는 단순히 새로 시작하고 싶은 경우...

이 가이드는 **Nintendo Switch 게임 세이브를 잃지 않고** 이러한 에뮬레이터를 **깨끗하게 재설치하는 방법**을 설명합니다.

### 왜 재설치해야 하는가?

**재설치가 필요한 일반적인 문제:**
- 에뮬레이터가 시작되지 않음
- 업데이트 후 성능 저하
- 구성 파일 손상
- 지속적인 그래픽 문제
- 반복되는 오류 메시지

## Yuzu 방법

### 1단계: 게임 데이터 백업

#### 세이브 위치 찾기

1. **Yuzu를 시작합니다**(가능한 경우).
2. 목록에서 원하는 게임을 **마우스 오른쪽 버튼으로 클릭**합니다.
3. **"세이브 데이터 위치 열기"**를 선택합니다.

#### 콘텐츠 백업

4. 열리는 폴더의 **모든 콘텐츠를 선택**합니다(Ctrl+A).
5. 파일을 **복사**합니다(Ctrl+C).
6. 바탕 화면이나 하드 드라이브에 **백업 폴더를 만듭니다**.
7. 이 폴더에 콘텐츠를 **붙여넣습니다**(Ctrl+V).

**정리를 위한 권장 구조:**
```
📁 백업_Yuzu_2025-06-04/
├── 📁 젤다의_전설_야생의_숨결/
│   ├── save_data.bin
│   └── 기타_파일...
├── 📁 슈퍼_마리오_오디세이/
│   ├── save_data.bin
│   └── 기타_파일...
└── 📁 포켓몬_레전드_아르세우스/
    ├── save_data.bin
    └── 기타_파일...
```

#### 모든 게임에 대해 반복

8. 세이브를 보존하려는 각 게임에 대해 **이 작업을 반복**합니다.
9. 복원을 용이하게 하기 위해 각 폴더의 **이름을 기록**합니다.

### 2단계: Yuzu 데이터 삭제

#### 구성 폴더 접근

1. **Windows + R**을 누릅니다.
2. `%appdata%`를 입력하고 **Enter**를 누릅니다.

#### Yuzu 폴더 삭제

3. **"yuzu"** 폴더로 이동합니다.
4. 이 폴더를 **완전히 삭제**합니다.
5. 공간을 확보하기 위해 **휴지통을 비웁니다**.

> **💡 팁**: 폴더가 사용 중인 경우 작업 관리자를 통해 Yuzu를 완전히 닫으십시오.

### 3단계: [Yuzu 재설치](/ko/post/yuzu/)

### 4단계: 세이브 복원

#### 세이브 파일 되돌리기

1. 세이브 폴더를 만들기 위해 게임을 한 번 **시작**합니다.
2. Yuzu를 완전히 **닫습니다**.
3. 각 게임을 **마우스 오른쪽 버튼으로 클릭** → "세이브 데이터 위치 열기"
4. 백업된 파일로 콘텐츠를 **교체**합니다.

**확인:**
- 게임을 시작하여 세이브가 제대로 로드되는지 확인합니다.
- 진행 상황, 통계 등을 확인합니다.

## Ryujinx 방법

### 1단계: Ryujinx 데이터 백업

#### 사용자 세이브 위치 찾기

1. **Ryujinx를 시작합니다.**
2. 원하는 게임을 **마우스 오른쪽 버튼으로 클릭**합니다.
3. **"사용자 세이브 디렉토리 열기"**를 선택합니다.

![Ryujinx 메뉴](path_save_1.png)
*Ryujinx의 세이브 옵션*

#### 추가 데이터 백업

4. 가능한 경우 **"장치 세이브 디렉토리 열기"** 및/또는 BCAT도 클릭합니다.
5. 모든 콘텐츠를 백업 폴더에 **복사**합니다.
6. Yuzu와 같이 게임 이름으로 **정리**합니다.

### 2단계: Ryujinx 삭제

#### 구성 폴더 위치 찾기

1. **Windows + R**을 누릅니다.
2. `%appdata%`를 입력하고 확인합니다.
3. **"Ryujinx"** 폴더로 이동합니다.
4. 이 폴더를 **완전히 삭제**합니다.

### 3단계: 재설치 및 복원

#### [Ryujinx 재설치](/ko/post/ryujinx/)

#### 세이브 복원

4. **각 게임을 한 번 시작**합니다.
5. Ryujinx를 **닫습니다**.
6. 세이브 파일을 백업으로 **교체**합니다.
7. 모든 것이 올바르게 작동하는지 **테스트**합니다.

### 중요한 확인 사항

**재설치 전:**
- ✅ 전체 백업 생성됨
- ✅ 게임 이름 기록됨
- ✅ 펌웨어/키 사용 가능

**재설치 후:**
- ✅ 에뮬레이터가 올바르게 시작됨
- ✅ 게임 감지됨
- ✅ 세이브 복원됨
- ✅ 만족스러운 성능

### 일반적인 문제 및 해결책

#### 🚫 세이브가 인식되지 않음

**가능한 원인:**
- 잘못된 파일 위치
- 호환되지 않는 에뮬레이터 버전
- 손상된 파일

**해결책:**
1. 정확한 파일 위치를 확인합니다.
2. 유사한 에뮬레이터 버전으로 시도합니다.
3. 이전 백업을 사용합니다.

## 결론

올바른 방법을 따르면 세이브를 보존하면서 **Yuzu**와 **Ryujinx**를 재설치하는 것은 간단한 과정입니다. 중요한 것은 **행동하기 전에 항상 백업**하고 단계별로 진행하는 것입니다.

### 기억해야 할 핵심 사항

> **🎯 절차 요약:**
> 1. 모든 중요한 게임 데이터를 **백업**합니다.
> 2. 구성 폴더를 **깨끗하게 삭제**합니다.
> 3. 최신 버전으로 에뮬레이터를 **재설치**합니다.
> 4. 세이브를 **복원**하고 테스트합니다.
> 5. 기본 설정에 따라 **구성**합니다.

### 예방 유지 보수

**재설치를 피하기 위해:**
- 정기적이지만 신중한 업데이트
- 주간 자동 백업
- 성능 모니터링
- 주기적인 캐시 정리

## FAQ - 자주 묻는 질문

**Q: Yuzu와 Ryujinx 간에 세이브를 전송할 수 있습니까?**
**A:** 예, 인터넷에 많은 튜토리얼이 있습니다.

**Q: 전체 절차는 얼마나 걸립니까?**
**A:** 가지고 있는 게임 수에 따라 완전히 다릅니다.

**Q: 재설치하기 전에 백업을 잊어버렸다면 어떻게 해야 합니까?**
**A:** Windows 휴지통을 확인하거나 파일 복구 도구를 사용하거나 시스템 백업에서 복원하십시오.
