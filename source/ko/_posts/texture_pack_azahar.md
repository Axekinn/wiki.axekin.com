# 전체 가이드: Azahar 에뮬레이터에 텍스처 팩 설치하기

## 소개

Citra를 기반으로 하는 **Azahar** 에뮬레이터를 사용하면 텍스처 팩을 통해 Nintendo 3DS 게임의 시각적 경험을 크게 향상시킬 수 있습니다. 이 팩은 원본 텍스처를 고화질 버전으로 교체하여 눈에 띄게 향상된 그래픽을 제공합니다.

## 1단계: 에뮬레이터 준비

### 1.1 텍스처 팩 지원 활성화
1. Azahar에서 **에뮬레이션** → **설정**으로 이동합니다.
2. **그래픽** 탭을 선택합니다.
3. **사용자 지정 텍스처 사용**을 선택합니다.
4. **사용자 지정 텍스처 미리 로드**를 선택합니다(선택 사항, 성능 향상).
5. **확인**을 클릭합니다.

## 2단계: 텍스처 팩 구하기

### 2.1 권장 소스
- **텍스처 팩 모음**: https://vikingfile.com/folder/-ryt9sVCLmtmdMo4zGrv5NfgK2MeJHIc_Y_2uVDYaYk (임시 링크, 곧 업데이트 예정)

### 2.2 호환성 확인
텍스처 팩이 다음에 맞는지 확인하세요.
- ROM 지역(EUR, USA, JPN)
- 게임 버전
- Azahar/Citra 호환성

## 3단계: 텍스처 팩 설치

### 3.1 게임의 텍스처 폴더에 액세스하기
1. Azahar의 게임 목록에서 원하는 게임을 **마우스 오른쪽 버튼으로 클릭**합니다.
2. 컨텍스트 메뉴에서 **열기**를 선택합니다.
3. **사용자 지정 텍스처 위치**를 클릭합니다.
4. 올바른 게임 ID(예: 0004000000055D00)가 있는 폴더가 자동으로 열립니다.

### 3.2 텍스처 팩 설치하기
1. 텍스처 팩(.zip 또는 .7z 파일)을 **다운로드**합니다.
2. 열린 폴더에 직접 내용을 **압축 해제**합니다.
3. 텍스처 파일(.png)이 이 폴더에 올바르게 배치되었는지 **확인**합니다.

### 3.3 최종 폴더 구조
압축 해제 후 폴더에는 다음이 포함되어야 합니다.

```
0004000000055D00/
├── tex1_0x12345678_14.png
├── tex1_0x87654321_14.png
└── ... (기타 파일)
```

### 올바른 폴더 구조

```
textures/
└── 0004000000055D00/
    ├── tex1_0x12345678_14.png
    ├── tex1_0x87654321_14.png
    └── ... (기타 텍스처 파일)
```

### 텍스처 팩 구성
체계적인 구조를 만듭니다.
```
textures/
├── Pokemon_X_Y/
├── Mario_Kart_7/
├── Zelda_ALBW/
└── backup/
```

## 팁 및 모범 사례

### 성능
- **SSD 권장**으로 로딩 시간 단축
- **16GB RAM**으로 영향 없이 미리 로드
- **최신 GPU**로 고해상도 구현

### 호환성
- 항상 저장 진행 상황의 백업으로 테스트하세요.

### 유지 관리
- Azahar를 정기적으로 업데이트하세요.
- 새 텍스처 팩 버전을 확인하세요.

## 결론

Azahar에 텍스처 팩을 설치하면 Nintendo 3DS 게임의 시각적 경험이 완전히 바뀝니다. 이 단계별 가이드를 따르면 좋아하는 게임을 향상된 그래픽으로 즐길 수 있습니다.

시스템에 맞는 완벽한 설정을 찾기 위해 다양한 텍스처 팩과 구성을 자유롭게 실험해 보세요.
