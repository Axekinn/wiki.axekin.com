## 소개

**Yuzu** 및 **Ryujinx** 에뮬레이터에 **DLC(다운로드 가능 콘텐츠)** 및 **업데이트**를 설치하는 것은 에뮬레이트된 Nintendo Switch 게임을 완전히 즐기기 위한 간단하지만 필수적인 과정입니다.

이 가이드는 가장 인기 있는 두 에뮬레이터에서 진행하는 방법을 **단계별로** 설명합니다.

### 지원되는 파일 형식

**허용되는 파일 유형:**
- **NSP** - Nintendo Submission Package (주요 형식)

**업데이트와 DLC의 차이점:**
- **업데이트**: 기본 게임 업데이트 (수정, 개선)
- **DLC**: 추가 유료 콘텐츠 (새로운 레벨, 캐릭터 등)

## Yuzu에 설치

### 글로벌 방법 (업데이트 + DLC)

#### 1단계: 설치 메뉴에 액세스

1. **Yuzu를 실행**하고 인터페이스가 로드될 때까지 기다립니다.
2. 메뉴 표시줄에서 **"파일"**을 클릭합니다.
3. **"NAND에 파일 설치"**를 선택합니다.

#### 2단계: 파일 선택

4. **파일 선택 창**이 열립니다.
5. DLC/업데이트 파일의 위치로 **이동**합니다.
6. 파일(.nsp)을 **선택**합니다.
7. **"열기"**를 클릭합니다.

#### 3단계: 설치 확인

8. Yuzu가 파일을 **분석**하고 정보를 표시합니다.
9. 확인을 요청하는 대화 상자가 나타납니다.
10. **"예"**를 클릭하여 설치를 진행합니다.
11. 설치가 완료될 때까지 **기다립니다** (진행률 표시줄).

#### 4단계: 확인

12. **성공 메시지가 표시되면** 설치가 완료됩니다.
13. Yuzu를 **다시 시작**하여 모든 것이 올바르게 인식되었는지 확인합니다.
14. 게임을 실행하여 DLC/업데이트가 활성화되었는지 확인합니다.

**성공 지표:**
- "설치가 성공적으로 완료되었습니다" 메시지
- 게임 내에서 새로운 콘텐츠 확인 가능
- 게임 속성에서 업데이트된 버전 표시

## Ryujinx에 설치

### 업데이트용

#### 1단계: 업데이트 관리 액세스

1. **Ryujinx를 실행**하고 목록에서 게임을 찾습니다.
2. 업데이트하려는 게임을 **마우스 오른쪽 버튼으로 클릭**합니다.
3. 컨텍스트 메뉴에서 **"업데이트 관리"**를 선택합니다.

#### 2단계: 업데이트 추가

4. **"업데이트 관리"** 창이 열립니다.
5. **"추가"** 또는 **"찾아보기"** 버튼을 클릭합니다.
6. 업데이트 파일(.nsp)을 **선택**합니다.
7. **"열기"**를 클릭하여 추가합니다.

#### 3단계: 업데이트 적용

8. 업데이트가 **버전**과 함께 목록에 나타납니다.
9. 업데이트 옆의 **확인란을 선택**하여 활성화합니다.
10. **"저장"**을 클릭하여 변경 사항을 적용합니다.

#### 업데이트 확인

11. 관리 창을 **닫습니다**.
12. 게임 목록에서 **새 버전**이 표시되어야 합니다.
13. **게임을 실행**하여 업데이트가 활성화되었는지 확인합니다.

### DLC용

#### 업데이트와 유사한 절차

**방법은 정확히 동일합니다:**
1. 게임을 **마우스 오른쪽 버튼으로 클릭** ← **"DLC 관리"** (업데이트 관리 대신)
2. **추가/찾아보기** ← DLC 파일 선택
3. DLC를 **선택**하여 활성화
4. **저장**하여 적용

### 설치된 버전 확인

#### Yuzu에서
1. 게임을 **마우스 오른쪽 버튼으로 클릭** ← **"속성"**
2. **"일반"** 탭 ← 버전 확인
3. **"설치된 콘텐츠"** 탭 ← DLC 목록

#### Ryujinx에서
1. 정보가 게임 목록에 **직접 표시**됩니다.
2. **마우스 오른쪽 버튼 클릭** ← **"업데이트/DLC 관리"**에서 세부 정보 확인
3. 버전이 게임 이름 옆에 표시됩니다.

## 일반적인 문제 해결

### 설치 문제

#### 🚫 파일을 인식할 수 없음

**가능한 원인:**
- 호환되지 않는 파일 형식
- 손상된 파일
- 암호 해독 키 누락

**해결책:**
1. 파일 형식(.nsp, .xci, .nca)을 확인합니다.
2. 손상된 경우 파일을 다시 다운로드합니다.
3. 업데이트된 `prod.keys`를 설치합니다.
4. 펌웨어 호환성을 확인합니다.

#### ⚠️ 설치 실패

**일반적인 오류 메시지:**
- "NAND에 설치 실패"
- "잘못된 서명"
- "키 누락"

**단계별 해결책:**
1. **사용 가능한 디스크 공간을 확인**합니다.
2. 에뮬레이터를 완전히 **닫고** 다시 시작합니다.
3. 필요한 경우 **관리자 권한으로 실행**합니다.
4. 에뮬레이터를 **최신 버전으로 업데이트**합니다.

## 결론

Yuzu 및 Ryujinx에 **DLC 및 업데이트**를 설치하는 것은 일단 익숙해지면 간단한 과정입니다. 주요 차이점은 접근 방식에 있습니다. Yuzu는 글로벌 방법을 사용하는 반면 Ryujinx는 더 세분화된 관리를 제공합니다.

## 자주 묻는 질문 (FAQ)

**Q: 동일한 DLC의 여러 버전을 설치할 수 있습니까?**
**A:** 아니요, DLC당 하나의 버전만 가능합니다. 새 버전을 설치하면 이전 버전이 자동으로 교체됩니다.

**Q: DLC가 다른 게임 지역에서 작동합니까?**
**A:** 일반적으로 아니요. DLC는 기본 게임의 지역(EU, US, JP)과 일치해야 합니다.

**Q: 설치가 중단되면 어떻게 해야 합니까?**
**A:** 에뮬레이터를 닫고 관리자 권한으로 다시 시작한 다음 다시 시도하십시오. 사용 가능한 디스크 공간도 확인하십시오.

**Q: 세이브가 DLC 설치에 영향을 받습니까?**
**A:** 아니요, 기존 세이브는 그대로 유지됩니다. 일부 DLC는 세이브에 새로운 콘텐츠를 추가할 수 있습니다.

**Q: DLC 또는 업데이트를 제거하는 방법은 무엇입니까?**
**A:** Ryujinx에서는 DLC/업데이트 관리에서 확인란을 선택 취소하면 됩니다. Yuzu에서는 에뮬레이터의 시스템 설정에서 제거해야 합니다.
