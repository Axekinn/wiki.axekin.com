## 🎮 Dolphin 소개

**Dolphin**은 세계에서 가장 진보된 Nintendo GameCube 및 Wii 에뮬레이터입니다. 2003년부터 개발되어 거의 완벽한 호환성, 화려한 그래픽 향상, 온라인 게임 및 가상 현실과 같은 독특한 기능을 제공합니다.

## 🔧 1단계: Dolphin 다운로드

### 옵션 1: 안정 버전 (권장)
```bash
# 공식 사이트
https://dolphin-emu.org/download/
```

1.  "Download Dolphin 5.0"을 **클릭**합니다.
2.  `dolphin-x64-X.0.zip`을 **다운로드**합니다.
3.  전용 폴더(예: `Documents\Dolphin` 또는 다른 드라이브의 루트, C 드라이브 제외)에 **압축을 풉니다**.

### 옵션 2: Dolphin Android
```bash
# 최적화된 모바일 버전
https://play.google.com/store/apps/details?id=org.dolphinemu.dolphinemu
```

## 게임 다운로드

### 1단계: 게임 라이브러리 접속

**[🎮 Axekin Games - Nintendo Wii](https://www.axekin.com/games?platform=wii)**로 이동하여 Nintendo Wii 게임 컬렉션에 액세스합니다.

### 2단계: 게임 다운로드

1.  목록에서 원하는 게임을 **검색**합니다.
2.  다운로드 버튼을 **클릭**합니다.
3.  다운로드가 완료될 때까지 **기다립니다**.

### 3단계: 파일 압축 풀기

다운로드가 완료되면 다음 소프트웨어 중 하나로 아카이브의 **압축을 풉니다**:

| 소프트웨어 | 다운로드 링크 | 호환성 |
|----------|---------------|---------------|
| **7-Zip** | [7-Zip 다운로드](https://www.7-zip.org/) | Windows/Linux |
| **WinRAR** | [WinRAR 다운로드](https://www.win-rar.com/) | Windows |

> **💡 팁**: 7-Zip은 무료이며 오픈 소스인 반면, WinRAR은 평가판 기간 이후 라이선스가 필요합니다.

**결과**: Dolphin에서 사용할 준비가 된 게임 파일을 얻게 됩니다! 🎯

## 📁 2단계: 폴더 구조

파일을 정리하기 위해 이 폴더 구조를 만듭니다:

라이브러리를 최적으로 구성하기 위한 권장 트리 구조는 다음과 같습니다:

````
Wii Games/
├── Alice in Wonderland/
│   ├── BASE/                # 🎮 기본 게임 (메인 파일)
│   ├── UPDATE/              # 🔄 게임 업데이트 (권장)
│   └── DLC/                 # 📦 다운로드 가능 콘텐츠 (선택 사항)
├── Alvin and the Chipmunks: Chipwrecked/
│   ├── BASE/                # 🎮 기본 게임
│   ├── UPDATE/              # 🔄 게임 업데이트
│   └── DLC/                 # 📦 추가 DLC
├── Animal Crossing: City Folk/
│   ├── BASE/
│   ├── UPDATE/
│   └── DLC/
````

### 콘텐츠 유형


- **BASE** 📁: 다운로드하는 메인 게임
- **UPDATE** 📁: 업데이트 및 패치 (권장)
- **DLC** 📁: 추가 유료 콘텐츠 (선택 사항이지만 권장)


✅ **자동 감지**: 게임이 에뮬레이터에 자동으로 추가됩니다.  
✅ **간소화된 관리**: 각 ROM을 수동으로 선택할 필요가 없습니다.  
✅ **명확한 구성**: 각 게임에 대해 어떤 콘텐츠가 있는지 쉽게 확인할 수 있습니다.  
✅ **쉬운 유지 관리**: 콘텐츠 추가/제거가 간소화됩니다.

> **📝 중요 참고**: 게임에 업데이트나 DLC가 없는 경우 해당 폴더를 만들 필요가 없습니다. 실제로 소유한 콘텐츠에 대해서만 폴더를 만드세요.

### 결과

이 구조가 마련되면 에뮬레이터가 "Wii Games" 폴더에 있는 모든 게임을 자동으로 감지하고 라이브러리에 올바르게 정리합니다! 🎯

## 💡 휴대용 구성 (권장)


**첫 실행 전**에 Dolphin 디렉토리의 루트에 `portable.txt`라는 파일을 만듭니다. 이 작업은 설치를 완전히 휴대용으로 만듭니다.

### 휴대용 모드의 장점

| 측면 | 일반 모드 | 휴대용 모드 |
|--------|-------------|---------------|
| **구성** | 시스템에 저장 | 폴더에 저장 |
| **저장** | 컴퓨터에 분산 | 폴더에 중앙 집중 |
| **휴대성** | ❌ 시스템에 연결 | ✅ 쉽게 이동 가능 |
| **백업** | 찾기 어려움 | 백업하기 쉬움 |

### 휴대용 파일 생성 후 구조

```
📁 Dolphin/
├── 🎮 Dolphin.exe
├── 📄 portable.txt          # ← 이 파일을 만드세요!
│   ├── 📁 bis/
│   ├── 📁 games/
│   ├── 📁 profiles/
│   └── 📁 system/
└── 📄 기타 파일...
```


**모든 데이터**(구성, 저장, 프로필)는 `portable.txt` 파일에 저장되어 관리 및 백업이 훨씬 쉬워집니다!

### 생성 지침

1.  Dolphin 설치 폴더로 **이동**합니다.
2.  정확히 `portable.txt`라는 새 파일을 **만듭니다**.
3.  Dolphin을 **실행**하면 자동으로 휴대용 모드를 감지합니다.

> **📝 참고**: 이 작업은 **첫 실행 전**에 수행해야 적용됩니다!

## ⚙️ 3단계: 설치 및 첫 실행

### 첫 실행
1.  `Dolphin.exe`를 **실행**합니다.
2.  데이터 전송을 **수락(또는 거부)**합니다.
3.  게임 폴더를 **구성**합니다: `옵션 → 구성 → 경로`
4.  이전에 본 ROM 폴더를 **추가**합니다.

## 🎯 4단계: 기본 시스템 구성

### 일반 설정
```
구성 → 일반
```

#### 일반 탭
- **듀얼 코어 활성화**: ✅ 활성화 (성능을 크게 향상시킴)
- **치트 활성화**: ✅ 활성화 (코드를 사용하려는 경우)

#### 인터페이스 탭
- **언어**: 원하는 언어 선택
- **테마**: 원하는 테마 선택
- **스타일**: 원하는 스타일 선택
- **게임 커버 등 다운로드**: ✅ 활성화 (아이콘용)

나머지는 기본값으로 둡니다.

### 컨트롤러 구성

> 이를 위해 자이로가 있는 특정 게임에서 더 잘 작동하는 몇 가지 구성이 있습니다. 이는 유튜버에게 맡기겠습니다. 그들은 보통 YouTube 바이오에 구성 파일을 남겨 둡니다. 저는 Mario Galaxy를 넣었지만, 여러분의 게임을 넣으면 해당 게임에 맞는 구성을 얻을 수 있습니다(있는 경우): [컨트롤러 튜토리얼](https://www.youtube.com/results?search_query=gyro+controller+configuration+dolphin+mario+galaxy)

## ⚙️ 5단계: 최적의 그래픽 구성

### 기본 그래픽 설정
```
그래픽 → 일반
```

#### 그래픽 백엔드
- **백엔드**: Vulkan (권장) 또는 Direct3D 11/12
- **장치**: 메인 GPU
- **화면 비율**: 자동 또는 16:9 강제
- **V-Sync**: 활성화, 전체 화면도 마찬가지

#### 시각적 향상

**필수 향상:**
- **내부 해상도**: 원하는 대로, 값이 높을수록 성능 소모가 많아짐
- **안티 앨리어싱**: 위와 동일
- **비등방성 필터링**: 위와 동일

다른 모든 것은 기본 설정으로 둡니다.

### 고급 설정
```
그래픽 → 고급
```

**최적 구성:**
- **FPS 표시**: ✅ 활성화 (성능 모니터링)
- **프로그레시브 스캔 활성화**: ✅ 활성화
- **백엔드 멀티스레딩**: ✅ 활성화
- **독점 전체 화면 선호**: ✅ 활성화 (지연 시간 감소)

## 🌐 고급 기능

### 넷플레이 (온라인 게임)

#### 넷플레이 구성
```bash
# 세션 호스트
1. 도구 → 넷플레이 시작
2. 호스트 탭 → 설정 구성:
   - 게임: 게임 선택
   - 플레이어: 플레이어 수
   - 버퍼: 4-8 (핑에 따라 다름)
3. 시작 → 세션 코드 공유

# 세션 참여
1. 도구 → 넷플레이 시작  
2. 연결 탭 → 코드 입력
3. 연결하고 동기화 대기
```

#### 최적의 넷플레이 설정
```bash
# 안정적인 경험을 위해
일반:
- 듀얼 코어: ❌ 비활성화
- 유휴 건너뛰기: ❌ 비활성화
- 오디오: DSP HLE (더 안정적)

그래픽:
- 백엔드: Direct3D 11 (Vulkan보다 안정적)
- 해상도: 1x 네이티브 (네트워크 부하 감소)
- 향상: 최소
```

## ❓ FAQ - 자주 묻는 질문

### 일반 질문

**Q: Dolphin은 모든 GameCube/Wii 게임을 플레이할 수 있나요?**
**A:** 약 95%의 게임이 완벽하게 또는 사소한 버그로 작동합니다. 호환성 목록을 확인하세요.

**Q: 실제 GameCube 컨트롤러를 사용할 수 있나요?**
**A:** 예, 공식 Nintendo GameCube-USB 어댑터 또는 호환되는 Mayflash를 사용하면 됩니다.

**Q: 실제 콘솔 저장이 작동하나요?**
**A:** 예, GCMM과 같은 홈브류로 저장을 가져오거나 내보낼 수 있습니다.

### 기술 질문

**Q: Vulkan 또는 Direct3D?**
**A:** Vulkan은 일반적으로 최신 GPU에서 더 나은 성능을 제공합니다. D3D11은 구형 시스템에서 더 안정적입니다.

**Q: 듀얼 코어를 활성화해야 하나요?**
**A:** 더 나은 성능을 위해 예, 단 넷플레이에서는 비활성화해야 합니다.

**Q: 어떤 내부 해상도를 선택해야 하나요?**
**A:** 2x 네이티브(856x528)가 좋은 절충안입니다. 강력한 GPU의 경우 4x 이상.

### 일반적인 문제

**Q: 강력한 PC에도 불구하고 게임이 지연되나요?**
**A:** 일부 게임은 더 까다롭습니다. 듀얼 코어를 활성화하고, 해상도를 줄이고, 백엔드를 변경하세요.

**Q: 오디오에 잡음이 들리나요?**
**A:** 오디오 백엔드를 변경하고, 지연 시간을 늘리고, 다른 오디오 앱을 닫으세요.

**Q: Wiimote가 연결되지 않나요?**
**A:** Bluetooth 어댑터를 확인하고, "연속 스캔"을 사용하고, Dolphin을 통해 페어링하세요.

**GameCube/Wii 클래식을 고화질로 다시 만나보세요! 🎮✨**
